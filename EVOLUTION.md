### 自我反思: 为何遗漏了生态系统分析?

**根因诊断 — 5个认知盲区:**

| # | 盲区 | 根因 | 表现 | 修正 |
|---|------|------|------|------|
| 1 | **加法思维** | CFA训练强调"部分之和"估值 (SOTP) | 将Google视为6个独立业务线简单相加 | 引入**乘法效应**(生态协同溢价) |
| 2 | **财务可见偏差** | 只分析资产负债表可见资产 | 忽略$700B+数据资产(搜索图谱/行为数据) | 引入**隐性资产估值层** |
| 3 | **静态快照偏差** | DCF/PE都是时间切片分析 | 未建模内部蚕食转型动态(AI→搜索) | 引入**蚕食转换矩阵** |
| 4 | **产品力=份额假设** | 简单将份额归因于产品优势 | 忽略分发垄断贡献(Safari协议≈$20B/yr) | 引入**分发vs产品力拆分** |
| 5 | **DCF确定性偏差** | 只折现可预测现金流 | 完全忽略Waymo/量子/DeepMind期权价值 | 引入**嵌入期权估值** |

**深层原因: 框架继承偏差**

```
v1.0-v7.0: 周期投资框架 → 设计目标=分析"单一产品线+明确周期"的公司
  ↓ 继承
v8.0: 心理学层 → 仍基于"单一信号→单一标的"逻辑
  ↓ 继承
CFA L2/L3: 估值+风控 → 经典教材假设"公司=现金流机器"
  ↓ 盲区
生态型公司: 价值来自"连接"而非"现金流", 来自"乘数"而非"加法"
```

**核心认知升级:**
- 周期股: 价值 = f(周期位置, 估值) — **线性函数**
- 生态股: 价值 = f(网络效应^n, 飞轮速度, 锁定复合度, 期权组合) — **非线性函数**
- 不能用分析DOW(化工)的框架去分析GOOGL(生态系统)

## 已提取的通用经验

### 经验 #1: PB优于PE（亏损期）
- 来源: energy
- 内容: PE在亏损期失效，PB更稳定。需同时配置PE和PB，亏损时自动切换
- 已应用于: energy, machinery

### 经验 #2: CAPEX收缩是底部信号
- 来源: energy (rig count YoY < -40%)
- 内容: 行业CAPEX大幅收缩代表供给侧出清，是强底部信号
- 已应用于: energy

### 经验 #3: PE陷阱 (Peak Earnings Trap)
- 来源: machinery (v2迭代)
- 内容: 利用率>78% + 增长正(>=10%) + PE<18 = 顶峰盈利陷阱。低PE不是便宜，是盈利最高点
- 已应用于: machinery
- 泛化规则: 任何行业在产能利用率极高+增长正+估值看似便宜时，都是周期顶部

### 经验 #4: 扩张保护
- 来源: machinery (v2迭代)
- 内容: YoY增长>25% + 配套指标正增长 = 扩张初/中期，不应卖出
- 已应用于: machinery
- 泛化规则: 极端正增长代表复苏动力强劲，逆周期不应在此时触发卖出

### 经验 #5: 衰退未触底信号
- 来源: machinery (v2迭代)
- 内容: 销量下降(-5%~-30%) + PE>25 = 盈利下滑速度快于股价，未到底部
- 已应用于: machinery
- 泛化规则: 当核心指标下降但估值仍高，不要误认为"跌到位了"

### 经验 #6: ISM收缩确认 (Contraction Confirmation)
- 来源: industrial (v1)
- 内容: ISM<50 + 工业产出负 + 资本品订单负 = 确认衰退，强卖出信号(-25分)
- 已应用于: industrial
- 泛化规则: 当领先指标跌破临界阈值，且滞后指标确认，信号极强

### 经验 #7: 大宗商品确认 (Commodity Confirmation)
- 来源: mining (v1)
- 内容: 铜价>$3 + 铁矿>$90 + CAPEX正增长 = 双重确认过度投资期，价格将下行
- 已应用于: mining
- 泛化规则: 多个价格指标同向确认比单一指标更可靠，收敛信号强于分歧

### 经验 #8: 乙烯价差极端值 (Extreme Spread Signal)
- 来源: chemicals (v1)
- 内容: 价差>=600$/ton + 库存<=25天 = 过热，修正即将到来
- 已应用于: chemicals
- 泛化规则: 利润率指标(价差/毛利)极端高+库存极低=不可持续的繁荣

### 经验 #9: 成本端PE陷阱 (Oil Cost Trap)
- 来源: airlines (v1)
- 内容: oil>90 + PE<10 + load>80 = 成本侵蚀利润，低PE是成本压缩假象
- 已应用于: airlines
- 泛化规则: 成本驱动型行业(航空/化工/运输)，输入成本极高时PE低不代表便宜，是利润被压缩的信号

### 经验 #10: 服务商衰退确认 (Service Company Decline Confirmation)
- 来源: energy (v2迭代)
- 内容: SLB PE>25 + rig_yoy<0 = 服务商盈利崩塌，行业衰退未触底
- 已应用于: energy
- 泛化规则: 上游服务商PE极高意味着盈利已崩塌但股价未跌够，行业仍在下行

### 经验 #11: 早期衰退陷阱 (Early Decline Value Trap)
- 来源: energy (v2迭代)
- 内容: rig刚开始下降(-20~0) + 油价中等(>50) + PB看似便宜(<1.6) = 假底部
- 已应用于: energy
- 泛化规则: 行业衰退初期估值看似便宜是最危险的陷阱，因为盈利还没开始真正下滑

### 经验 #14: 服务商PE阻断底部背离 (Service PE Blocks Bottom Divergence)
- 来源: energy 验证 (CVX)
- 内容: SLB PE>30(盈利崩塌极端) → 底部背离信号不成立，行业仍在下行
- 已应用于: energy(验证)
- 泛化规则: 当服务商盈利极端恶化时，即使估值便宜+价格低，底部信号不可信

### 经验 #15: 高PE恢复信号 (High PE Recovery Signal)
- 来源: machinery 验证 (DE)
- 内容: PE>35 + 活动恢复(yoy>0) + 利用率中(60-78) = 谷底盈利导致的高PE，非贵
- 已应用于: machinery(验证)
- 泛化规则: 周期底部盈利极低→PE极高，应看活动恢复而非PE本身

### 经验 #16: PE极端无效化 (Extreme PE Invalidation)
- 来源: airlines 验证 (UAL)
- 内容: PE>100或亏损时PE无参考意义，改用环境因子(油价+客座+RPK)判断
- 已应用于: airlines(验证)
- 泛化规则: 当PE极端时（>100或负），估值判断应转向业务环境和PB

### 经验 #17: 低成本首次盈利保护 (Low-Cost First Profit Protection)
- 来源: airlines 验证 (UAL)
- 内容: 油价低(<60) + 客座高(>82) + RPK正增长 + PE极端 = 首次盈利期，看多
- 已应用于: airlines(验证)
- 泛化规则: 成本利好+需求强劲时的首次盈利/微利阶段，是趋势起点而非终点

### 经验 #18: 衰退确认增强 (Enhanced Recession Confirmation)
- 来源: machinery 验证 (DE)
- 内容: 销量下降(-30~-5) + PE>15(不便宜) + 利用率>63(未到极底) + 建筑负 = 仍在下行
- 已应用于: machinery(验证)
- 泛化规则: 行业衰退中期（非极端）时PE不极低意味着盈利仍在下滑

### 经验 #12: 价差背离 (Price Divergence Dampening)
- 来源: steel (v1)
- 内容: HRC<900(已从峰值回落) + 铁矿>100(全球需求仍在) = 供给修正非需求崩塌
- 已应用于: steel
- 泛化规则: 当产出品价格下跌但原材料仍坚挺，说明是供给调整而非系统性崩溃，跌幅有限

### 经验 #13: 库存紧缺扩张保护 (Tight Inventory Expansion Protection)
- 来源: auto (v1)
- 内容: SAAR>15.5(需求健康) + inv<30(库存极低) + PB>1.5(估值偏高) = 需求支撑的短缺不是见顶
- 已应用于: auto
- 泛化规则: 需求端仍强劲时的供给短缺≠周期见顶，应等待需求走弱或库存恢复再判断

### 以下为 v6.0 新增经验（来自实盘数据压力测试 2026-01-23）

### 经验 #19: 累积衰退优于单年衰退 (Cumulative Decline > YoY Decline)
- 来源: 实盘验证 - 能源行业钻机数连续3年下降但YoY仅-7%
- 内容: 行业连续下降>=3年，累积跌幅>25% → 底部更近，加分+15；连续1年仅-7%不应触发"早期陷阱"
- 缺陷修复: 原框架用YoY=-7%判定"刚开始下降"，实际是第3年累积-32%
- 泛化规则: 周期分析必须看持续时间，同样的YoY变化在衰退第1年和第3年意义完全不同
- 实现方式: 新增 `cycle_duration` 参数，连续下降年数影响最终评分

### 经验 #20: 行业性亏损确认底部 (Industry-Wide Loss = Bottom Confirmation)
- 来源: 实盘验证 - DOW(EPS=-1.61) + LYB(EPS=-4.03) 同时亏损
- 内容: 龙头公司亏损 + 行业内多家亏损 + 产能关停 = 周期底部最强确认信号(+12)
- 缺陷修复: 原框架PE负时仅切换PB，未将"亏损本身"作为正面信号
- 泛化规则: 行业性亏损证明周期已经足够深，盈利不可能更差 → 边际改善概率极高
- 区别于"假底": 公司仍盈利时PB看似便宜 = 可能是假底（如CVX PE=23仍赚钱）

### 经验 #21: 财务韧性是生存前提 (Financial Resilience Filter)
- 来源: 实盘验证 - DOW高杠杆+持续亏损的再融资风险
- 内容: 周期底部买入前提是公司"活得到复苏"
- 过滤规则:
  - 净债务/EBITDA > 5x (或EBITDA为负): 评分打折 ×0.8
  - 现金可覆盖 >= 2年亏损: 通过
  - 18个月内有大额债务到期: 评分打折 ×0.7
- 泛化规则: PB<1.5是必要条件，但高杠杆公司可能在底部被迫稀释或重组，需验证生存能力

### 经验 #22: 公司特异性折价陷阱 (Company-Specific Discount Trap)
- 来源: 实盘验证 - X (US Steel) PB=0.77 但存在结构性劣势
- 内容: PB<1.0 + 同行业龙头PB>2.0 → 可能是公司问题非周期机会
- 过滤规则:
  - if 公司PB < 行业中位数×0.5: 检查结构性劣势
  - if 有明确运营劣势(如X vs Nucor mini-mill): val_score × 0.7
- 泛化规则: PB<1不一定是周期机会。同行估值正常而你极端便宜，问题可能在公司本身

### 经验 #23: 底部阶段三阶段模型 (Bottom Phase ABC Model)
- 来源: 实盘验证 - DOW触发"扩张保护"但实际仍在底部未回升
- 内容: 区分底部三阶段替代原来不分阶段的"扩张保护"
  - Phase A "仍在恶化": 亏损加深 + 无产能关停 → 不加分（太早）
  - Phase B "到达底部": 亏损 + 产能开始关停 + PB接近book → 加分+8（当前DOW位置）
  - Phase C "开始复苏": 亏损收窄 + 利用率回升 + PB仍低 → 加分+20
- 缺陷修复: 原"扩张保护"+15不区分在底/离底，导致过早加分
- 泛化规则: 相同的低估值+低活动，在恶化期/触底期/复苏期应给予不同权重

### 经验 #24: 宏观叠加层 (Macro Overlay)
- 来源: 实盘验证 - ISM=47.9影响所有周期行业但未被计入
- 内容: 宏观环境对所有周期行业形成系统性影响
  - ISM < 45: 所有周期评分 × 0.85（深度衰退可能加剧）
  - ISM 45-50: 所有周期评分 × 0.95（温和逆风）
  - ISM 50-55: 无调整
  - ISM > 55: 所有周期评分 × 1.05（顺风）
- 泛化规则: 行业周期不独立于宏观周期，制造业整体收缩时所有周期底部可能同步推迟

### 底部反转时机识别框架

> 核心命题: STRONG BUY信号 ≠ 底部已反转。信号给出"方向"，但反转需要"确认"。

**反转三条件模型（需同时满足）**:

```
条件1 - 供给出清确认:
  - 累计关停产能 > 新增产能 (净产能减少)
  - 至少2家以上主要生产商关闭装置
  - 行业CAPEX连续2年下降

条件2 - 盈利拐点确认:
  - 龙头公司（DOW）季度EPS从负转正（或亏损收窄>50%）
  - 乙烯价差从<$200回升至>$300（突破盈亏线）
  - 环比改善连续2个季度

条件3 - 估值底部确认:
  - PB从最低点回升>10%（但仍<2.0，说明未被高估）
  - 行业内出现M&A或战略投资（聪明钱进入）
  - 分析师从"减持"转向"持有/买入"
```

**反转信号强度评级**:
- 3/3条件满足: **确认反转** — 加仓至满仓, Phase C得分+20
- 2/3条件满足: **可能反转** — 维持仓位, Phase B→C过渡
- 1/3条件满足: **尚未反转** — 维持观察仓位, 仍为Phase B
- 0/3条件满足: **仍在恶化** — 降低仓位, 可能从Phase B退回Phase A
