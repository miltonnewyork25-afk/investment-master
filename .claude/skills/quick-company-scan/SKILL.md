# /quick-scan - 半导体公司快速扫描技能

## 触发命令
`/quick-scan [公司代码]`
`/quick-scan [公司名称]`

## 功能描述
在不启动完整深度分析的情况下，快速获取半导体公司关键信息：
- 10-15分钟内完成
- 单次会话可完成
- 不创建正式报告
- 适合初步筛选或快速了解

## 与深度分析的区别

| 维度 | /quick-scan | /analyze-sector |
|------|-------------|-----------------|
| **时长** | 10-15分钟 | 12-15小时 |
| **深度** | L2-L3 | L4.5+ |
| **字数** | 2,000-5,000 | 240,000+ |
| **阶段** | 单次会话 | 5阶段 |
| **产出** | 对话内展示 | 正式MD报告 |
| **用途** | 初筛/快速了解 | 投资决策依据 |

## 数据来源优先级

```yaml
优先使用:
  1. 公司最新财报 (A级)
  2. 预测市场数据 (A级)
  3. WSTS/SIA行业数据 (A级)
  4. TrendForce/IC Insights (B级)
  5. 主流财经网站 (B级)

避免使用:
  - 过时数据 (>30天)
  - 未经验证的消息
  - 主观判断无数据支撑
```

## 半导体子分类自动识别

```yaml
子分类映射:
  代工厂: [TSM, UMC, SMIC, GFS]
  设计公司: [NVDA, AMD, QCOM, AVGO, MRVL]
  设备厂商: [ASML, LRCX, AMAT, KLA, TEL]
  存储芯片: [MU, SK Hynix, Samsung(存储)]
  IDM: [INTC, TXN, Samsung(逻辑)]
  IP/EDA: [ARM, SNPS, CDNS]

自动调整:
  - 代工厂: 增加产能利用率+ASP趋势
  - 设计公司: 增加AI曝光度+设计胜出率
  - 设备厂商: 增加订单积压+周期先行指标
  - 存储芯片: 增加价格趋势+库存周转
```
