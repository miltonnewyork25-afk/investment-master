## 体系概览

专为生态科技公司投资分析设计的完整技能生态系统，集成Polymarket预测市场分析，提供机构级投资决策支持能力。

- **ESG原生价值发现**：识别真正的可持续发展价值创造能力
- **环境影响货币化**：将环境效益转化为具体投资价值
- **技术商业化路径**：深度分析绿色技术的商业化前景
- **政策敏感性管理**：全面评估政策风险和机会
- **预测市场前瞻性**：**集成Polymarket强相关预测概率权重调整**
- **长期价值投资**：聚焦可持续竞争优势和护城河

## 完整技能架构 (12个专业技能)

### 🎯 主要技能 (Primary Skills)

#### 1. **eco-tech-master-framework**
- **描述**：生态科技公司机构级投资分析主框架
- **功能**：15模块整合分析，3阶段执行流程，8.3/10置信度目标
- **适用**：所有生态科技公司投资分析的统一入口
- **🆕 Polymarket集成**：自动触发强相关预测市场分析

#### 2. **environmental-impact-quantifier**
- **描述**：环境影响量化器
- **功能**：CO2减排量化、环境效益货币化、生命周期评估
- **核心**：社会碳成本法($185/tCO2e)、生态服务价值化

#### 3. **carbon-economics-modeler**
- **描述**：碳经济学建模器
- **功能**：碳市场分析、碳信贷评估、碳价格预测
- **覆盖**：EU ETS、加州碳市场、中国ETS、自愿碳市场

#### 4. **green-tech-evaluator**
- **描述**：绿色技术评估器
- **功能**：TRL评估、商业化路径、技术经济性分析
- **重点**：技术成熟度、规模化可行性、竞争定位

#### 5. **policy-scenario-analyzer**
- **描述**：政策情景分析器
- **功能**：4情景建模、监管风险评估、政策敏感性量化
- **场景**：支持型、温和型、中性、限制型政策环境
- **🆕 Polymarket增强**：结合预测市场政策概率调整

#### 6. **sustainability-valuator**
- **描述**：可持续发展价值评估器
- **功能**：ESG原生价值、竞争优势评估、利益相关者价值
- **特色**：ESG调整DCF、可持续护城河分析

#### 🆕 7. **polymarket-predictor-analyzer**

- **核心**：Tesla自动驾驶、交付量、政策支持等强相关预测分析

### 概率权重调整方法论

#### 估值调整公式
```javascript
Tesla调整估值 = 基础汽车价值 + (FSD价值 × 概率权重)

FSD概率权重计算：
技术实现概率 = 75% (Polymarket)
监管批准概率 = 60% (Polymarket)
市场接受概率 = 80% (基本面分析)
综合概率权重 = 75% × 60% × 80% = 36%

示例：
基础FSD价值 = $200B
调整后FSD价值 = $200B × 36% = $72B
```

#### 风险调整框架
```javascript
预测不确定性溢价 = 基础风险溢价 × (1 + 预测分歧系数)
调整贴现率 = 基础WACC + 预测不确定性溢价
概率敏感性 = dValuation / dProbability
```
