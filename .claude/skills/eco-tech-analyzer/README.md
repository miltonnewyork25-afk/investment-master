# 生态科技投资分析技能体系 v1.0

## 体系概览

专为生态科技公司投资分析设计的完整技能生态系统，提供机构级投资决策支持能力。

### 核心设计理念

- **ESG原生价值发现**：识别真正的可持续发展价值创造能力
- **环境影响货币化**：将环境效益转化为具体投资价值
- **技术商业化路径**：深度分析绿色技术的商业化前景
- **政策敏感性管理**：全面评估政策风险和机会
- **长期价值投资**：聚焦可持续竞争优势和护城河

---

## 完整技能架构 (11个专业技能)

### 🎯 主要技能 (Primary Skills)

#### 1. **eco-tech-master-framework**
- **描述**：生态科技公司机构级投资分析主框架
- **功能**：15模块整合分析，3阶段执行流程，8.3/10置信度目标
- **适用**：所有生态科技公司投资分析的统一入口

#### 2. **environmental-impact-quantifier**
- **描述**：环境影响量化器
- **功能**：CO2减排量化、环境效益货币化、生命周期评估
- **核心**：社会碳成本法($185/tCO2e)、生态服务价值化

#### 3. **carbon-economics-modeler**
- **描述**：碳经济学建模器
- **功能**：碳市场分析、碳信贷评估、碳价格预测
- **覆盖**：EU ETS、加州碳市场、中国ETS、自愿碳市场

#### 4. **green-tech-evaluator**
- **描述**：绿色技术评估器
- **功能**：TRL评估、商业化路径、技术经济性分析
- **重点**：技术成熟度、规模化可行性、竞争定位

#### 5. **policy-scenario-analyzer**
- **描述**：政策情景分析器
- **功能**：4情景建模、监管风险评估、政策敏感性量化
- **场景**：支持型、温和型、中性、限制型政策环境

#### 6. **sustainability-valuator**
- **描述**：可持续发展价值评估器
- **功能**：ESG原生价值、竞争优势评估、利益相关者价值
- **特色**：ESG调整DCF、可持续护城河分析

### 🔧 支持技能 (Supporting Skills)

#### 7. **circular-economy-analyzer**
- **描述**：循环经济分析器
- **功能**：循环商业模式、材料流分析、废物价值化
- **维度**：资源循环效率、循环度量、价值保留策略

#### 8. **regulatory-pathway-mapper**
- **描述**：监管路径映射器
- **功能**：合规要求、审批流程、监管成本建模
- **覆盖**：多司法管辖区、时间线预测、风险缓解

#### 9. **esg-native-assessor**
- **描述**：ESG原生评估器
- **功能**：ESG真实性验证、原生特征识别、深度评估
- **核心**：区分ESG包装vs原生、可持续DNA分析

#### 10. **green-finance-modeler**
- **描述**：绿色金融建模器
- **功能**：绿色融资成本、ESG评级影响、影响力投资
- **工具**：绿色债券定价、ESG调整WACC、气候风险建模

#### 11. **eco-monitoring-builder**
- **描述**：生态科技监控系统构建器
- **功能**：实时监控、预警机制、绩效跟踪
- **系统**：50+指标体系、异常检测、预测分析

### 💡 前瞻视野
- **长期视角**：20年战略定位，考虑能源转型全周期
- **政策敏感**：4情景政策建模，监管风险量化管理
- **技术预见**：TRL进步速度，技术突破点识别

### 🏗️ 系统集成
- **端到端流程**：从初始筛选到持续监控完整闭环
- **模块化设计**：可灵活组合，适应不同分析需求
- **标准化输出**：机构级投资委员会就绪材料
