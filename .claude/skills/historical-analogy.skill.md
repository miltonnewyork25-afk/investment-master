# 历史类比引擎 v1.0

## 核心理念

> **"历史不会重复，但会押韵。"** — 马克·吐温

生态科技行业的每一个"新故事"在历史上几乎都有先例。本引擎强制用历史类比验证当前投资判断。

## 触发条件

- 深度分析Phase 3（护城河+估值）时自动触发
- 用户说"历史类比" / "有没有先例" / "以前发生过吗"
- 检测到"这次不一样"类判断时强制触发反驳

## 四大类比库

### 类比库1: 技术扩散S曲线

```yaml
核心模型: 技术采用生命周期（Innovators→Early Adopters→Majority→Laggards）

生态科技S曲线对标:

  光伏 vs 半导体:
    类比逻辑: 学习曲线降本（每累积翻倍→成本下降X%）
    半导体: 摩尔定律，每18月性能翻倍，学习率~20%
    光伏: Swanson定律，每累积翻倍成本-24%（1976-2024已验证）
    当前位置: 光伏处于S曲线中段（全球渗透率~5%→可能加速到30%）
    投资启示: S曲线加速阶段是最佳投资窗口，但要警惕过度产能
    关键指标: 渗透率突破10%后通常加速

  风电 vs 天然气涡轮:
    类比逻辑: 大型设备制造的规模效应+技术迭代
    天然气涡轮: 1960-1990，30年从实验到主流，学习率~15%
    风电: 2000-2030，陆上已成熟，海上正在S曲线早期
    当前位置: 陆上风电=成熟期，海上风电=早期采用者阶段
    投资启示: 海上风电类似1970年代天然气涡轮的投资机会

  储能 vs 锂电池(消费电子):
    类比逻辑: 电池技术从消费电子→电动车→电网储能的扩散路径
    消费电子锂电: 1991-2010，成本从$3000/kWh→$300/kWh
    电网储能: 2015-2030，成本从$350/kWh→目标$100/kWh
    当前位置: 储能处于S曲线早期加速段
    投资启示: 类似2012年EV电池投资窗口

  氢能 vs 天然气管道:
    类比逻辑: 基础设施建设从零到一的周期
    天然气: 1950-1980，30年建成全国管网
    氢能: 2020-2050?，基础设施从零开始
    当前位置: 极早期，类似1955年天然气
    投资启示: 基础设施投资先行，应用端回报需等10+年

S曲线位置判断方法:
  指标:
    - 全球渗透率（vs可寻址市场）
    - 成本vs传统替代品（平价距离）
    - 年增速（>25%=加速中）
    - 政策支持强度（补贴/RPS）
  输出: "当前处于S曲线[创新/早期/加速/成熟/衰退]阶段"
```

### 类比库2: 政策周期类比

```yaml
核心模型: 政策周期=（热情→补贴→过热→退坡→整合→可持续）

历史政策周期案例:

  案例1_中国光伏531新政(2018):
    背景: 补贴驱动装机爆发→财政压力→突然政策刹车
    时间线: 2017高补贴→2018.5.31突然削减→H2需求暴跌40%→产业链价格战
    股价影响: 行业龙头跌幅50-70%
    恢复时间: 18个月（2020年需求恢复，但估值未完全恢复）
    教训: 补贴依赖度>30%营收的公司=高风险
    预警信号: 财政赤字扩大+补贴总额激增+政策吹风会频率降低

  案例2_西班牙FIT追溯性削减(2010-2013):
    背景: 慷慨FIT吸引过度投资→政府财政无法承受→追溯性削减
    时间线: 2008慷慨FIT→2010冻结新项目→2012追溯削减→投资者巨亏
    股价影响: Abengoa最终破产，Gamesa跌90%
    恢复时间: 部分公司永远未恢复
    教训: "追溯性"政策变化是最致命的，必须评估法律保护
    预警信号: 政府债务/GDP飙升+电力系统赤字+民意转向

  案例3_美国PTC到期与续期博弈(2000-2020):
    背景: 风电PTC(生产税收抵免)反复到期→行业周期性波动
    时间线: 每次PTC到期前抢装→到期后需求断崖→续期后恢复
    股价影响: 每次PTC不确定性→行业股票波动30-50%
    恢复时间: PTC续期后3-6个月
    教训: 政策依赖行业的"政策周期"比行业周期更重要
    预警信号: 国会日程+两党博弈+替代方案讨论

  案例4_德国能源转型反思(2020-2024):
    背景: 激进弃核+可再生能源转型→电价飙升→工业竞争力下降→政策调整
    时间线: 2011弃核决定→2022能源危机→2023电价全球最高→2024政策修正
    股价影响: 德国清洁能源股先涨后跌
    教训: 过快转型可能导致政策反弹
    预警信号: 工业电价vs竞争国差距+制造业外迁+民调变化

政策周期位置判断:
  当前位置对照: 检查目标公司所在地区处于政策周期哪个阶段
  输出: "当前处于政策周期[热情/补贴高峰/过热/退坡/整合]阶段"
```

### 类比库3: 泡沫与崩盘模式

```yaml
核心模型: Kindleberger泡沫5阶段（置换→繁荣→狂热→恐慌→崩溃）

生态科技泡沫案例:

  案例1_2008清洁科技泡沫:
    时间线: 2006-2007资本涌入→2008金融危机→80%清洁科技VC归零
    代表: Solyndra(破产), A123 Systems(破产), Fisker(破产)
    原因: 技术不成熟+成本太高+油价暴跌+资本退潮
    恢复: 行业10年后才真正成熟(2018+)
    教训: 技术不成熟时的资本狂热=必然崩盘
    当前对照指标:
      - 行业PE vs 历史均值
      - VC/IPO融资额 vs 实际营收
      - 概念股vs盈利股比例

  案例2_2021-2022 SPAC清洁能源泡沫:
    时间线: 2020-2021 SPAC热潮→氢能/固态电池股暴涨→2022暴跌80-90%
    代表: QS(-85%), PLUG(-80%), BLNK(-90%)
    原因: 零利率推高概念股→加息刺破泡沫→大多数公司仍无盈利
    恢复: 至今未恢复
    教训: 区分"真技术"和"PPT技术"，盈利能力是终极验证
    当前对照指标:
      - 盈利公司vs亏损公司比例
      - 利率环境
      - 估值倍数vs实际增长

  案例3_中国光伏2010-2013周期:
    时间线: 2010产能大扩张→2011欧洲双反→2012-2013破产潮
    代表: 无锡尚德(破产), 赛维LDK(破产), 英利(衰落)
    原因: 盲目扩产+贸易壁垒+补贴退坡
    恢复: 存活公司(隆基/通威)在2016后崛起
    教训: 行业洗牌淘汰80%公司，但存活者成为巨头
    当前对照指标:
      - 行业产能vs需求比
      - 贸易壁垒风险
      - 公司现金流/负债率

泡沫预警评分:
  | 指标 | 当前值 | 泡沫阈值 | 得分(0-10) |
  |------|-------|---------|-----------|
  | PE vs 5年均值 | | >2x | |
  | 亏损公司IPO占比 | | >50% | |
  | VC融资/行业营收 | | >30% | |
  | 散户参与度 | | 极高 | |
  | 媒体狂热度 | | 极高 | |
  | **泡沫评分** | | >7=危险 | |
```

### 类比库4: 公司路径对比（成功vs失败）

```yaml
核心方法: 配对比较法（匹配相似起点，追踪不同结局）

生态科技配对案例:

  对比1_隆基vs尚德(光伏):
    相似起点: 中国光伏龙头，大规模扩产
    分叉点: 技术路线选择(单晶vs多晶) + 财务纪律(保守vs激进)
    隆基(成功): 押注单晶+控制负债率+纵向整合→成为全球龙头
    尚德(失败): 多晶路线+过度举债+盲目扩产→破产
    教训: 技术选择正确+财务纪律 = 穿越周期
    对照检查: 目标公司是否有类似隆基的特质？

  对比2_NextEra vs SunEdison(清洁能源):
    相似起点: 美国清洁能源开发商，快速增长
    分叉点: 扩张节奏(稳健vs疯狂) + 融资结构(保守vs yieldco杠杆)
    NextEra(成功): 稳健扩张+低杠杆+蓝筹电力基础→全球最大可再生能源公司
    SunEdison(失败): 疯狂收购+yieldco杠杆叠加→破产
    教训: 清洁能源行业杠杆是双刃剑
    对照检查: 目标公司资产负债率和扩张速度是否合理？

  对比3_宁德时代vs A123 Systems(电池):
    相似起点: 锂电池制造商，供应整车厂
    分叉点: 市场时机+客户绑定+成本控制
    宁德时代(成功): 踩准EV爆发期+绑定全球车企+成本领先
    A123(失败): 过早进入(2008)+客户单一(Fisker)+成本过高→破产
    教训: 技术+时机+客户多元化缺一不可
    对照检查: 目标公司是否"对的技术在对的时间"？

路径匹配方法:
  1. 用WebSearch获取目标公司关键特征
  2. 在案例库中匹配最相似的配对
  3. 检查目标公司更接近"成功路径"还是"失败路径"
  4. 输出: "与[公司]路径相似度X%，关键差异是[Y]"
```
