# Polymarket预测市场投资分析集成 - 半导体特化版 v2.0

## 概述

将Polymarket预测市场概率数据自动集成到半导体投资分析框架中，专注于地缘政治、技术竞争、AI需求和供应链风险的概率量化。

### 设计理念

- **强相关性原则**：只整合与半导体投资直接相关的预测事件
- **概率权重调整**：基于预测概率调整估值和风险评估
- **自动触发机制**：半导体分析自动激活预测市场集成
- **PPDA背离分析**：发现市场定价与真实概率的系统性偏差
- **PMSI情绪构建**：半导体专用预测市场情绪指数

## 半导体强相关预测事件

### 地缘政治事件（权重40%）
✅ **核心监控**：

- 半导体供应链重组概率 (结构性变化)

### 技术竞争事件（权重30%）
✅ **核心监控**：
- 技术领先地位保持概率 (工艺代差)
- 新架构颠覆概率 (RISC-V/光子芯片等)
- AI芯片竞争格局变化概率 (NVDA vs AMD vs 定制芯片)
- 工艺节点突破时间概率 (2nm/GAA等)

### 需求周期事件（权重20%）
✅ **核心监控**：
- AI需求持续性概率 (核心增长驱动)

- 半导体周期复苏概率 (周期性验证)

### 供应链事件（权重10%）
✅ **核心监控**：
- 供应中断概率 (自然灾害/地缘冲击)
- 产能利用率变化概率 (供给侧)
- 关键材料供应风险概率 (稀有气体/光刻胶)

❌ **不相关预测**：
- 选举结果预测（除非直接影响半导体政策）
- 体育赛事/娱乐事件
- 与半导体业务无直接关系的预测

## PPDA概率-价格背离分析算法

### 核心算法
```
背离度 = (市场隐含概率 - Polymarket实际概率) / 实际概率

信号强度:
  背离度 > 20%: 强信号 → 存在显著套利机会
  背离度 10-20%: 中等信号 → 关注并跟踪
  背离度 < 10%: 弱信号 → 市场基本有效

TSM校准基准:
  台海风险: 市场隐含35% vs Polymarket 18.3% = 91.3%背离 → 强信号
  套利Alpha: 15-25%年化 (TSM验证)
```

### 半导体PPDA必须分析事件（≥6个）
1. 台海冲突概率 vs 市场隐含定价
2. 技术制裁升级概率 vs 供应链折价
3. AI需求持续概率 vs AI溢价水平
4. 技术领先概率 vs 技术溢价
5. 经济衰退概率 vs 周期折价
6. 竞争威胁概率 vs 市场份额溢价

## PMSI半导体情绪指数

### 计算公式
```python
PMSI = (地缘40% + 技术30% + 需求20% + 供应链10%) × 100

地缘政治模块 = (1-台海冲突概率) × 0.6 + (1-制裁概率) × 0.4
技术竞争模块 = 技术领先概率 × 0.8 + (1-竞争威胁概率) × 0.2
需求周期模块 = AI需求增长概率 × 0.6 + 数据中心扩张概率 × 0.4
供应链模块 = (1-供应中断概率) × 0.7 + 产能利用率概率 × 0.3
```

### TSM验证基准
- PMSI = 76.4 (乐观区间)
- 股价相关性 = 0.847
- 历史准确率 = 84.2%

### 投资信号
```yaml
PMSI > 80: 强烈买入信号
PMSI 60-80: 买入信号
PMSI 40-60: 中性
PMSI 20-40: 卖出信号
PMSI < 20: 强烈卖出信号
```

## 质量保证标准

### 预测市场质量要求（v20.0升级）
- **最低流动性**：≥$500K
- **参与者数量**：≥200个独立交易者
- **时间跨度**：预测期限 ≤2年
- **事件明确性**：结算条件清晰无歧义
- **历史准确率**：≥80%

### 数据等级标注
```yaml
A级预测市场数据:
  流动性 ≥ $500K AND 参与者 ≥ 200 AND 历史准确率 ≥ 80%
  标注: [A:Polymarket+概率+流动性]

B级预测市场数据:
  流动性 $100K-$500K OR 参与者 100-200
  标注: [B:Polymarket+概率+限制]

不可用:
  流动性 < $100K OR 参与者 < 100
  处理: 说明数据缺失，提供替代分析
```

## 监控预警系统

### 概率变化阈值
- **重大变化**：概率变化 >10个百分点 → 重新评估所有阈值
- **趋势确认**：连续3天同向变化 >5个百分点 → 更新PMSI
- **交易异常**：交易量增加 >300% → 调查异常原因

### Kill Switch联动
```yaml
预测市场概率触发Kill Switch:
  台海冲突 > 25%: Tier1致命级 → 立即清仓

  AI需求见顶 > 35%: Tier2严重级 → 减仓50%
```
