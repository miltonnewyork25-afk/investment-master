# 通用分析模块（所有行业必须执行）

## Phase 1: 定位与生态（6个模块）

| # | 模块 | 输出 | 深度要求 |
|---|------|------|---------|
| U1 | 公司类型识别 | 周期/成长/价值/生态型 | L2 |
| U2 | 产业链定位 | 所处Layer + 信号传导时间 | L3 |
| U3 | 生态图谱 | 上下游Property Graph | L3 |
| U4 | Mermaid可视化 | 供应链/生态关系图 | L3 |
| U5 | 历史教训检索 | ≥3条相关lessons | L2 |
| U6 | 行业复杂度评估 | 复杂度系数 + 目标字数 | L2 |

**检查点1**: 6项全部完成 → 进入Phase 2
