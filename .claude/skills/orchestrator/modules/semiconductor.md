## 半导体 Kill Switch（追加）

- 🔴 产能利用率 <70% 持续2季度
- 🔴 HBM/先进封装价格战爆发
- 🔴 关键客户流失（占比>20%）
- 🔴 技术代差被追平（<6个月差距）
- 🔴 出口管制升级影响>15%营收

## 半导体周期雷达信号

```
Layer 0 (领先18月): 设备订单 → ASML/LRCX 订单变化
Layer 1 (领先12月): CapEx计划 → 晶圆厂资本支出
Layer 2 (领先6月):  库存/价格 → DRAM/NAND/HBM 价格趋势
Layer 3 (同步):     营收盈利 → 确认用，非预测用
```
