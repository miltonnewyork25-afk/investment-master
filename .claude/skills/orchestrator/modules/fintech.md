## 独创框架1: 6层飞轮动力学 (Flywheel Dynamics)

> **核心洞察**: Fintech的竞争优势不是静态护城河，而是动态飞轮。旋转速度决定竞争结局。

```
Layer 1: 获客引擎 ──→ Layer 2: 产品钩子 ──→ Layer 3: 交叉销售
    ↑                                                    │
    │                                                    ↓
Layer 6: 信任资本 ←── Layer 5: 资金优势 ←── Layer 4: 数据飞轮
```

| 层级 | 名称 | 代理指标 | 健康态 | 病态 | Kill Switch |
|:---:|------|---------|--------|------|------------|
| L1 | 获客引擎 | CAC趋势、有机获客占比、品牌搜索量 | CAC稳定/下降 + 有机>40% | CAC持续上升 + 付费>70% | CAC/LTV>0.5持续2Q |
| L2 | 产品钩子 | 30/60/90天留存率、首产品NPS | 90天留存>60% + 月活>70% | 90天留存<40% | 90天留存<35%持续2Q |
| L3 | 交叉销售 | 人均产品数、cross-sell转化率 | 人均>3 + 多产品LTV>3x | 人均<1.5 + 转化<5% | 人均产品数连续3Q停滞 |
| L4 | 数据飞轮 | NCO率vs行业、审批通过率、模型AUC | NCO<行业 + AUC>0.80 | NCO>行业 + 收紧审批 | NCO连续2Q超行业50% |
| L5 | 资金优势 | 存款增长率、CoF、NIM | 存款增长 + CoF<同业 | 依赖仓库贷款 + CoF>同业 | 存款转负 + CoF↑>50bp |
| L6 | 信任资本 | 监管评级、NPS、品牌搜索趋势 | NPS>50 + 品牌搜索上升 | 监管警告 + NPS下降 | 重大罚款>年收入5% |

**层间传导系数**:

| 传导路径 | 机制 | 量化估计 |
|---------|------|---------|
| L1→L2 | 低成本获客→更多产品测试 | CAC↓10% → 产品实验↑15% |
| L2→L3 | 高留存→交叉机会增加 | 留存率↑10pp → cross-sell率↑3pp |
| L3→L4 | 多产品使用→数据维度丰富 | 产品数↑1 → 数据维度↑30% |
| L4→L5 | 精准风控→信用损失降低 | AUC↑0.05 → NCO↓50bp |
| L5→L6 | 低资金成本→更优利率→满意度 | CoF↓100bp → NPS↑5 |
| L6→L1 | 高信任→口碑推荐→低CAC | NPS↑10 → 有机获客↑5pp |

**飞轮速度评估**:
- 飞轮速度 = Σ(Layer健康评分 × 相邻层传导系数)
- 断裂风险 = 最弱Layer评分 / 最强Layer评分（<0.3 = 高风险）
- 加速度 = 本季飞轮速度 / 上季飞轮速度（<0.9 = 减速预警）

## 独创框架2: 5层信贷周期风暴矩阵 (Credit Cycle Storm Matrix)

> **核心洞察**: 类似半导体6层周期雷达，为Fintech信贷风险提供12-18个月预警窗口。

```
T-18月          T-12月          T-9月           T-6月          T-3月         T=0
  │               │               │               │              │            │
  ▼               ▼               ▼               ▼              ▼            ▼
L1:收益率倒挂 → L1:金融条件  → L2:裁员潮    → L3:储蓄率  → L4:早期   → L5:NCO
   HY利差走阔     NFCI>0        初领失业金↑     下降<4%      逾期上升     飙升
```

## 三维SOTP估值框架

SoFi类Fintech不能用单一估值体系，需要分业务估值：

### 分部估值

| 分部 | 估值方法 | 关键指标 | 可比公司 | 可比倍数范围 |
|------|---------|---------|---------|------------|
| 借贷业务 | PB-ROE框架 | NIM, NCO率, ROE | USB, PNC, ALLY | PB 0.8-1.5x |
| 金融服务 | 收入倍数 + LTV | ARPU, 人均产品数, 会员费 | HOOD(8-12x), PYPL(3-5x) | P/S 3-12x |
| 技术平台(BaaS) | EV/Revenue | 启用账户, 收入增速, 留存率 | MQ(12-18x), FI(6-10x) | EV/Rev 6-18x |
| 品牌溢价/折价 | 飞轮L6评分 | NPS, 品牌搜索, 用户满意度 | — | +5~15% / -5~10% |

### 辅助估值法

| 方法 | 公式 | 适用场景 | 注意事项 |
|------|------|---------|---------|
| Forward P/E | 股价 / 下一财年EPS | 已盈利Fintech | Fintech P/E范围极大(17-59x) |
| P/S | 市值 / TTM收入 | 高增长阶段 | 行业均值~5.7x |
| P/B | 市值 / 账面价值 | 银行维度估值 | 与ROE挂钩 |
| 用户估值 | 市值 / 活跃用户 | 增长叙事 | 差异巨大($100-$1,100) |
| 超额回报模型 | BV × (1 + Σ(ROE-CoE)/(1+CoE)^n) | 长期价值 | Simply Wall St方法 |

### 协同效应调整

```
SOTP加总 = 借贷价值 + 金融服务价值 + BaaS价值
最终估值 = SOTP加总 × (1 + 协同系数)

协同系数 = 交叉销售协同(+5~15%, 飞轮L3评分驱动)
          + 数据共享协同(+3~8%, 飞轮L4评分驱动)
          + 品牌溢价/折价(±5~15%, 飞轮L6评分驱动)
          - 集团折价(0~-10%, 复杂度折价)
```

|------|------|---------|-----|-----|---------|
| SOFI | 数字银行 | 47-59x | 9.3x | 3.2x | 会员增长+银行转型 |
| NU | 数字银行 | 17-22x | ~5.8x | 7.3x | 拉美主导+极低成本 |
| HOOD | 交易平台 | 18-22x | ~8x | ~4x | 加密+散户交易 |
| SQ | 支付生态 | 25-30x | ~3x | ~5x | Cash App+商户 |
