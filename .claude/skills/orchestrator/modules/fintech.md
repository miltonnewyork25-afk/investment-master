## 独创框架1: 6层飞轮动力学 (Flywheel Dynamics)

> **核心洞察**: Fintech的竞争优势不是静态护城河，而是动态飞轮。旋转速度决定竞争结局。

```
Layer 1: 获客引擎 ──→ Layer 2: 产品钩子 ──→ Layer 3: 交叉销售
    ↑                                                    │
    │                                                    ↓
Layer 6: 信任资本 ←── Layer 5: 资金优势 ←── Layer 4: 数据飞轮
```

| 层级 | 名称 | 代理指标 | 健康态 | 病态 | Kill Switch |
|:---:|------|---------|--------|------|------------|
| L1 | 获客引擎 | CAC趋势、有机获客占比、品牌搜索量 | CAC稳定/下降 + 有机>40% | CAC持续上升 + 付费>70% | CAC/LTV>0.5持续2Q |
| L2 | 产品钩子 | 30/60/90天留存率、首产品NPS | 90天留存>60% + 月活>70% | 90天留存<40% | 90天留存<35%持续2Q |
| L3 | 交叉销售 | 人均产品数、cross-sell转化率 | 人均>3 + 多产品LTV>3x | 人均<1.5 + 转化<5% | 人均产品数连续3Q停滞 |
| L4 | 数据飞轮 | NCO率vs行业、审批通过率、模型AUC | NCO<行业 + AUC>0.80 | NCO>行业 + 收紧审批 | NCO连续2Q超行业50% |
| L5 | 资金优势 | 存款增长率、CoF、NIM | 存款增长 + CoF<同业 | 依赖仓库贷款 + CoF>同业 | 存款转负 + CoF↑>50bp |
| L6 | 信任资本 | 监管评级、NPS、品牌搜索趋势 | NPS>50 + 品牌搜索上升 | 监管警告 + NPS下降 | 重大罚款>年收入5% |

**飞轮速度评估**:
- 飞轮速度 = Σ(Layer健康评分 × 相邻层传导系数)
- 断裂风险 = 最弱Layer评分 / 最强Layer评分（<0.3 = 高风险）
- 加速度 = 本季飞轮速度 / 上季飞轮速度（<0.9 = 减速预警）

## 独创框架2: 5层信贷周期风暴矩阵 (Credit Cycle Storm Matrix)

> **核心洞察**: 类似半导体6层周期雷达，为Fintech信贷风险提供12-18个月预警窗口。

```
T-18月          T-12月          T-9月           T-6月          T-3月         T=0
  │               │               │               │              │            │
  ▼               ▼               ▼               ▼              ▼            ▼
L1:收益率倒挂 → L1:金融条件  → L2:裁员潮    → L3:储蓄率  → L4:早期   → L5:NCO
   HY利差走阔     NFCI>0        初领失业金↑     下降<4%      逾期上升     飙升
```

| 层级 | 名称 | 领先时间 | 核心指标 | KS联动 |
|:---:|------|---------|---------|--------|
| L1 | 宏观金融条件 | 12-18月 | 收益率曲线、NFCI、HY OAS | 全部信贷类KS频率提升 |
| L2 | 就业与收入 | 9-15月 | 初领失业金、**科技行业裁员**(Fintech特有)、薪资 | KS-011 L1 |
| L3 | 消费者财务健康 | 6-12月 | 储蓄率、债务收入比、信用卡利用率、BNPL逾期 | KS-011 L2 |
| L4 | 借款人行为 | 3-6月 | 30天逾期、还款模式异常、展期比例 | KS-001 L1 |
| L5 | 微观资产质量 | 同步 | NCO率、90天+逾期、拨备覆盖率 | KS-001 L2/L3 |

**综合评估**: 全绿=季度监控 | 2黄=准备减仓 | 3黄=停止加仓 | 1红=触发KS L2 | 3红+=全面危机

> **详见**: `data/research/sofi_credit_cycle_storm_matrix.md`

## 独创框架3: 12信号双轨哨兵 (Dual-Track Sentinel) — L5

> **核心洞察**: Fintech同时运行在科技逻辑和金融逻辑两条轨道上，任何一条恶化都可能致命。

### 科技轨 (6信号)

| ID | 信号 | 频率 | 预警阈值 |
|----|------|------|---------|
| SEN_FT01 | 用户增长 | 周 | MAU周环比<-3% |
| SEN_FT02 | 参与度 | 周 | DAU/MAU<40% 或 日登录<-10% |
| SEN_FT03 | App评分 | 周 | 评分<4.0 或 周均1星>15% |
| SEN_FT04 | 交叉销售率 | 月 | 人均产品数月环比<-0.05 |
| SEN_FT05 | 获客效率 | 周 | 付费CAC周环比>10% |
| SEN_FT06 | 竞品动态 | 周 | 竞品下载量超过本产品20% |

### 金融轨 (6信号)

| ID | 信号 | 频率 | 预警阈值 |
|----|------|------|---------|
| SEN_FT07 | 逾期率 | 周 | 30+DPD周环比>20bp |
| SEN_FT08 | 存款流动 | 周 | 日均存款周环比<-2% |
| SEN_FT09 | 资金成本 | 周 | 短期融资利率周环比>15bp |
| SEN_FT10 | 监管信号 | 实时 | 监管公告/处罚/新规任何负面事件 |
| SEN_FT11 | 信用利差 | 周 | CDS/债券利差扩大>25bp |
| SEN_FT12 | 宏观利率 | 周 | 联邦基金期货隐含利率周变化>10bp |

### 双轨交叉预警

| 级别 | 条件 | 行动 |
|------|------|------|
| 🔴 红色 | 科技轨≥3 + 金融轨≥2 同时触发 | 立即减仓 |
| 🟠 橙色 | 任一轨≥3信号触发 | 停止加仓 + 密切监控 |
| 🟡 黄色 | 任一轨≥2信号触发 | 加入观察清单 |

## 三维SOTP估值框架

SoFi类Fintech不能用单一估值体系，需要分业务估值：

| 分部 | 估值方法 | 关键指标 | 可比公司 |
|------|---------|---------|---------|
| 借贷业务 | PB-ROE框架 | NIM, NCO率, ROE | USB, PNC, ALLY |
| 金融服务 | 收入倍数 + LTV | ARPU, 人均产品数, 会员费 | HOOD, PYPL |
| 技术平台(BaaS) | EV/Revenue | 启用账户, 收入增速, 留存率 | MQ, FI, JKHY |
| 品牌溢价/折价 | 飞轮L6评分 | NPS, 品牌搜索, 用户满意度 | +5~15% / -5~10% |
