# 逆向压力测试引擎 v1.0

## 核心理念

> **"告诉我我会死在哪里，我就不去那里。"** — 查理·芒格

每一个投资论点都有致命弱点。本引擎的唯一任务：找到它们。

## 触发条件

- report-assembler完成投资建议后自动触发
- 用户说"压力测试" / "反驳" / "熊市情景" / "证伪"
- 任何投资评级为"关注"或以上时强制执行

## 五层压力测试框架

### Layer 1: 投资论点解构

```yaml
论点解构流程:
  1. 提取核心投资论点（从report-assembler输出）
  2. 拆解为独立假设（每个论点→3-5个底层假设）
  3. 为每个假设赋予"脆弱性评级"

假设脆弱性评级:
  S级(钢铁): 有硬数据支撑，几乎不可能被推翻
  A级(坚固): 强证据链，但存在边界条件
  B级(一般): 依赖趋势延续或共识判断
  C级(脆弱): 依赖单一数据源或主观判断
  F级(致命): 一旦被推翻，整个投资逻辑崩塌

输出格式:
  | 核心论点 | 底层假设 | 脆弱性 | 证伪条件 | 证伪概率 |
  |---------|---------|--------|---------|---------|
```

### Layer 2: 熊市情景建模（生态科技专用）

```yaml
标准熊市情景库:

  场景1_补贴悬崖:
    触发: 政府补贴突然削减>50%
    历史先例: 2018中国光伏531新政，2012西班牙FIT削减
    影响链: 补贴削减→项目IRR下降→需求暴跌→产能过剩→价格战→利润归零
    量化影响:
      营收: -20%~-40%
      利润率: 压缩5-15个百分点
      估值: PE下修30-50%
    概率评估: 用WebSearch查最新政策信号→赋概率

  场景2_技术颠覆:
    触发: 替代技术突破使现有路线过时
    历史先例: 薄膜vs晶硅(2008), 锂铁vs三元(2020)
    影响链: 新技术效率突破→现有产能贬值→研发投入归零→市场份额流失
    量化影响:
      资产减值: 10-40%产能减记
      竞争力: 技术领先变落后
      估值: 成长溢价消失
    概率评估: 用tech-maturity数据→判断替代技术TRL

  场景3_原材料危机:
    触发: 关键材料价格暴涨>100%或供应中断
    历史先例: 2021硅料暴涨(3x), 2022锂价暴涨(10x)
    影响链: 原材料涨价→成本暴涨→LCOE恶化→需求推迟→库存积压
    量化影响:
      毛利率: 压缩10-25个百分点
      LCOE: 恶化15-40%
      需求: 推迟6-18个月
    概率评估: 用supply-chain-mapper数据→判断瓶颈风险

  场景4_碳政策逆转:
    触发: 碳中和政策大幅退坡或延期
    历史先例: 美国退出巴黎协定(2017), 澳大利亚废除碳税(2014)
    影响链: 政策退坡→碳价暴跌→绿色溢价消失→化石燃料回归竞争力
    量化影响:
      碳信用价值: 归零
      Greenium: 消失
      长期需求预期: 下修20-40%
    概率评估: 用policy-impact数据→判断政策稳定性

  场景5_利率冲击:
    触发: 利率持续高位(>5%)超过2年
    历史先例: 2022-2023加息周期对可再生能源股的影响
    影响链: 高利率→WACC上升→项目IRR下降→融资困难→估值下杀
    量化影响:
      LCOE: 每100bps利率→LCOE+5-8%
      估值: 高成长股PE压缩30-50%
      融资: 绿色债券发行减少
    概率评估: 用WebSearch查利率前景

  场景6_黑天鹅(尾部风险):
    触发: 无法预见的极端事件
    案例库:
      - 全球贸易战升级(关税>50%)
      - 核电复兴导致可再生能源边缘化
      - 重大安全事故(储能爆炸/风机倒塌)引发监管冻结
      - 新技术使整个行业过时(核聚变突破)
    量化: 单一事件概率<5%，但组合概率15-25%
```

### Layer 3: 关键假设敏感性矩阵

```yaml
敏感性测试变量（生态科技通用）:

  宏观变量:
    - GDP增长: 基准±2%
    - 利率: 基准±200bps
    - 通胀: 基准±3%
    - 汇率: 基准±15%

  行业变量:
    - LCOE下降速度: 基准±50%
    - 装机增速: 基准±30%
    - 产能利用率: 基准±15%
    - 原材料价格: 基准±50%

  公司变量:
    - 市场份额: 基准±5pp
    - 毛利率: 基准±8pp
    - CAPEX: 基准±30%
    - 研发效率: 基准±40%

输出格式:
  | 变量 | 悲观值 | 基准值 | 乐观值 | 股价影响(悲观) | 股价影响(乐观) |
  |------|-------|-------|-------|--------------|--------------|

  龙卷风图(Tornado Chart)建议:
  按股价影响绝对值排序，识别TOP5最敏感变量
```

### Layer 4: 反向DCF（从股价倒推隐含假设）

```yaml
反向DCF方法:
  步骤:
    1. 取当前股价作为"市场认为的公允价值"
    2. 倒推市场隐含的增长率、利润率、WACC
    3. 对比公司实际基本面和行业现实
    4. 识别市场隐含假设中的不合理之处

  生态科技特有检查:
    隐含增长率 vs 行业天花板:
      "市场隐含的30%复合增长需要X GW新装机，全球总需求仅Y GW？"
    隐含利润率 vs 行业均值:
      "市场隐含25%净利率，但行业平均仅12%，靠什么维持？"
    隐含WACC vs 实际融资成本:
      "市场隐含6% WACC，但公司绿色债券利率已经5.5%？"

  输出:
    | 隐含假设 | 市场预期值 | 实际/合理值 | 差距 | 风险方向 |
    |---------|----------|-----------|------|---------|
```

### Layer 5: Pre-Mortem分析

```yaml
Pre-Mortem方法:
  假设: "现在是2028年，这笔投资亏损了50%"
  倒推: "最可能的失败路径是什么？"

  生态科技失败路径模板:
    路径A_增长陷阱:
      时间线: 2026Q2装机增速放缓→2026Q4产能过剩确认→2027价格战→2028利润归零
      概率: X%
      预警信号: [可提前观察的指标]

    路径B_技术陷阱:
      时间线: 2026新技术论文发表→2027中试成功→2028现有技术被替代
      概率: X%
      预警信号: [专利活动/研发方向/行业会议动态]

    路径C_政策陷阱:
      时间线: 2026选举政策转向→2027补贴削减→2028需求断崖
      概率: X%
      预警信号: [选举民调/政策草案/游说活动]

    路径D_资本陷阱:
      时间线: 2026利率居高不下→2027融资断裂→2028现金流危机
      概率: X%
      预警信号: [利率期货/债券利差/再融资日程]
```

## 与其他skills联动

```yaml
数据输入:
  ← report-assembler: 投资建议+Kill Switch（测试对象）
  ← lcoe-analyzer: 成本数据（敏感性变量）
  ← tech-maturity: 技术风险（颠覆情景）
  ← policy-impact: 政策风险（政策情景）
  ← supply-chain-mapper: 供应链风险（材料危机情景）

数据输出:
  → report-assembler: 更新后的Kill Switch + 仓位调整
  → eco-tech-orchestrator: 投资论点脆弱性评级
```
