#### Buffett核心选股标准

**财务指标要求**:
- ROE > 20%（过去10年平均，无单年<15%）
- 低负债或无负债（偏好D/E < 0.5）
- 强劲自由现金流
- 一美元测试：每保留1美元盈利，市值至少增加1美元

**定性标准**:
- 简单易懂的业务模式
- 持久竞争优势（护城河）
- 优秀管理层（诚实、能干）
- 长期增长前景

**估值标准**:
- 内在价值显著折价
- P/E通常低于市场（但高质量公司可支付溢价）
- P/B历史上偏好<1.5，但现代投资更看重无形资产

**投资时机**:
- 市场恐慌时买入（"别人恐惧时贪婪"）
- 长期持有（"我们最喜欢的持有期是永远"）
- 不择时：有好机会就买

**来源**: [Morningstar: Warren Buffett Was Never Just a Value Investor](https://www.morningstar.com/markets/warren-buffett-was-never-just-value-investor-heres-real-secret-his-investing-success)

#### Ackman的8条投资戒律

1. **简单可预测的业务**: 容易分析和估值
2. **产生自由现金流**: 不投"钱从底部漏出"的公司
3. **高ROIC**: 资本回报率高于资本成本
4. **市场主导地位**: 偏好行业龙头（如Hilton、Chipotle、Netflix）
5. **不依赖外部资本**: 3年内能用FCF还清长期债务

7. **有限外部风险**: 对不可控因素敞口小
8. **有吸引力的估值**: 在临时问题导致股价下跌时买入

**集中度策略**: 只持有8-12个仓位，75%资产集中在6只股票

**买入时机**: 寻找因短期问题被过度抛售的优质公司（如食品安全危机后的Chipotle）

**来源**: [Baupost Group 13F](https://hedgefollow.com/funds/Baupost+Group+Ma)

#### Klarman的《安全边际》核心原则

**投资哲学**:
- 避免损失是首要目标
- 安全边际必不可少（因为估值是艺术，未来不可预测）
- 价值投资 = 以显著折价购买证券

**安全边际的三个来源**:
1. 价格与价值的差距
2. 业务质量本身
3. 有形资产优于无形资产

**选股标准**:
- 不受欢迎但前景良好
- 相对市价有显著折扣
- 存在价值实现催化剂
- 高度复杂或流动性差（减少竞争）
- 因市场无效而价格偏低

**现金管理**: 持有30-50%现金，等待最佳机会

**自1982年成立以来年化回报**: ~20%

**来源**: [Seth Klarman Margin of Safety Summary](https://jamesclear.com/book-summaries/margin-of-safety-risk-averse-value-investing-strategies-for-the-thoughtful-investor)

## 第五部分：顶级投资者共性规律提炼

### 5.1 选股共性（7条核心规律）

| 规律 | 采用基金/投资者 | 量化标准建议 |
|------|------------------|--------------|
| **高ROE** | Buffett, Ackman, 多数价值基金 | ROE > 15%（优选>20%） |
| **强自由现金流** | 全部投资者 | FCF/净利润 > 0.8 |
| **低负债** | Buffett, Ackman, Klarman | D/E < 0.5（金融除外） |
| **持久竞争优势** | Buffett, Ackman, Danoff | 宽护城河评级 |
| **简单可理解** | Buffett, Ackman | 避免复杂业务 |
| **优质管理层** | 全部投资者 | 管理层持股+承诺兑现 |
| **市场主导地位** | Ackman, ValueAct | 行业Top 3 |

### 5.2 估值共性（5条规律）

| 规律 | 采用者 | 量化标准建议 |
|------|--------|--------------|
| **内在价值折价** | Buffett, Klarman | 折价 > 25-30% |
| **P/E合理** | Graham门徒 | P/E < 15或低于同业 |
| **P/B合理** | Graham门徒 | P/B < 1.5 或 P/E × P/B < 22.5 |
| **安全边际** | Klarman, Buffett | 越不确定越需要更大边际 |
| **逆向买入** | 全部价值投资者 | 在悲观时买入 |

### 5.3 时机共性（4条规律）

| 规律 | 采用者 | 实操建议 |
|------|--------|----------|
| **危机买入** | Buffett, Tepper | 恐慌抛售时加仓 |
| **短期问题长期机会** | Ackman, Danoff | 因暂时问题下跌时买入 |
| **估值修复卖出** | Klarman | 价格回到内在价值时考虑卖出 |
| **更好机会出现** | Klarman | 发现更好标的时调仓 |

### 5.4 回避共性（6条规律）

| 规律 | 为什么回避 | 量化筛选 |
|------|------------|----------|
| **高杠杆** | 财务脆弱 | D/E > 2排除 |
| **亏损公司** | 盈利不确定 | EPS < 0排除 |
| **太复杂业务** | 无法准确估值 | 主观判断 |

| **无护城河** | 竞争侵蚀 | 毛利率持续下降 |

## 第六部分：行业专家投资方法论

### 6.1 Benjamin Graham防守型投资者标准

Graham在《聪明的投资者》中提出的7条标准：

1. **规模充足**: 年销售额 > $100M
2. **财务状况强劲**: 流动比率 > 2:1
3. **盈利稳定**: 过去10年每年盈利
4. **股息记录**: 过去20年持续分红
5. **盈利增长**: 10年EPS增长 > 33%
6. **估值合理**: P/E < 15（基于3年平均盈利）
7. **资产估值合理**: P/B < 1.5 或 P/E × P/B < 22.5

### 6.2 Piotroski F-Score

用于区分廉价但健康 vs 廉价但有问题的股票：

| 指标 | 得分条件 |
|------|----------|
| ROA正 | +1 |
| 经营现金流正 | +1 |
| ROA改善 | +1 |
| 现金流 > 净利润 | +1 |
| 长期债务减少 | +1 |
| 流动比率改善 | +1 |
| 未增发股票 | +1 |
| 毛利率改善 | +1 |
| 资产周转率改善 | +1 |

**评分**: 8-9分优秀，0-2分危险

## 第七部分：最成功和最失败的投资案例

### 7.1 最成功投资案例

#### 案例1: Buffett的Apple投资

- **买入**: 2016年开始，总成本约$40B
- **峰值**: 2023年达$174B，占投资组合50%
- **回报**: ~4-5倍
- **成功因素**: 品牌护城河、服务业务转型、股票回购

#### 案例2: Ackman的Canadian Pacific投资

- **买入**: 2011年9月，$49/股，14.2%股权
- **行动**: 代理权争夺→换CEO→战略改革
- **卖出**: 2014-2016年，$220/股
- **利润**: $1.45B
- **成功因素**: 活动投资+运营改善

#### 案例3: Buffett的Coca-Cola投资

- **买入**: 1988年，成本$1.3B
- **当前价值**: $28.1B
- **年化股息收益率（成本基准）**: 63%
- **持有时间**: 37年
- **成功因素**: 永续品牌+全球分销+复利

### 7.2 最失败投资案例

#### 案例1: Ackman的Valeant惨败

- **买入**: 2015年，数十亿美元
- **问题**: 药价争议、会计问题、过度并购
- **卖出**: 2017年
- **损失**: >$4B
- **教训**: 过度信任激进管理层，忽视合规风险

#### 案例2: Ackman的Herbalife做空

- **建仓**: 2012年，$1B做空
- **论点**: "金字塔骗局"
- **结果**: Carl Icahn买入对抗，股价上涨
- **损失**: 数亿美元
- **教训**: 公开对抗可能招致反击

#### 案例3: Bridgewater的近期表现

- **10年回报**: 43%（vs 60/40组合90%）
- **问题**: 高费率（~2-3%/年）侵蚀回报
- **教训**: 费率对长期回报影响巨大
