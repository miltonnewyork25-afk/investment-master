## 数据分级标注体系（建议采用）

### 标注符号定义

```markdown
✅ **VERIFIED** - 真实可验证数据
   来源: 公司财报、SEC文件、交易所官方数据
   例: Tesla Q3 2025汽车毛利率18.7%（来源: 10-Q第XX页）

📊 **ESTIMATED** - 基于模型的估算
   方法: DCF/可比公司/趋势外推
   例: 储能业务合理估值$550-650亿（基于DCF，假设见附录）

🎯 **MODELED** - 情景分析/敏感性测试
   说明: 基于假设的情景推演
   例: 如果Robotaxi 2027年商业化，隐含价值$245亿

⚠️ **SAMPLE** - 示例数据框架
   用途: 演示分析方法，非真实数据
   例: Engine 1内部人交易监控框架（实际需接入SEC API）

❌ **UNVERIFIED** - 待验证或删除
   状态: 无可靠来源，不应作为投资依据
   例: [已删除]
```

### 使用示例

```markdown
# 修正前
Q1 EPS预测: $0.78（Engine 6预测）

# 修正后
📊 **ESTIMATED** Q1 EPS预测: $0.75-0.82（中值$0.78）
   方法: 基于Q4趋势外推，假设汽车毛利率维持18%，储能增速+120%
   共识预期: $0.75（Bloomberg汇总）
   置信区间: 70%
   风险因素: 中国销量、Cybertruck亏损、储能确认时间
```
