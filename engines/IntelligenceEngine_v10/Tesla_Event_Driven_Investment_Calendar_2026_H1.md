## VII. 执行纪律与心理准备

### B. 出场纪律

**5条原则**:
1. **目标收益达成**: 若策略达到预期收益下限,立即平仓50%,剩余50%移至保护性止损
2. **止损触发**: 所有策略设置-40%硬止损(个别高风险策略-60%),触发立即无条件平仓
3. **时间止损**: 若策略持有期超过预设出场日期+3日仍未达目标,平仓(避免时间耗损)
4. **证伪条件**: 若持有期间"证伪条件"触发,立即平仓(无论盈亏)
5. **组合止损**: 若组合当月累计亏损>8%,暂停所有策略,休息1周重新评估

### C. 心理准备

**预期管理**:
- 12个策略不可能全胜,预期胜率55-65%
- 单季度可能亏损(尤其若Tesla横盘整理)
- 最坏情景: 连续3个策略失败,组合回撤-12-18%
- 最好情景: 70%策略获胜,组合回报+35-55%(6个月)
- 现实预期: 60%策略获胜,组合回报+18-28%(6个月)

**常见心理陷阱**:
1. **过度自信**: 前2个策略成功后加大杠杆 → 第3个策略失败导致重大损失
2. **报复性交易**: 策略失败后立即开仓"赢回来" → 违反纪律
3. **锚定偏误**: 股价$449时买入,跌至$420后"等待回本" → 忽视基本面变化
4. **确认偏误**: 只关注支持自己观点的信息 → 忽视反面证据
5. **近因效应**: 最近一次财报大涨,认为"每次都会涨" → 忽视历史62.5%下跌概率

**应对方法**:
- 每周日晚复盘本周所有交易,记录决策质量(非结果)
- 每月末评估每个策略的"决策正确率"vs"结果胜率"(分开评估)
- 若连续3次违反纪律,强制休息2周
- 使用检查清单(Checklist),每次开仓前逐项确认

## XII. 附录

### A. 术语表

| 术语 | 解释 |
|------|------|
| **IV Crush** | 隐含波动率暴跌,通常发生在财报等重大事件后,导致期权价值迅速下降 |
| **Iron Condor** | 由4个期权组成的策略:卖出Call+买入更高Call+卖出Put+买入更低Put,赚取时间价值 |
| **Straddle** | 同时买入/卖出相同行权价的Call和Put,赌波动率而非方向 |
| **Max Pain** | 期权到期时使期权持有者损失最大的股价,通常股价会被"钉"在此价格附近 |
| **Put/Call Ratio** | Put期权成交量/Call期权成交量,>1表示看跌情绪占优 |
| **OI(Open Interest)** | 未平仓合约数,衡量期权市场参与度 |
| **反身性** | Soros理论,价格变化反过来影响基本面,形成正/负反馈循环 |
| **Beta** | 股票相对大盘的波动倍数,Tesla约2.1,即大盘涨1%,Tesla平均涨2.1% |
| **ATM** | At The Money,行权价等于当前股价的期权 |
| **SOTP** | Sum of The Parts,分部估值法 |
