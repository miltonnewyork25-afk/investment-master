*目标：从86分→98分的顶级报告*

---

## 任务背景

基于 ChatGPT 的5条升级建议，构建机构级研报。

---

## 一、核心分析框架（5层）

### 1. Yield-Learning 模型层
**目标**：把"工艺与良率"从描述升级为可校准的模型

**具体任务：**
- 建立良率爬坡模型：N3/N2/CoWoS/SoIC 每条产品线
- 设定 Base/Bull/Bear 三场景参数：爬坡速度、成熟良率、learning rate
- 构建传导链：良率 → 成本曲线 → 毛利率 → 自由现金流
- 输出敏感性分析：良率每变动5%对毛利率的弹性

**学术背书**：参考 ScienceDirect 良率学习建模文献

### 2. EUV 技术风险定价层
**目标**：把技术风险量化进估值

**具体任务：**
- 评估 EUV 风险源：随机缺陷(stochastics)、光子噪声、线边粗糙(LER)
- 建立"节点风险评分"：N7/N5/N3/N2 各节点
- 识别非线性崩塌阈值：缺陷率超过多少会导致利润率崩塌
- 输出：节点风险热力图 + 风险传导链

### 3. 供应链韧性(SCRES)网络模型
**目标**：把地缘政治从"观点"变成"网络系统评估"

**具体任务：**
- 绘制依赖网络：设备/材料/EDA/IP/封装测试/电力水资源/物流
- 标注每个节点：替代性(高/中/低)、恢复时间、管制敏感度
- 构建情景树：封锁/隔离/制裁/局部冲突 → 二阶效应
- 输出：供应链依赖图 + 情景概率扇形图

### 4. 市场隐含预期反证层
**目标**：把估值做成"下注条款"

**具体任务：**
- 反推当前股价隐含的假设：先进节点份额、GM中枢、capex强度、WACC
- 列出每个假设的"证伪条件"
- 建立 5 个 kill switches + 5 个可验证预测
- 输出：预期曲面图 + 证伪仪表盘

### 5. SOTP 估值整合
**具体任务：**
- 三场景估值（Base/Bull/Bear）
- 每个场景写成"下注条款"：必须发生哪3个具体可证伪事件
- 驱动因子瀑布图：Wafer starts → Yield → ASP → GM → FCF → 估值

## 五、⚡反常识洞察卡（至少5张）

### 格式
```
⚡ [结论]
为什么反直觉：[1句]
证据：
1. [证据1] (等级: A/B/C/D/E)
2. [证据2] (等级: X)
3. [证据3] (等级: X)
证伪条件：[1条]
```

### 示例
```
⚡ 周期拐点不是需求驱动，而是 ramp 学习率是否被 EUV stochastics 打断
为什么反直觉：市场通常关注需求端（手机出货量、AI芯片订单）
证据：
1. N3 ramp 延迟6个月导致2023Q2毛利率下降3% (A)
2. 历史数据：良率拐点领先股价拐点平均4个月 (B)
3. 管理层电话会议12次提及"yield improvement" (A)
证伪条件：如果需求暴跌>30%但良率持续提升，股价仍下跌，则此论点错误
```
