## 2. 美国对华半导体出口管制

### 2024-2026关键时间线

| 时间 | 事件 | 对TSM影响 |
|------|------|----------|
| 2024年12月 | BIS新增140家中国实体至实体清单，扩大FDPR范围 | 间接限制TSM中国客户选择 |
| 2025年1月15日 | BIS发布AI Diffusion Rule，将全球分为三级(Tier I/II/III) | 规范AI芯片出口框架 |
| 2025年9月 | BIS宣布TSM南京厂VEU豁免将于2025年底到期 | 短期引发市场担忧 |
| 2025年12月31日 | VEU豁免正式到期 | TSM需新的出口许可 |
| 2026年1月1日 | **美国授予TSM一年期出口许可**，覆盖南京厂 | 运营中断风险消除 |

[硬数据: CNBC, 2026-01-01; TrendForce, 2025-12-26]

**TSM南京厂概况**: 月产能约20,000片12英寸晶圆(16/12nm) + 40,000片(28/22nm)，2024年占TSM总收入约**2.4%**。美国明确限制该许可不得用于扩产或升级先进技术。[硬数据: TrendForce, 2025-09-03]

**BIS关闭VEU漏洞**: 原先外资在华fab可通过VEU免许可出口，新规要求此类企业必须申请出口许可，原VEU参与者有120天过渡期。三星电子和SK海力士也获得了类似的年度许可。[硬数据: BIS.gov, 2025]

## 3. 美国亚利桑那工厂

### 总投资与规模

TSM亚利桑那总投资已从$65B提升至**$165B**，规划fab数从3座增至**6座**，另含2座先进封装设施和1个研发中心。[硬数据: TSMC官方公告, 2025-03]

### Fab进度一览

| 工厂 | 制程 | 状态 | 量产时间 | 月产能 |
|------|------|------|---------|--------|
| **Fab 21 Phase 1** | N4 (4nm) | **已量产** | 2024年底/2025年初 | — |
| **Fab 21 Phase 2** | N3 (3nm) | 建设中，设备搬入2026 Q3 | **2027年** (原计划2028，提前数季度) | — |
| **Fab 21 Phase 3** | N2/A16 (2nm/1.6nm) | 2025年4月开工 | **预计2027-2028年** (原计划提前约1年) | — |
| Fab 4-6 | 待定 | 规划中 | 2020年代末 | — |

[硬数据: TrendForce, 2025-12-18; Tom's Hardware, 2025; TSMC PR, 2025]

### 成本与良率

- **成本溢价**: 亚利桑那晶圆加工成本比台湾**高不到10%** [硬数据: TechInsights分析]
- **劳动力占比**: 美国工资约为台湾的3倍，但劳动力占现代晶圆厂总成本**不到2%** [硬数据: TechInsights]
- **良率表现**: Fab 21 Phase 1良率比台湾同等设施**高约4个百分点** [硬数据: TechPowerUp, 2025]
- **毛利率影响**: 海外工厂前5年预计拉低集团毛利率**约2-3个百分点** [硬数据: TSMC管理层说明]
- **当前员工**: Fab 21约**3,000人**，6座fab全部投产后预计需**6,000+人** [硬数据: Axios Phoenix, 2025-05]

## 6. CHIPS Act补贴

| 指标 | 数据 |
|------|------|
| 直接拨款 | **$6.6B** |
| 受益实体 | TSMC Arizona Corporation |
| 支持项目 | 亚利桑那3座先进fab (原计划投资$65B) |
| 拨付方式 | 里程碑式拨付，2024年底前已拨付至少$1B |
| 协议性质 | 约束性合同(binding agreement)，不可无故撤销 |

### Guardrail条款

| 限制类型 | 具体内容 |
|----------|----------|
| **中国扩产限制** | 10年内禁止在中国等"关注国"进行28nm以下逻辑芯片产能的"实质性扩张"(>5%产能增长) |
| **技术转让限制** | 禁止与"关注国实体"进行涉及国家安全技术的联合研究或技术许可 |
| **股票回购限制** | 禁止使用CHIPS资金进行股票回购或向股东分红 |
| **违规追回(Clawback)** | 违反以上任一条款，须全额退还补贴+利息；违规方有45天纠正期 |
| **通知义务** | 任何可能违规的交易须提前通知商务部 |

[硬数据: NIST/Commerce Dept, 2024-11; Crowell & Moring LLP分析; CHIPS Act原文]

## 7. Section 232半导体关税

| 指标 | 数据 |
|------|------|
| 税率 | **25% ad valorem** |
| 生效日期 | **2026年1月15日** |
| 覆盖范围 | 窄定义——先进逻辑半导体及含此类半导体的产品 |
| 下游影响 | 分析师预计现货市场价格上涨**8-12%** |
| 审查节点 | 90天内(~2026年4月)商务部长和USTR提交更新报告 |
| 数据中心审查 | 2026年7月1日前提交数据中心用半导体市场评估 |
| 潜在调整 | 总统可能根据谈判结果调整税率，并推出"关税抵消计划"(针对在美投资的企业) |

[硬数据: White House Proclamation, 2026-01; White & Case LLP, 2026; EY Global Tax, 2026; PwC, 2026]

**对TSM的影响**: TSM作为在美大规模投资的企业，可能受益于未来的"关税抵消计划"(tariff offset program for companies investing in U.S. semiconductor production)。但短期内，从台湾进口的先进芯片仍受25%关税影响，这增加了客户在亚利桑那fab投片的经济激励。[合理推断: 基于关税结构+TSM美国投资规模]

- **建筑标准**: TSM fab建筑标准**超过**台湾建筑法规要求 (1999年7.6级地震后升级)
- **设备防护**: 阻尼器(dampers)和隔震平台(seismic isolation platforms)保护敏感设备
- **冗余设计**: 制造设施内建冗余系统
