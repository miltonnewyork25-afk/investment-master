### 1.2 商业模式本质

**纯晶圆代工模式 (Pure-Play Foundry)**

台积电不设计芯片、不销售芯片，只做制造 - 这是其核心竞争优势的来源：

```
客户关系优势:
┌──────────────────────────────────────────────────┐
│                                                  │
│  传统IDM模式 (Intel/Samsung):                    │
│  [设计] → [制造] → [销售]                        │
│     ↓         ↓        ↓                         │
│  与客户竞争  技术泄露风险  价格冲突               │
│                                                  │
│  台积电Pure-Play模式:                            │
│  [客户设计] → [TSMC制造] → [客户销售]            │
│       ↓            ↓             ↓               │
│  不竞争      技术中立      利益一致               │
│                                                  │
│  结果: 客户愿意分享roadmap，深度技术合作          │
└──────────────────────────────────────────────────┘
```

**关键洞察**: Intel最终在2024年宣布拆分foundry业务，恰恰证明了台积电模式的优越性 - 当你既是裁判员又是运动员时，没有人愿意把最先进的设计交给你制造。

### 2.2 盈利能力：历史性突破

**毛利率的结构性提升**

| 时期 | 毛利率 | 驱动因素 |
|------|--------|---------|
| 2020-2021 | 50-52% | 7nm/5nm成熟，但价格竞争激烈 |
| 2022-2023 | 54-56% | 3nm初期爬坡，N3E良率提升 |
| 2024 | 57.2% | 3nm规模量产，CoWoS产能紧张 |
| **2025 Q4** | **59.9%** | 3nm满载 + 2nm起量 + 定价权提升 |
| 2026E (管理层指引) | 58-60% | 持续高位，但Arizona工厂拖累 |
| 2027E+ (长期目标) | **56%+** | 结构性毛利率目标上调 |

**关键洞察 - 毛利率59.9%的三重驱动**

1. **制程组合优化** (贡献 ~2 ppts)
   - 3nm营收占比从2024年的15% → 2025年的25%+
   - 2nm在2025年Q4开始贡献营收 (占比<5%，但毛利率>70%)

2. **定价权转移** (贡献 ~2 ppts)
   - 3nm wafer price: $16,000-18,000 → $18,000-20,000 (涨价12%)
   - 部分客户加急订单溢价50-100%

3. **先进封装溢价** (贡献 ~1 ppt)
   - CoWoS产能从2024年的60K wpm → 2025年的95K wpm
   - CoWoS附加值提升ASP和毛利率

**营业利润率50.8%的全球对比**

| 公司 | 营业利润率 | 备注 |
|------|-----------|------|
| **TSMC** | **50.8%** | 2025 Q4 TTM |
| Samsung Foundry | 15-20% | 估算，Samsung不单独披露foundry利润率 |
| Intel Foundry | 负利润 | 2024年亏损$70亿 |
| NVIDIA | 62% | Fabless，无制造成本 |
| ASML | 32% | 设备商，但垄断EUV |

**台积电营业利润率50.8%意味着什么？**
- 每制造$100的芯片，台积电赚$51的营业利润
- 这是制造业中极其罕见的利润率水平，通常只有软件/平台公司能达到
- 反映了其**"类软件公司"的定价权** - 技术护城河带来的超额利润

### 2.3 杜邦分析：ROE 34.2%的拆解

```
杜邦三级拆解 (2025 Q4 TTM)
═══════════════════════════════════════════════════════

ROE = 净利率 × 总资产周转率 × 权益乘数

34.2% = 43.1% × 0.54 × 1.46

详细拆解:
┌────────────────────────────────────────────────────┐
│                                                    │
│  Tier 1: ROE = 34.2%                              │
│          └─ vs 2024: 24.8% (+9.4 ppts) ★★★        │
│                                                    │
│  Tier 2: 驱动因素                                  │
│          ├─ 净利率: 43.1% (vs 2024: 36.5%)        │
│          │   → 贡献: +4.7 ppts ★最大贡献           │
│          ├─ 资产周转率: 0.54x (vs 2024: 0.53x)    │
│          │   → 贡献: +0.6 ppts                    │
│          └─ 权益乘数: 1.46x (vs 2024: 1.58x)      │
│              → 贡献: -1.2 ppts (去杠杆)           │
│                                                    │
│  Tier 3: 净利率拆解                                │
│          43.1% = 毛利率59.9% - 费用率16.8%        │
│          ├─ 毛利率: 59.9% (vs 2024: 54.3%)        │
│          │   → 制程组合优化 + 定价权               │
│          └─ 费用率: 16.8% (vs 2024: 17.8%)        │
│              ├─ 研发: 6.5% (绝对值增加，但占比下降)│
│              └─ SG&A: 2.6% (规模效应)             │
│                                                    │
└────────────────────────────────────────────────────┘
```

**关键发现**

1. **ROE提升主要来自净利率扩张**，而非杠杆或周转率
   - 这是"质量型"ROE提升，可持续性强
   - 对比：如果是靠加杠杆提升ROE，风险大

2. **资产周转率0.54x偏低**，原因是资本密集型特性
   - 台积电每$1资产只能产生$0.54营收
   - 但高毛利率弥补了低周转率
   - 公式：低周转×高毛利 = 高ROE (奢侈品模式)

3. **权益乘数下降** (1.58x → 1.46x)，台积电在去杠杆
   - 净债务为负$547亿 (账上现金远超债务)
   - 即使去杠杆，ROE依然提升，说明盈利能力极强

### 3.2 技术壁垒：2nm领先竞争对手2-3年

**先进制程技术路线对比 (2026年1月)**

| 制程节点 | TSMC | Samsung | Intel | 量产时间差距 |
|---------|------|---------|-------|------------|
| 7nm | ✓ 成熟 | ✓ 成熟 | ✓ (Intel 4) | 持平 |
| 5nm | ✓ 成熟 | ✓ 成熟 | ✓ (Intel 3) | TSMC领先1年 |
| 3nm | ✓ 满载 (N3E良率>90%) | ✓ 良率60-70% | ✗ 跳过 | TSMC领先1.5年 |
| **2nm** | **✓ 2025 Q4量产** | 计划2027年 | 计划2027年(Intel 20A) | **TSMC领先2年** |
| 1.4nm (A14) | 计划2027年 | 未公布 | 未公布 | TSMC领先3年+ |

**2nm技术的三大突破**

1. **GAA (Gate-All-Around) 晶体管架构**
   - 从FinFET升级到GAAFET
   - 电流控制能力提升30%，功耗降低25-30%
   - Samsung 3nm GAA良率问题严重，台积电2nm GAA一次成功

2. **BSPDN (Backside Power Delivery Network)**
   - 后段供电网络，降低IR drop
   - 性能提升6-8%，功耗降低5-8%
   - Intel 20A也有类似技术，但量产时间晚2年

3. **超高NA EUV光刻**
   - 使用ASML最新High-NA EUV设备 (0.55 NA)
   - 台积电2024年已采购3台，Samsung/Intel尚未部署

**为什么2nm领先如此重要？**

```
摩尔定律的经济学 (2026年视角):
┌────────────────────────────────────────────────┐
│                                                │
│  每代制程(~2年) 晶体管密度翻倍                 │
│                                                │
│  3nm → 2nm:                                   │
│  ├─ 晶体管密度: +1.1x (vs传统摩尔定律2x)      │
│  ├─ 性能提升: 10-15% (同功耗)                 │
│  ├─ 功耗降低: 25-30% (同性能)                 │
│  └─ Wafer Cost: +30% (but 每颗芯片成本-15%)   │
│                                                │
│  客户支付意愿:                                 │
│  - Apple: 愿意为A20处理器支付2nm溢价          │
│  - NVIDIA: B300 AI芯片必须用2nm才能达到性能目标│
│  - AMD: MI400 GPU需要2nm降低功耗              │
│                                                │
│  【结论】2nm是2026-2028年的"必选项"，不是"可选项"│
```

### 3.3 垄断资源：ASML EUV设备的唯一规模化用户

**EUV光刻机分布 (2025年底)**

| 公司 | EUV设备数量 | High-NA EUV | 说明 |
|------|-----------|------------|------|
| **TSMC** | **180+台** | **3台** | 全球最多，占ASML总出货55%+ |
| Samsung | 60-70台 | 1台 | 但良率问题导致利用率低 |
| Intel | 25-30台 | 2台 | 18A/20A尚未量产 |
| SK海力士 | 5-10台 | 0 | 仅用于HBM制造 |

**ASML EUV产能瓶颈**

- ASML每年最多生产80-90台EUV (包括High-NA)
- 每台售价$1.5-2亿美元，交期2-3年
- 台积电2024-2026年订购120台，几乎垄断新增产能

**这意味着什么？**

```
EUV垄断的战略意义:
┌──────────────────────────────────────────────┐
│                                              │
│  假设Samsung想追赶台积电3nm/2nm:             │
│                                              │

│                                              │

│  - 2nm已经成熟并扩产                         │
│  - 1.4nm(A14)开始量产                        │

│                                              │
└──────────────────────────────────────────────┘
```

**关键洞察**: EUV设备是"物理护城河" - 不是专利、不是知识产权，而是实实在在的硬件瓶颈。竞争对手即使有钱、有技术，也买不到足够的设备来追赶。
