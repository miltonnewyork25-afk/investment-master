### 3.1 层级决策框架（Level 0-4结构化决策）

#### Level 0: 存在性检验（First Principles三问）

**问题1: 这个公司解决的问题是否真实存在？**

**TSM解决的核心问题**:
- 芯片设计公司（Fabless）需要先进制程制造能力，但自建Fab成本$15-30B+，周期5-7年
- 全球仅3家能生产≤7nm芯片（TSM、Samsung、Intel），TSM占93%份额

**验证问题存在性的证据**:
1. Apple自研芯片但依赖TSM制造（转换成本$200M+ + 24个月）
2. NVIDIA、AMD完全Fabless，100%依赖TSM先进制程
3. Intel放弃部分自制，寻求TSM代工（Arc GPU）

**结论**: ✅ 问题真实且无法自行解决（资本壁垒+技术壁垒）

---

**问题2: TSM的解决方案是否不可替代？**

**可替代性分析**:

| 替代方案 | 可行性 | 转换成本 | 时间成本 | 结论 |
|---------|--------|----------|----------|------|
| Samsung代工 | 部分可行 | $100-150M | 18个月 | 技术差距+良率问题 |
| Intel Foundry | 试验阶段 | $150-200M | 24个月 | NVIDIA暂停测试，不确定性高 |
| 自建Fab | 理论可行 | $20-30B | 5-7年 | NVIDIA/AMD资本不足 |
| 中国SMIC | 不可行 | N/A | N/A | 受限≤14nm，无法替代先进制程 |

**结论**: ✅ 短期（2-3年）不可替代，中期（5年）仍具显著优势

**问题3: 这个行业的终局是什么？**

**半导体代工终局推演**:

**情景A: TSM一家独大**（概率40%）

### 3.2 护城河深度分析（7 Powers Framework）

基于Hamilton Helmer的**7 Powers**框架，分析TSM的竞争优势来源：

#### Power 1: Scale Economies（规模经济）★★★★★

**定义**: 单位成本随产量增加而下降，形成成本优势

**TSM的规模经济体现**:

1. **制造规模**:
   - TSM年产能: 1,600万片12英寸晶圆（2025）
   - Samsung年产能: 600万片
   - TSM单位成本优势: 15-20%

2. **R&D摊销**:
   - TSM R&D支出: $9.3B（2025）
   - 摊销到1,600万片 → $581/片
   - Samsung摊销到600万片 → $1,000/片（假设R&D $6B）
   - **TSM R&D单位成本低42%**

3. **设备采购议价**:
   - TSM是ASML最大客户（55% EUV机台）
   - 采购价格折扣估计10-15%
   - 交期优先权（EUV机台2年等候期，TSM优先交付）

4. **供应链规模效应**:
   - 光刻胶、电子特气等材料批量采购
   - 与供应商联合研发（分摊成本）

**量化规模经济**:

```
学习曲线效应（Wright's Law）:
C(x) = C₁ · x^(-b)

其中:
C(x) = 累积产量x时的单位成本
C₁ = 初始单位成本
b = 学习指数 = -log₂(1-k)
k = 学习率

TSM历史数据（2015-2025）:
- 3nm制程学习率k = 0.14
- 累积产量翻倍 → 成本下降14%

示例:
- 3nm首批10K片晶圆单位成本: $15,000/片
- 累积达到20K片时: $15,000 × (1-0.14) = $12,900/片
- 累积达到40K片时: $12,900 × (1-0.14) = $11,094/片
- 累积达到80K片时: $11,094 × (1-0.14) = $9,541/片

→ 累积产量每翻倍，成本下降14%
→ Samsung累积产量仅TSM的1/3，成本劣势30%+
```

**规模经济护城河评分**: ★★★★★ (5/5)
- 优势明显且难以复制（需数年积累）
- Samsung要追上需3-5年

#### Power 2: Network Effects（网络效应）★★★☆☆

**定义**: 用户越多 → 产品价值越高 → 吸引更多用户

**TSM的网络效应**:

1. **生态系统网络**:
   - 客户越多 → PDK（Process Design Kit）成熟度越高 → 新客户设计更容易
   - TSMC有300+客户 vs Samsung <50客户
   - 成熟PDK → 芯片设计周期缩短30%

2. **供应链网络**:
   - 合作伙伴越多（材料商、设备商）→ 创新速度越快
   - ASML优先为TSM定制EUV功能
   - 应用材料(AMAT)与TSM联合开发薄膜设备

3. **人才网络**:
   - 工程师越多 → 知识积累越深 → 吸引更优秀工程师
   - TSM 65K工程师 vs Samsung 30K

**但网络效应有限**:
- 晶圆代工不是平台型业务（不同于Facebook、微信）
- 客户之间无直接互动
- 效应主要体现在B2B生态，不是C端用户

**网络效应护城河评分**: ★★★☆☆ (3/5)
- 存在但不是核心护城河

#### Power 3: Counter-Positioning（反向定位）★★☆☆☆

**定义**: 新商业模式优于现有模式，但现有玩家因路径依赖无法模仿

#### Power 4: Switching Costs（转换成本）★★★★★

**定义**: 客户切换供应商的成本极高，形成锁定

**TSM的转换成本分析**:

**技术转换成本**:

| 转换环节 | 成本（每个产品） | 时间 | 说明 |
|---------|----------------|------|------|
| 芯片重新设计 | $50-100M | 12-18个月 | 需适配新厂PDK |
| 流片验证 | $20-50M | 6-12个月 | 多次流片确保良率 |
| 供应链调整 | $10-30M | 3-6个月 | 测试设备、封装厂对接 |
| 认证/资质 | $5-10M | 3-6个月 | 汽车/医疗等需重新认证 |
| **总计** | **$100-200M** | **18-24个月** | - |

**案例: Apple若从TSM转向Samsung**

```
假设场景: Apple A20芯片从TSM N2转向Samsung SF2

直接成本:
- 芯片重新设计（适配SF2 PDK）: $80M
- 流片验证（4次Tape-out）: $40M
- 供应链调整（测试/封装）: $15M
- 小计: $135M

机会成本:
- 上市延迟12个月 → iPhone延迟 → 营收损失$10-20B
- 良率风险（SF2良率<60% vs N2 70%+）→ 利润率下降5-10ppt

隐性成本:
- 与TSM长期关系破裂 → 未来谈判地位下降
- 专利交叉授权重新谈判

总成本: $135M + $10B机会成本 = 不可接受
```

**结论**: Apple、NVIDIA、AMD几乎不可能在未来3-5年切换代工厂

**转换成本护城河评分**: ★★★★★ (5/5)
- 最强护城河之一，客户锁定效应极强

#### 产品飞轮分析

**TSM的自我强化飞轮**:

```
飞轮启动点: 先进制程技术领先

         技术领先（N2量产领先18个月）
                 ↓
         吸引顶级客户（Apple独占2nm）
                 ↓
         获得大额订单（Apple预付$5B保证金）
                 ↓
         规模经济效应（1,600万片/年产能）
                 ↓
         单位成本下降（学习率k=0.14）
                 ↓
         研发投入增加（$9.3B R&D，营收6.5%）
                 ↓
         技术继续领先（N1.4 2027年量产）─┐
                 │                      │
                 └──────────────────────┘
                     飞轮加速旋转
```

**飞轮关键节点分析**:

1. **技术领先 → 吸引顶级客户**
   - Apple只选择最先进制程（iPhone需要性能+能效）
   - NVIDIA AI芯片需要最高算力密度
   - **验证**: 2025年Apple+NVIDIA占TSM营收49%

2. **大额订单 → 规模经济**
   - Apple A20芯片订单: 80M颗/年 → TSM N2产能60%
   - 固定成本摊销 → 单位成本下降
   - **验证**: TSM毛利率从2021年51% → 2025年59.9%（+890bps）

3. **成本下降 → R&D投入**
   - 毛利率提升 → FCF增加 → R&D预算增长
   - **验证**: R&D从2021年$6.5B → 2025年$9.3B（+43%）

4. **R&D投入 → 技术继续领先**
   - N1.4制程研发 → 2027年量产 → 继续领先Samsung
   - **验证**: Samsung N1.4预计2028年量产（晚12个月）

**飞轮加速度量化**:

```
飞轮每转一圈的时间缩短:

2015-2018（7nm周期）: 3年
2018-2021（5nm周期）: 3年
2021-2024（3nm周期）: 3年
2024-2026（2nm周期）: 2年 ← 加速

飞轮转速加速原因:
1. AI需求爆发 → 订单增长加速
2. CoWoS封装协同 → 客户锁定更深
3. 竞争对手落后 → 市场份额快速提升
```

**飞轮脆弱性分析**:

⚠️ **潜在断点**:
1. 技术失误（N2良率不达预期）→ 客户流失 → 订单下降 → 飞轮逆转
2. 地缘冲突（台海危机）→ 客户分散供应链 → 份额下降
3. 竞争对手突破（Intel 18A成功）→ 技术领先缩窄 → 定价权下降

**飞轮健康度**: 8.5/10（目前加速旋转，但存在地缘风险）
