# 深挖Q2: Optimus BOM→售价gap的闭合路径

> **问题路由**: [深挖] — 制造成本工程分解=跨行业技术对比的AI优势
> **演绎方法**: 从BOM组件成本出发，推导规模效应能否闭合gap

---

## 演绎链

```
前提1: Gen2 BOM约$55K (手部$9.5-12K最贵)
前提2: 目标售价$20-30K
前提3: BOM > 售价 → 需要(a)BOM降至$15-20K 或 (b)非卖断模式(RaaS租赁)
前提4: 手部BOM目标从$12K→<$3K (Tesla内部目标)
→ 如果手部成功降至$3K，其他组件呢？总BOM能降到多少？
→ 规模效应曲线: 从1K台→1M台的成本下降幅度可参照消费电子
```

## 手部: 为什么最贵且难以降价

手部占Gen2 BOM的17-22%，是最昂贵的单一子系统。原因:

**Gen2设计**: 每只手13组微型滚珠丝杠，用于11-22个自由度(DoF)的灵巧操控。[硬数据: RobotToday]

**Gen3重新设计**: 从13→17组微型滚珠丝杠，可能重新引入齿轮箱+丝杠+腱绳驱动混合方案(gearbox + lead screw + tendon drive)。[硬数据: RobotToday, 供应链信息]

**为什么手部昂贵**:
1. **精密机械**: 每个手指关节需要微型执行器+减速器+力矩传感器+位置编码器——零件数量是机器人其他关节的3-5倍
2. **触觉传感器**: 指尖触觉反馈需要高密度传感器阵列——目前无消费级供应链
3. **可靠性要求**: 手是最频繁接触物体的部件，磨损最快
4. **精度vs成本的矛盾**: 人手有27个DoF+~17000个触觉感受器，任何人工复制都是巨量精密零件

**Tesla内部目标: $12K → <$3K** [硬数据: Standard Bots]
→ 需要75%成本削减。路径: 减少精密组件数量(简化DoF) + 更廉价材料 + 规模采购
→ 但简化DoF意味着牺牲灵巧性——如果Optimus不能灵巧操作，市场吸引力大降

## 结论: gap能否闭合？

**乐观路径**: 手部$12K→$3K + 执行器$15K→$6K(Sanhua规模效应) + 其他组件-30% → 总BOM $20-25K → 售价$25-30K可行。但需要**100K+台产量**和**2-3年时间**。

**现实制约**:
1. **零外部客户** — 100K台产量需要大量外部订单，目前没有
2. **消费级sales目标2027年底** — 还有22个月
3. **$20-30K定价可能需要RaaS模式**(Robot-as-a-Service, 类似汽车租赁) — 不一定是卖断
4. **$1.1B+供应链订单是正面信号** — 但订单≠交付≠客户

**应追踪的信号**:
1. **首个外部付费客户公告** → 从内部验证到商业验证的质变
2. **Sanhua/Tuopu产能爬坡数据** → 实际制造进度
3. **Gen3在Tesla工厂的实际工时/天** → 可靠性的最硬指标
4. **手部成本是否达到<$3K目标** → 核心成本驱动因素
