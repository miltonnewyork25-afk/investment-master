# 深挖Q1: FSD纯视觉方案的L4天花板

> **问题路由**: [深挖] — 端到端架构拆解+物理约束推导=AI分析核心优势
> **演绎方法**: 从物理第一性原理出发，推导纯视觉方案的理论边界

---

## 演绎链

```
前提1: 摄像头是光学传感器，依赖可见光反射 → 有物理约束(暴雨/浓雾/强逆光)
前提2: L4要求在所有ODD(运行设计域)内无接管 → 包括极端天气场景
前提3: 存在某些ODD场景使摄像头信噪比低于安全阈值 → LiDAR在这些场景性能衰减更小
前提4: NHTSA 2025指南要求L4"单一关键传感器故障时仍能安全运行"
→ 推论: 纯视觉方案在物理层面存在ODD天花板，除非(a)限制ODD范围或(b)证明NN能补偿传感器退化
→ 但(a)限制ODD = 不是真正的L4全域，(b)目前无证据
```

这不是"FSD好不好"的问题。这是一个**结构性问题**: 纯视觉路线的理论极限在哪里？

---

## 端到端架构现状

### FSD v13→v14的技术跃迁

**v13** (2025年中): 首次在HW4上使用所有摄像头的原生分辨率（此前降采样），端到端驾驶网络完全重写。FSD v13.2.1是首个利用HW4全摄像头原生分辨率的版本。[硬数据: Tesla release notes, NotATeslaApp]

**v14** (2025.10发布): 底层NN规模扩大**10倍**（跳过了原计划的4倍版本），导航和路由集成到视觉NN中（不再依赖外部地图API进行实时路径规划）。[硬数据: Electrek, Tesla Oracle 2025.10]

这意味着v14是一个**单一巨型transformer**——感知、规划、导航全部端到端，输入是8个摄像头的原始像素流，输出是方向盘/油门/刹车控制信号。[合理推断: 基于Tesla公开架构描述]

### 训练基础设施变化

- **Dojo**: 2025.08关闭，Musk称"进化死胡同"(evolutionary dead end)。核心团队20人离职创立DensityAI，团队负责人Peter Bannon离开。总投入估计$5B+。[硬数据: TechCrunch 2025.08, Reuters]
- **AI5芯片**: 设计接近完成(2026.01.19 Musk确认)，计划2026年底量产，Samsung Taylor TX工厂生产。单SoC性能约HW4的10倍。[硬数据: TrendForce 2025.09, GlobalChinaEV]
- **Dojo 3概念**: Musk提出将AI5/AI6芯片组成板卡用于训练/推理，"可以叫它Dojo 3"——本质是放弃定制ASIC训练芯片路线，转向用推理芯片堆算力。[硬数据: TechCrunch 2025.09]
- **当前训练**: 主要依赖NVIDIA GPU集群(H100/B200)。[合理推断: Dojo关闭后的唯一选项]

**Dojo失败的启示**: $5B+投入关闭说明定制训练ASIC的工程难度远超预期。Musk公开承认技术路线错误("evolutionary dead end")。但Tesla并未放弃自研芯片——AI5/AI6的目标是推理芯片(车载)，不是训练芯片(数据中心)。[合理推断: 这是一个战略收缩到核心需求的决策]

## 纯视觉的物理约束

### 摄像头 vs LiDAR的底层物理

| 维度 | 摄像头(Tesla方案) | LiDAR(Waymo方案) |
|------|-----------------|-----------------|
| 感知原理 | 被动接收环境反射光 | 主动发射近红外激光+接收 |
| 深度信息 | 单目/多目立体匹配(NN推断) | 直接飞行时间测量 |
| 暴雨性能 | **严重退化**(水珠散射) | 退化但可穿透(波长差异) |
| 浓雾性能 | **严重退化**(散射+衰减) | **也退化**(但检测距离衰减~25%) [硬数据: PMC研究] |
| 夜间性能 | 依赖环境光/车灯(低光噪点↑) | 不受光照影响(主动发射) |
| 强逆光 | **致盲**(动态范围饱和) | 不受影响 |
| 成本/单位 | ~$50(消费级CMOS) | $500-5000(取决于规格) |
| 分辨率 | 高(百万像素级) | 低(点云稀疏) |

[硬数据: Foresight Auto, ArcadianAI, PMC 2023 empirical study]

### 关键推论

**摄像头有3个硬物理限制**:
1. **暴雨/浓雾**: 水分子散射可见光，SNR(信噪比)降至安全阈值以下。NN无法从噪声中恢复信息——这不是算法问题，是信息论极限(Shannon定理)。[合理推断: 物理第一性原理]
2. **强逆光**: CCD/CMOS动态范围有限(~120dB最优)，太阳直射+阴影同时出现时必然丢失一端信息。[合理推断: 传感器物理]
3. **夜间无灯场景**: 农村公路无路灯时，摄像头的有效探测距离大幅缩短。[合理推断: 光学原理]

**但LiDAR也不完美**:
- 浓雾中LiDAR检测距离衰减约25% [硬数据: PMC研究]
- 大雪时LiDAR也会产生虚假点云 [硬数据: Gear Musk 2025]
- 关键差异: LiDAR在恶劣天气下**退化但仍可工作**，摄像头在某些场景下**接近失效**

### 监管要求

- **NHTSA 2025安全评估指南**: L4系统必须具备"感知系统冗余"——任何单一关键传感器故障时仍能安全运行。[硬数据: NHTSA Safety Assessment Guidelines 2025]
- **EU法规**: L4车辆必须通过"极端天气适应性测试"，包括暴雨、浓雾、逆光场景验证。[硬数据: EU regulatory guidance]

这对纯视觉方案构成**结构性挑战**: 8个摄像头都基于相同物理原理(可见光)，一个极端天气场景可能同时使所有8个摄像头退化——这不是"冗余"，因为失效模式是共模的(common-mode failure)。[合理推断: 工程可靠性原理]

## 计算路径: HW3→HW4→AI5

| 平台 | 算力(TOPS) | 时代 | 意义 |
|------|-----------|------|------|
| HW3 (FSD Computer) | ~72 | 2019-2023 | 第一代端到端NN |
| HW4 (AI4) | ~300-500 | 2023-present | v13/v14的基础 |
| AI5 (设计中) | ~3000-5000 | 2026年底量产 | 10x HW4 [硬数据: Tesla/TrendForce] |

**核心问题**: 10倍算力能否解决纯视觉的天花板？

**答案可能是"不能"**: 问题在输入端(摄像头物理限制)，不在计算端。更强的NN可以更好地利用现有像素信息，但无法创造不存在的信息——暴雨中被水珠散射的光子不会因为GPU更快而变清晰。[主观判断: 但这是有物理依据的判断]

**算力的真正价值可能在**:
- 更复杂的时序融合(利用前几秒的清晰帧预测当前被遮挡的信息)
- 世界模型(即使看不清也能基于物理规律推断物体位置)
- 但这些方法增加了latency和不确定性

## Waymo对照: 两种路径的结构性差异

| 维度 | Tesla FSD | Waymo |
|------|----------|-------|
| 传感器 | 8摄像头(纯视觉) | 摄像头+LiDAR+雷达(融合) |
| 运营模式 | L2+(监督)→计划L4 | **已运营L4**(6+城市, 450K+周rides) [硬数据: Waymo] |
| 覆盖范围 | 全美(有FSD订阅的车) | 限定城市的限定区域(高精地图) |
| 数据规模 | 60B+英里(影子模式) | ~100M+英里(但100% L4) |
| 扩张模式 | OTA升级→立即全车队 | 逐城市映射→缓慢扩展 |
| 监管状态 | 无L4批准 | 加州/亚利桑那/德州等L4许可 |
| 每英里成本 | 未知(无运营数据) | 估计$2-3+/英里(含安全员远程监控) |

**不判断谁会赢。但结构性差异值得注意**:
- Waymo在**精度和安全验证**上领先数年
- Tesla在**规模和成本**上有潜在优势(如果L4能实现)
- 两种路径不一定是零和——出行市场足够大 [主观判断]

## 结论: 天花板是否存在？

**诚实回答: 我们不知道。但可以界定问题的边界。**

**物理层面**: 纯视觉方案在极端天气场景存在理论天花板(信息论限制)。这不是工程问题(可以用更好的算法解决)，而是物理问题(传感器获取的信息本身不足)。

**但"天花板"不等于"不可用"**:
- 如果Tesla将ODD限制在"晴天/薄云城市道路"，纯视觉L4可能可行
- 问题是: 受限ODD的L4商业价值有多大？(仍然有限的Robotaxi运营)
- Waymo的方案也有ODD限制(高精地图覆盖区域)，只是限制来源不同

**应追踪的信号**:
1. **Tesla是否悄悄在任何车型增加非摄像头传感器** → 如果增加=内部承认天花板
2. **FSD在暴雨/夜间/浓雾场景的接管率趋势** → 如果这些场景接管率不降=天花板证据
3. **NHTSA是否对纯视觉方案给予L4豁免** → 监管方向决定商业化可行性
4. **v14+在极端场景的第三方安全测试数据** → 独立验证比Tesla自发布更可信

[主观判断: 纯视觉L4在受限ODD内可能实现(概率中等)，全域L4可能需要补充传感器(概率较高)。但这只是判断，不是确定性。]
