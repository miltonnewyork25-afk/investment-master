## Ch27: 极端压力测试

> 核心方法: 对Phase 2 SOTP Base $41.24和AI调整SOTP $48.94施加6种极端场景冲击, 每个场景量化到每股估值, 附概率权重。所有场景使用稀释后股数2.549B `[硬数据: DM-MKT-007 v1.0]`, 当前股价$137.65 `[硬数据: DM-MKT-001 v1.0]`。

### 27.1 AI寒冬场景 (概率15-20%)

**触发条件**: 企业AI支出在FY2027出现断崖式缩减 — GPT-6/Claude 4等通用模型解决了80%的"AI用例"但无需平台层, 企业AI预算从"战略投资"降级为"IT运营", AIP Bootcamp转化率从当前估计50-60%暴降至20%以下。

**量化推导:**

| 分部 | Phase 2 Bull FY2029 | AI寒冬 FY2029 | 差值 | 逻辑 |
|------|:---:|:---:|:---:|------|
| 美国商业 | $12.5B | $4.0B | -68% | AIP增速从+115%骤降至+20%, FY2027开始 `[合理推断: 参照SNOW从106%→36%的2年压缩]` |
| 美国政府 | $4.5B | $3.2B | -29% | 政府合同惯性更强, 但新合同冻结 `[合理推断: 政府IT合同通常有12-24月执行延迟]` |
| 国际政府 | $1.5B | $0.9B | -40% | NATO AI计划缩减 `[主观判断: 基于欧洲国防预算周期性]` |
| 国际商业 | $2.0B | $0.5B | -75% | 本就+2%增速, 寒冬中率先被砍 `[硬数据: DM-SEG-004 v1.0 国际商业+2%]` |
| **合计** | **$20.5B** | **$8.6B** | **-58%** | — |

**估值推算:**
- FY2029营收$8.6B, 寒冬中P/S压缩至15x (参照SNOW从95x→15x的先例) `[硬数据: Agent A Ch20.3 SNOW历史数据]`
- 企业价值 = $8.6B x 15 = $129B
- 折现至今(WACC 12%, 3年): $129B / 1.12^3 = $91.8B
- 减去净债务调整(+$7.2B现金): ($91.8B + $7.2B) / 2.549B = **$38.84/股**
- 从当前$137.65下跌 **-71.8%** `[合理推断: 三阶段推导 — 营收压缩→倍数压缩→折现]`

**与Phase 2一致性**: Phase 2 Bear Case $22.60假设更极端的倍数压缩(10-12x), 此处$38.84介于Bear($22.60)和Base($41.24)之间, 因为AI寒冬不等于公司消亡 — PLTR仍有$7.2B现金和政府合同底仓。

**So What**: AI寒冬场景下PLTR跌至$38-39, 接近SOTP Base $41.24。这恰好验证了Phase 2的Base估值是合理的"正常化"锚点 — 当前$137.65中有$99的溢价建立在AI叙事之上。

### 27.2 竞争挤压场景 (概率20-25%)

**触发条件**: Databricks H2 2026 IPO成功(估值$134B→上市后$180B+) + Microsoft Fabric企业版全面铺开 + AWS Bedrock Agents进入"操作系统"层。PLTR的"唯一性"溢价被直接挑战。

**量化推导:**

| 指标 | 当前值 | 竞争挤压后 | 逻辑 |
|------|:---:|:---:|------|
| 美商增速 | +137% (Q4) | +30-40% | Databricks从15,000+客户基数切入PLTR的960客户 `[硬数据: DM-GRW-002 v1.0; DM-COMP-002 v1.0]` |
| NDR | 139% | 110% | 客户在PLTR和竞品间分配预算, 扩展放缓 `[合理推断: SNOW NDR从178%→127%的先例]` |
| 客户流失率 | <5% (估计) | 10% | 中低端客户被Databricks性价比吸引 `[主观判断: Databricks定价通常低于PLTR 30-40%]` |
| P/S | 72x | 30-35x | 向Databricks IPO估值(~28x)收敛 `[合理推断: Agent A Ch22反证3的Databricks估值比较]` |

**估值推算:**
- FY2026营收按竞争挤压调整: $6.0B (vs 指引$7.19B, -17%) `[合理推断: 增速从61%降至34%]`
- P/S取32x (PLTR与Databricks的中值): $6.0B x 32 / 2.549B = **$75.32/股**
- 从当前$137.65下跌 **-45.3%**

**与SOTP关系**: $75.32远高于SOTP Base $41.24, 因为竞争挤压不改变PLTR的存量收入, 只是增速放缓。这意味着即使在竞争压力下, PLTR仍有显著高于"基本面锚"的估值空间 — 但前提是增速不低于30%。

**So What**: 竞争挤压是概率最高的负面场景(20-25%)。Databricks IPO是确定性事件(已宣布H2 2026), 其估值将直接成为PLTR的对标锚。$75是这一场景的合理底部。

### 27.3 治理危机场景 (概率10-15%)

**触发条件**: CEO持续减持引发信任危机(Karp从141M股降至6.4M股, 0.27%流通股) `[硬数据: Agent A Ch20.2 反证2]` + 散户恐慌抛售 + 纽约养老基金($9亿)正式质询升级为诉讼。

**量化推导:**
- 散户占比从59%快速降至30%: 约30%的流通股(~$97B市值)需要找到新买家 `[合理推断: 59%→30%=29个百分点 x $323.9B=~$94B]`
- 机构不接盘原因: Class F治理折价15-20% + ESG筛选排除 `[主观判断: 基于Agent A Ch22反证4的治理折价分析]`
- 流动性真空期: 3个月内股价回落至$60-80
- 治理折价永久化: 在当前P/S基础上折价15-20%

**估值推算:**
- 短期冲击底: 当前$137.65 x (1-0.42) = **$79.84** (散户退出引发的42%回撤) `[合理推断: 散户占比42%全部出清的极端假设]`
- 稳态治理折价: AI调整SOTP $48.94 x (1-0.175) = **$40.38** `[合理推断: SOTP基础上叠加17.5%治理折价]`
- 3个月急跌底: $60-80区间, 取中值**$70**

**与Phase 2一致性**: 治理危机的稳态估值$40.38与SOTP Base $41.24高度吻合, 说明Phase 2的Base Case已隐含了治理折价。但3个月内的流动性冲击可能暂时打穿$41, 制造$60-70的买入窗口。

**So What**: 治理危机的核心不是估值改变, 而是流动性改变。散户占比59%本身就是一个脆弱性因子 — 当CEO的行动(大规模减持)与言语(极度看好)矛盾时, 散户的信仰最先崩塌。

### 27.4 多重冲击叠加 (概率5-8%)

**触发条件**: AI寒冬(27.1) + 治理危机(27.3) + DOGE政策反转(政府合同审查导致$10B陆军合同暂停) 三重叠加。

**量化推导:**

| 冲击 | 单独影响 | 叠加权重 | 叠加后 |
|------|:---:|:---:|------|
| AI寒冬 → 营收 | -58% (FY2029) | 100% | 营收$8.6B (FY2029) |
| 治理危机 → 流动性 | -42% (3个月) | 50% (部分重叠) | 额外-21%流动性折价 |
| DOGE反转 → 政府 | -$2.0B (政府线) | 100% | 营收再减$2.0B → $6.6B |
| **累计** | — | — | **FY2029营收$6.6B, P/S 8-10x** |

**估值推算:**
- 企业价值 = $6.6B x 9 (P/S中值) = $59.4B
- 折现至今(WACC 14%, 3年, 风险溢价上升): $59.4B / 1.14^3 = $40.1B
- 加回现金$7.2B, 减治理折价17.5%: ($40.1B + $7.2B) x 0.825 / 2.549B = **$15.31/股**
- 从当前$137.65下跌 **-88.9%** `[合理推断: 四步推导 — 营收x倍数→折现→现金加回→治理折价]`

**与Phase 2极端底一致性检查:**
- Phase 2极端底$11.73 vs 本场景$15.31 — **差异$3.58 (30.5%)**
- 差异来源: Phase 2极端底假设P/S降至5-6x(彻底丧失增长溢价), 而本场景仍给9x(保留政府合同底仓价值)
- 结论: $11.73需要PLTR同时失去增长溢价+政府合同底仓, 概率进一步降低至3-5%, Phase 2的15%概率可能偏高 `[合理推断: 三重叠加本身5-8%, 进一步恶化至极端底需额外条件]`

**So What**: 多重冲击是尾部风险中的尾部。$15.31而非$11.73更可能是真实底部, 因为$7.2B现金(约$2.82/股)和政府合同存量提供了硬底。Phase 2的极端底$11.73需要L5范式修正(投资逻辑根本改变), 不应作为概率场景纳入, 而应视为"黑天鹅保险"。

## Ch28: CQ验证

> 核心方法: 逐一审视7个CQ, 评估Phase 1-3给出的答案是否经受住Phase 4对抗审查, 并标注需要修正的判断。

### 28.1 CQ1: 72倍P/S的估值溢价能否被持续高增长证明合理?

**Phase 1-3答案总结**: 72x P/S = 50%锁定溢价 + 50%AIP期权定价。4年近乎完美执行是隐含假设, FY2029时P/S才降至19x。

**证据强度**: 中偏弱

**反证检查**:
- Agent A的可得性偏误分析(Ch20.3)提供了关键反证: SNOW从95x→15x(-84%), ZM从90x→5x(-94%), PLTR自身从45x→5x(-89%) `[硬数据: Agent A Ch20.3历史基准率表]`
- Phase 2给出的"4年完美执行"假设在历史上几乎没有先例 — 唯一接近的案例是Amazon(2013-2017), 但AMZN有电商现金流支撑 `[合理推断: SaaS公司在$5B ARR后维持>50%增速超过3年的案例为零]`
- 72x P/S倒推FY2028需$15B+营收, 即FY2026-2028 CAGR>45%, 目前指引(FY2026 +61%)虽然支持第一年, 但此后减速不可避免 `[合理推断: 基于SaaS增速自然衰减曲线]`

**置信度**: 35% — 72x P/S能被证明合理的概率仅35% `[合理推断: 基于历史基准率, 没有先例支持$5B+ ARR公司维持72x P/S超过18个月]`

**Phase 4验证后调整**: Phase 1-3的"50%锁定+50%期权"框架是合理的分析工具, 但结论应更偏空: 72x P/S更可能是65%叙事溢价+35%基本面支撑(而非Phase 3给出的64.4%叙事/35.6%基本面, 两者高度一致)。**无需修正, Phase 3的64.4%纯叙事溢价判断经受住了验证。**

### 28.2 CQ2: AIP能否将PLTR转型为企业AI操作系统平台?

**Phase 1-3答案总结**: 五不变量5/5通过, L2.5/S3.5行业最高。Agentic AI三情景: 成功60%/通用替代30%/泡沫10%。

**证据强度**: 中

**反证检查**:
- "操作系统"类比可能过度乐观 — Windows/iOS的平台地位建立在数十亿终端用户+开发者生态之上, PLTR仅954客户 `[硬数据: DM-GRW-007 v1.0]`
- Databricks(15,000+客户)在"数据操作系统"赛道上有更广的客户基础, 可能更适合"操作系统"类比 `[硬数据: DM-COMP-002 v1.0]`
- AIP Bootcamp转化率未公开披露, 成功案例可能存在选择偏差 `[主观判断: 企业软件PoC到合同转化通常40-60%]`

**置信度**: 50% — AIP成为企业AI操作系统的概率约50%, 低于Phase 3给出的60% `[合理推断: 960客户vs竞品15,000+, 网络效应仅3.5/10, 操作系统地位需要更大客户基数]`

**Phase 4验证后调整**: Phase 3的Agentic AI成功概率60%应下调至**50%**, 通用替代概率从30%上调至**35%**, 泡沫维持10%。调整理由: 客户基数过窄(960 vs 平台所需的5,000+)是操作系统路径的结构性障碍, Phase 1-3未充分考虑这一约束。

### 28.3 CQ3: DOGE+国防重构对政府收入是净正还是净负?

**Phase 1-3答案总结**: DOGE净正效应65%正面。政府护城河7-10年。

**证据强度**: 中

**反证检查**:
