## 4.1 品牌资产估值 (~2,000字) ⭐ 消费品特有模块

### Interbrand 方法论应用

```
品牌资产估值框架:

Interbrand估值模型:
品牌价值 = 品牌收益 × 品牌强度乘数

其中:
├─ 品牌收益 = 经济利润中归属于品牌的部分
├─ 品牌强度 = 10个维度的综合评分 (0-100)
└─ 乘数 = f(品牌强度) → 10-25x
```

├─ 终端FCF = $14.2B × (1+2.5%) = $14.6B

├─ 终端价值现值 = $292B / (1.075)^5 = $204B
└─ 终端价值占比: 79%

├─ 减: 净债务 $25B
├─ 股权价值 = $232.7B
├─ 股数 = 2.35B
└─ 每股价值 = $99

## 4.5 估值桥梁与锚定检查 ⭐ v18.3强制

### 估值差距分析

```
估值方法对比:

┌─────────────────────────────────────────────────────────────────┐
│  估值方法           每股价值      相对当前股价($172)           │
├─────────────────────────────────────────────────────────────────┤
│  Reverse DCF        $172         0% (当前价格隐含)             │
│  Forward DCF        $103         -40%                          │
│  SOTP               $86          -50%                          │
│  分析师目标价       $175-185     +2% to +8%                    │
│  当前股价           $172         --                            │
└─────────────────────────────────────────────────────────────────┘

⚠️ 关键发现:
├─ DCF/SOTP估值 << 当前股价
├─ 分析师目标价 ≈ 当前股价
├─ 差距需要解释
└─ 这是"估值锚定检查"的核心
```

### 估值溢价因素分析

| 溢价因素 | 溢价幅度 | 理由 | 可持续性 |
|---------|---------|------|---------|
| **防御性溢价** | +20% | 经济衰退中的避风港 | 🟢 高 |
| **股息溢价** | +15% | 稳定股息+股息增长历史 | 🟢 高 |
| **品牌溢价** | +15% | 品牌资产无形价值 | 🟡 中 |
| **管理层溢价** | +10% | 执行记录良好 | 🟡 中 |
| **流动性溢价** | +5% | 大市值+高流动性 | 🟢 高 |
| **合计溢价** | **+65%** | -- | -- |

### 估值桥梁计算

```
估值桥梁:

起点: SOTP概率加权 = $86

调整因子:
├─ 防御性溢价: $86 × 1.20 = $103
├─ 股息溢价: $103 × 1.15 = $118
├─ 品牌溢价: $118 × 1.15 = $136
├─ 管理层溢价: $136 × 1.10 = $150
├─ 流动性溢价: $150 × 1.05 = $157

调整后目标价: $157 (相对当前$172，下跌9%)

与SOTP差距: ($157 - $86) / $86 = +83%

⚠️ 差距仍然较大，但可以通过以下解释:
├─ P&G是防御性蓝筹的典范
├─ 机构投资者必配
├─ 在不确定环境中的避险价值
├─ 但这也意味着上行空间有限
```

### 估值锚定结论

| 估值方法 | 目标价 | 与当前差距 | 评估 |
|---------|--------|-----------|------|
| **SOTP概率加权** | $86 | -50% | 纯财务估值 |
| **调整后目标价** | $157 | -9% | 含溢价因素 |
| **分析师共识** | $180 | +5% | 市场预期 |
| **当前股价** | $172 | -- | -- |

**结论**: P&G当前股价包含约65%的"质量溢价"。这个溢价在不确定的宏观环境中是合理的，但也意味着：
1. 上行空间有限（分析师目标价仅+5%）
2. 如果溢价收缩，下行风险显著
3. 投资者买的是"安全"而非"增长"
