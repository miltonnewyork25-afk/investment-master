### 数据获取顺序

```
1. FMP API
   ├── /stable/profile → 公司基础信息
   ├── /stable/income-statement-ttm → 损益表
   ├── /stable/balance-sheet-statement-ttm → 资产负债表
   ├── /stable/cash-flow-statement-ttm → 现金流
   ├── /stable/financial-scores → Z-Score, F-Score
   ├── /stable/insider-trading/statistics → 内部人交易
   └── /api/v3/quote/{peers} → 同业对比

2. 100Baggers API
   └── POST /api/generate-summary → 先行指标（排除雷达图）
```
