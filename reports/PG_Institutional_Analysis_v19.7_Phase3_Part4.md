**强度评分**: 2/10 (很弱)

**分析**: 消费品行业几乎没有网络效应。消费者买Tide不会因为其他人买Tide而获得更多价值。

```
P&G网络效应分析:

直接网络效应: ❌ 不存在
├─ 我的洗衣液选择不影响你
└─ 没有用户间直接互动

间接网络效应: ⭐ 微弱
├─ 更多消费者 → 更多零售商铺货
├─ 更多铺货 → 更容易购买
└─ 但这个效应很弱

平台网络效应: ❌ 不适用

社交网络效应: ⭐⭐ 有限
├─ SK-II送礼文化 (我送你→你送别人)
├─ 品牌口碑传播
└─ 但这更多是品牌效应而非网络效应
```

## 3.7 PVM 增长质量分析 (~2,000字) ⭐ 核心模块

### PVM 框架导论

```
PVM增长拆解框架:

收入增长 = Price (价格) + Volume (销量) + Mix (结构)

健康增长模式:
├─ Price-Led: 提价成功，Volume稳定
├─ Mix-Led: 高端化成功，毛利↑
└─ Volume-Led: 市场扩张，份额↑

不健康增长模式:
├─ Hollow: Price↑ 但 Volume↓ (空心化)
├─ Promo-Dependent: 促销>60%
└─ Mix-Down: 被迫下沉
```

### PVM诊断与Kill Switch

**空心化增长警告 - Baby Care** 🔴:

```
Baby Care空心化分析:

数据:
├─ 有机增长: -5%
├─ Price贡献: +3%
├─ Volume贡献: -8%
└─ 这是典型的空心化模式

诊断:
├─ 提价成功 (Price +3%)
├─ 但销量崩塌 (Volume -8%)
├─ 消费者在流失
├─ 要么切换到私牌
├─ 要么切换到Huggies
└─ 提价无法弥补销量损失

Kill Switch触发条件:
├─ 如果Volume连续4季度<-5%
├─ 且市场份额连续下滑>2pp
└─ 需要战略性重新评估品类
```

**关键洞察**:

> **洞察**: P&G整体PVM健康，但Baby Care品类出现严重空心化。这不是定价策略问题，而是结构性问题（出生率下滑+私牌竞争）。P&G需要考虑：(1) 加速中国等增长市场投资；(2) 北美聚焦高端、放弃低端；(3) 长期评估品类剥离可能性。
