```python

正常: λ = 1 (理性)
实际: λ = 2.5 (损失厌恶严重)
```

**校正策略 - 动态止盈止损**:
```python
止盈价 = 成本 × (1 + 目标回报/λ)  # 推迟止盈
止损价 = 成本 × (1 - 最大亏损×λ)  # 提前止损
```

Tesla示例:
- 成本: $449
- 目标回报: 30%
- 最大亏损: 15%
- λ校正后:
  - 止盈价: $449×(1+30%/2.5) = **$503** (vs 原始$583)
