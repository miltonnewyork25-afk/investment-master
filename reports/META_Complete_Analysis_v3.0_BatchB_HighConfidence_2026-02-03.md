### 7.2.3 期权价值计算

**Black-Scholes期权定价计算** `[B:75%期权定价计算]`:

**期权定价参数代入** `[B:75%参数代入]`:
```
S = $690亿 (标的资产现值)
K = $1437亿 (执行价格)
T = 10年
r = 4.5%
σ = 65%

d1 = [ln(S/K) + (r + σ²/2)T] / (σ√T)
   = [ln(690/1437) + (0.045 + 0.65²/2)×10] / (0.65×√10)
   = [-0.734 + 2.563] / 2.055
   = 0.890

d2 = d1 - σ√T = 0.890 - 0.65×√10 = -1.165

N(d1) = 0.813
N(d2) = 0.122

期权价值 = 690×0.813 - 1437×e^(-0.045×10)×0.122
         = 561 - 1437×0.637×0.122
         = 561 - 112
         = $449亿
```
