## 半导体专家工作原则

### 🔬 **技术驱动原则**
```

```

```

- 区分AI硬件销售 vs AI平台生态价值
- 评估传统芯片厂商的AI转型能力
- 识别AI原生公司对传统巨头的威胁
- 量化AI需求的结构性 vs 周期性成分

### 🌍 **地缘风险整合原则**
```
技术竞争常态化：
- 出口管制是新常态，非短期扰动

#### 2. **PMSI Sentiment Index Calculator**
**集成位置**: `frameworks/v10.0-prediction-market/algorithms/pmsi_algorithm.py`
**功能**: 半导体专用预测市场情绪指数

**核心算法**:
```python
# PMSI-半导体计算公式
PMSI = (
    地缘政治模块 × 40% +
    技术竞争模块 × 30% +
    需求周期模块 × 20% +
    供应链模块 × 10%
)

地缘政治模块 = (1-台海冲突概率) × 0.6 + (1-制裁概率) × 0.4
技术竞争模块 = 技术领先概率 × 0.8 + (1-竞争威胁概率) × 0.2
需求周期模块 = AI需求增长概率 × 0.6 + 数据中心扩张概率 × 0.4
供应链模块 = (1-供应中断概率) × 0.7 + 产能利用率概率 × 0.3
```

**输出要求**:
- PMSI指数值(0-100)及其历史分位数
- 四模块分解得分和权重说明
- 与股价的历史相关性分析(≥90天数据)
- 投资信号生成(买入/持有/卖出)

#### 3. **PPDA Algorithm Engine**
**算法文件**: `frameworks/v10.0-prediction-market/algorithms/ppda_algorithm.js`
**功能**: 概率-价格背离分析的核心计算引擎

**分析流程**:
```javascript
1. 市场隐含概率提取:
   - 从当前股价反推市场对各风险的隐含定价
   - 基于历史价格敏感性计算影响系数
   - 考虑估值水平和风险溢价的影响

2. 背离度量化计算:
   背离度 = (市场隐含概率 - Polymarket实际概率) / 实际概率

3. 套利信号生成:
   - 背离度>20%: 强信号
   - 背离度10-20%: 中等信号
   - 背离度<10%: 弱信号

4. 置信度评估:
   - 基于预测市场数据质量
   - 考虑历史背离修复的时间和幅度
   - 结合市场流动性和参与者结构
```
