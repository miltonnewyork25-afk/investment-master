# 投资者分析边界指南 v1.0

> 逆向工程投资者行为是有价值的，但必须清晰区分事实与推测。

**版本**: 1.0  
**创建日期**: 2026-01-29  
**来源**: 台积电报告升级复盘 (91.75 → 96 → 98.5)

---

## 一、核心原则

### 1.1 数据诚信原则

```
┌─────────────────────────────────────────────────────┐
│                 数据诚信三原则                       │
├─────────────────────────────────────────────────────┤
│  1. 有证据 → 正常陈述                                │
│  2. 有框架推测 → 标注[分析推测]                      │
│  3. 纯猜测 → 跳过或明确说明                          │
└─────────────────────────────────────────────────────┘
```

### 1.2 为什么重要？

- **建立信任**: 承认不知道比假装知道更专业
- **避免误导**: 读者知道哪些是事实、哪些是推测
- **法律风险**: 投资报告需要区分事实与推测
- **自我约束**: 写作时更谨慎，避免夸大

## 二、可以做的分析（有数据支撑）

### 2.1 基于13F的持仓变动分析 ✅

**数据来源**: SEC 13F Filing  
**数据频率**: 季度（滞后约45天）  
**数据限制**: 仅反映季末持仓快照

**可做分析**:

| 分析类型 | 例子 | 写法 |
|---------|------|------|
| 增减仓幅度 | Q4增持150万股 | "根据13F披露，Baillie Gifford Q4增持150万股" |
| 持仓占比 | 占该基金2.1% | "台积电占该基金AUM的2.1%" |
| 历史持仓模式 | 连续6个季度增持 | "该基金已连续6个季度增持台积电" |
| 新建仓vs加仓 | 新建仓位 | "该基金本季度新建立台积电仓位" |

**代码示例**:
```markdown
根据SEC 13F披露（截至2025年12月31日）：
- Baillie Gifford: 增持150万股 (+8.2%)
- Fisher Investments: 增持80万股 (+12.1%)
- Primecap: 持仓不变
```

### 2.2 基于公开访谈的投资风格总结 ✅

**数据来源**: 公开访谈、演讲、股东信  
**数据频率**: 不定期  
**数据限制**: 可能有时效性

**可做分析**:

| 分析类型 | 例子 | 写法 |
|---------|------|------|
| 投资风格 | 长期成长型 | "Baillie Gifford以长期成长投资著称（见其公开投资哲学）" |
| 公开表态 | 看好AI | "该基金在2025年致股东信中明确表示看好AI基础设施" |
| 历史偏好 | 偏好科技 | "历史上该基金科技配置占比达35%" |

**引用规范**:
```markdown
Baillie Gifford在2025年致股东信中表示：
"我们相信AI将是本世纪最重要的技术变革，
而半导体基础设施是这一变革的核心。"
```

### 2.3 历史预测准确率统计 ✅

**数据来源**: 历史公开预测 vs 实际结果  
**数据限制**: 需要有可追溯的公开预测

**可做分析**:

| 分析类型 | 例子 | 写法 |
|---------|------|------|
| 预测准确率 | 70%准确 | "该分析师过去5年的预测准确率为70%（基于可追溯的公开预测）" |
| 特定领域准确率 | 半导体预测较准 | "在半导体领域，该机构的预测准确率高于市场平均" |

### 2.4 持仓相关性分析 ✅

**数据来源**: 13F + 公司关系数据  
**数据限制**: 仅反映静态持仓，不反映策略

**可做分析**:

| 分析类型 | 例子 | 写法 |
|---------|------|------|
| 同板块持仓 | 重仓半导体 | "该基金同时持有ASML、台积电、英伟达，显示对半导体板块的偏好" |
| 产业链布局 | 上下游覆盖 | "该基金持仓覆盖AI产业链：芯片（英伟达）、制造（台积电）、应用（Meta）" |

## 三、需要标注推测的分析

### 3.1 投资者的具体买入理由 ⚠️

**除非有直接引用，否则是推测**

**错误写法**:
```markdown
Baillie Gifford看好台积电的AI需求前景。
```

**正确写法**:
```markdown
[分析推测] 基于Baillie Gifford的长期成长投资风格
和其公开表达的AI乐观态度，推测其增持台积电
可能与AI需求前景相关。
```

### 3.2 投资者的目标价 ⚠️

**除非有公开表态，否则是推测**

**错误写法**:
```markdown
该基金的目标价可能在$280左右。
```

**正确写法**:
```markdown
[估算] 基于该基金历史持仓模式（通常在估值达到
25x P/E时开始减持），推算其隐含目标价可能在
$250-280区间。此为分析推测，非官方目标价。
```

### 3.3 投资者的投资期限 ⚠️

**错误写法**:
```markdown
该基金通常持有3-5年。
```

**正确写法**:
```markdown
[分析推测] 基于该基金过去10年的持仓周期数据，
其核心持仓平均持有期为3-5年。台积电作为其
重仓股，推测可能遵循类似的持有期限。
```

### 3.4 标注模板

```markdown
推测标注模板：

[分析推测]
基于：[数据来源/分析框架]
推测：[推测内容]
信心度：[高/中/低]
注意：此为分析推测，非官方信息。

---

[估算]
方法：[估算方法]
结果：[估算结果]
假设：[关键假设]
注意：此为分析估算，存在不确定性。
```

## 四、应该跳过的分析

### 4.1 投资者的下一步动作预测 ❌

**原因**: 无法验证，误导性强

**不要写**:
```markdown
预计该基金下季度将继续增持。
```

**替代方案**: 跳过，或改为条件式陈述
```markdown
如果台积电估值保持在当前水平，
基于该基金的历史行为模式，
增持的可能性较高（非预测，仅为情景分析）。
```

### 4.2 投资者之间的私下交流 ❌

**原因**: 无数据来源，纯猜测

**不要写**:
```markdown
据悉，多家机构投资者私下交流后
达成了对台积电的看多共识。
```

### 4.3 投资者的具体交易价格 ❌

**原因**: 13F只披露季末持仓，不披露交易细节

**不要写**:
```markdown
该基金在$180左右大量买入。
```

**替代方案**:
```markdown
该基金Q4增持期间，台积电股价区间为$175-210。
具体买入价格未知。
```

### 4.4 投资者的内部讨论/决策过程 ❌

**原因**: 无数据来源

**不要写**:
```markdown
该基金内部可能对台积电的地缘风险存在分歧。
```

## 五、数据来源标注规范

### 5.1 13F数据

```markdown
数据来源: SEC 13F Filing
报告期: 2025年Q4（截至2025年12月31日）
披露日期: 2026年2月14日
数据限制: 
- 仅反映季末持仓快照
- 不包含空头头寸
- 不包含小于$200M AUM的机构
- 滞后约45天
```

### 5.2 公开访谈

```markdown
数据来源: [媒体名称/活动名称]
日期: [YYYY-MM-DD]
链接: [URL]
原文引用: "..."
```

### 5.3 研报/分析

```markdown
数据来源: [机构名称] 研究报告
报告标题: [标题]
发布日期: [YYYY-MM-DD]
分析师: [姓名]
```

### 5.4 估算/推测

```markdown
数据来源: 分析师估算
方法: [估算方法描述]
假设: [关键假设列表]
不确定性: [高/中/低]
```

## 六、完整示例

### 6.1 正确的投资者分析章节

```markdown
## 7. 投资者信号分析

### 7.1 顶级机构持仓变化

数据来源: SEC 13F Filing（截至2025年12月31日）

╔══════════════════════════════════════════════════╗
║             顶级机构持仓信号 (Q4 2025)            ║
╠══════════════════════════════════════════════════╣
║  机构              变化        占比     信号     ║
║  ─────────────────────────────────────────────   ║
║  Baillie Gifford  +150万股    2.1%     🟢       ║
║  Fisher Inv.      +80万股     1.8%     🟢       ║
║  Primecap         持平        3.2%     🟡       ║
║  Capital Group    +50万股     4.5%     🟢       ║
╠══════════════════════════════════════════════════╣
║  综合信号: 4/4看多或持有 = 强烈看多               ║
╚══════════════════════════════════════════════════╝

### 7.2 投资风格解读

[分析推测] 基于各机构的公开投资哲学：
- Baillie Gifford: 长期成长型，历史上重仓科技创新公司
- Fisher: 价值成长型，偏好护城河强的龙头
- Capital: 长期价值型，换手率极低

推测增持原因：
- AI需求带来的长期成长前景
- 技术领先带来的护城河加深
- 估值相对历史仍有吸引力

注意：以上为基于公开信息的分析推测，非官方表态。

### 7.3 数据诚信声明

- 持仓数据来源: SEC 13F Filing，滞后约45天
- 投资风格描述: 基于各机构公开材料
- 增持原因分析: [分析推测]，非官方信息
- 本章节不对投资者未来行为做预测
```
