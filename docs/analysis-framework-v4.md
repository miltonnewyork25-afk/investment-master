# 半导体供应链投资分析框架 4.0

> 基于历史周期回测优化的完整分析框架

## 核心等式

**设备商收入 ≈ 下游晶圆厂CAPEX**

```
终端需求 → 芯片需求 → 产能利用率 → 晶圆厂盈利 → CAPEX决策 → 设备商订单 → 设备商收入
```

## 一、长鞭效应与周期方法论

### 1.1 长鞭效应（Bullwhip Effect）

终端需求的微小波动，经过供应链层层放大，到达最上游时产生巨大振幅：

| 供应链层级 | 波动放大倍数 |
|-----------|------------|
| 终端消费者需求 | ±10% |
| OEM/品牌商订单 | ±20% |
| 芯片客户采购 | ±40% |
| 晶圆厂产能规划 | ±60% |
| **设备商收入** | **±100%+** |

### 1.2 周期股估值悖论

| 周期位置 | 盈利 | PE | 正确操作 |
|----------|------|-----|---------|
| **顶峰期** | 最高 | 最低 | **卖出** |
| 下行期 | 下滑 | 上升 | 观望 |
| **谷底期** | 最低/亏损 | 极高/N/A | **买入** |
| 上行期 | 恢复 | 下降 | 持有 |

**核心洞察**：周期股要在"PE最高或亏损时买入，PE最低时卖出"。

### 1.3 三大核心原则

1. **看预期，不看现实**：股价反映未来6-12个月预期
2. **逆向思维**：极度乐观=见顶，极度悲观=见底
3. **关注边际变化**：不看绝对水平，看变化方向

**布局顺序**（上行周期）：
1. Memory价格企稳 → 买入MU/SK
2. 利用率回升 → 买入TSM
3. 设备订单回暖 → 买入LRCX/AMAT

**撤退顺序**（下行周期）：
1. 设备订单见顶 → 先卖设备商
2. CAPEX下调 → 减持TSM
3. Memory价格见顶 → 减持MU/SK
