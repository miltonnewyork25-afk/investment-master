# 三层置信度标注系统 v1.0

> 替代旧的 `[置信度: 85%]` 装饰性数字。每个数据点/判断必须归入三层之一，标注推理链。

---

## 设计哲学

**旧系统**: `[置信度: 85%]` — 不知道85%怎么来的，无法校准，纯装饰
**新系统**: 三层分类 — 读者一眼看出"这是硬数据"还是"这是分析师拍脑袋"

---

## 三层定义

### 层级1: [硬数据]

直接来自一级/二级来源的可验证事实。

**标注格式**: `[硬数据: 来源名, 日期]`

**判定标准**:
- 数据来自 SEC 文件 / 财报 / 官方披露 / 权威数据库
- 第三方可独立验证
- 无推导、无估算、无主观成分

**示例**:
1. "FY2025营收$254.2B" `[硬数据: COST 10-K FY2025]`
2. "会员续费率92.9%" `[硬数据: COST Q1 2026 Earnings Call]`
3. "门店数量897家" `[硬数据: COST IR Page, 2026-01]`
4. "P/E 54.3x" `[硬数据: Yahoo Finance, 2026-02-06]`
5. "机构持仓占比72%" `[硬数据: Nasdaq, 2026-01-31]`
6. "FSD获批概率62%" `[硬数据: Polymarket, 2026-02-05]`
7. "股息率0.51%" `[硬数据: COST IR, 2026-02-01]`

### 层级2: [合理推断]

基于硬数据的逻辑推导。**必须标注推理链**。

**标注格式**: `[合理推断: 推理链简述]`

**判定标准**:
- 基于≥1条硬数据
- 推理逻辑可审查（读者能追溯推导过程）
- 合理人会得出相同结论

**示例**:
1. "Kirkland占比预计2027达30%" `[合理推断: 当前28%+年增1-2pp趋势, 基于FY2023-2025数据]`
2. "会员费提价将贡献约$1.8B增量" `[合理推断: 73M会员×$5提价×50%净增=~$1.8B]`
3. "中国市场单店年销≥$300M" `[合理推断: 基于前7家门店平均数据外推]`
4. "AI对仓储运营可节省3-5%成本" `[合理推断: 同行(Walmart)已实现3%+COST仓储标准化优势]`
5. "新建门店IRR约25-30%" `[合理推断: 基于$30M建店成本+$250M年销+4%营业利润率]`
6. "竞争者追平Kirkland品质需3-5年" `[合理推断: 基于供应链深度+规模效应差距]`
7. "毛利率将维持在12.5-13%区间" `[合理推断: 近5年波动区间12.2-13.1%+管理层指引]`

### 层级3: [主观判断]

分析师观点、定性评估、市场情绪解读。

**标注格式**: `[主观判断: 判断依据简述]`

**判定标准**:
- 涉及定性评估（"优秀"/"强"/"弱"）
- 涉及未来预测（非趋势外推）
- 涉及比较判断（"最好的"/"独特的"）
- 合理人可能不同意

**示例**:
1. "管理层执行力优秀" `[主观判断: 基于20年一致性记录+同行对比]`
2. "品牌忠诚度是零售业最强" `[主观判断: NPS 79 vs 行业均值32, 但NPS方法论有局限]`
3. "估值溢价合理" `[主观判断: 护城河持久性+增长可见性支撑, 但50x PE历史少见]`
4. "中国扩张将成功" `[主观判断: 基于日本/韩国先例, 但中国零售环境独特]`
5. "Kirkland不会蚕食供应商关系" `[主观判断: 管理层表态+历史模式, 但利益冲突客观存在]`
6. "该股适合长期持有" `[主观判断: 基于护城河持久性评估, 3年+时间框架]`

## 使用规则

### 密度要求

| 报告层级 | 最低标注密度 | 三层比例指导 |
|---------|:---:|------|
| Tier 3 | ≥15个/万字符 | 硬数据≥40%, 合理推断≤40%, 主观判断≤20% |
| Tier 2 | ≥10个/万字符 | 硬数据≥30%, 其余不限 |
| Tier 1 | 不强制 | 关键数字标注即可 |

### 禁止事项

- 禁止不标注层级的数据点（每个数字/判断必须归类）
- 禁止"合理推断"不附推理链
- 禁止"主观判断"伪装为"硬数据"
- 禁止使用旧格式 `[置信度: XX%]`

### 与旧标注系统的兼容

| 旧标注 | 新标注 | 说明 |
|-------|-------|------|
| `[A: 来源, 日期]` | `[硬数据: 来源, 日期]` | 一级数据→硬数据 |
| `[B: 来源, 日期]` | `[硬数据: 来源, 日期]` | 二级权威数据→硬数据 |
| `[P: 来源, 日期]` | `[硬数据: 来源, 日期]` | 预测市场数据→硬数据(概率本身是事实) |
| `[E: 推导说明]` | `[合理推断: 推导说明]` | 估算→合理推断 |
| `[置信度: XX%]` | **废弃** | 用三层替代 |
