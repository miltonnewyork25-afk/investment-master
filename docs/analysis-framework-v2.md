# 半导体供应链投资分析框架 2.0

## 核心理念

**核心等式**：设备商收入 ≈ 下游晶圆厂CAPEX

投资半导体设备商的关键在于理解CAPEX传导链：
```
终端需求 → 芯片需求 → 产能利用率 → 晶圆厂盈利 → CAPEX决策 → 设备商订单 → 设备商收入
```

---

### 第三层：公司财务深度分析（权重25%）

#### 3.1 资产负债表健康度

| 指标 | 优秀标准 | 警戒线 |
|------|----------|--------|
| 净负债/EBITDA | <1.0x | >3.0x |
| 流动比率 | >2.0 | <1.5 |
| 应收账款周转天数 | <60天 | >90天 |
| 存货周转天数 | <120天 | >180天 |

#### 3.2 现金流质量分析

| 指标 | 计算方式 | 健康标准 |
|------|----------|----------|
| OCF/NI | 经营现金流/净利润 | >1.0（盈利质量高） |
| FCF Margin | 自由现金流/收入 | >15% |
| FCF Conversion | FCF/EBITDA | >50% |
| 应计利润比率 | (NI-OCF)/总资产 | <5%（低操纵风险） |

#### 3.3 盈利能力深度

| 指标 | 优秀标准 | 意义 |
|------|----------|------|
| 毛利率 | >45% | 产品定价权 |
| 营业利润率 | >25% | 运营效率 |
| ROIC | >20% | 资本效率 |
| 增量ROIC | >25% | 边际资本回报 |

#### 3.4 收入质量分析

| 指标 | 观察重点 |
|------|----------|
| 递延收入变化 | 环比增长 → 订单健康 |
| 客户集中度 | Top 3客户占比<50% |
| 区域分布 | 单一市场依赖度<40% |
| 业务结构 | 服务收入占比提升 → 稳定性↑ |

### 第四层：估值与风险量化（权重20%）

#### 4.1 估值因子

| 指标 | 计算方式 | 合理区间 |
|------|----------|----------|
| P/E TTM | 市值/净利润 | 15-25x（周期中枢） |
| P/E NTM | 市值/预期净利润 | 根据增速调整 |
| EV/EBITDA | 企业价值/EBITDA | 10-15x |
| P/FCF | 市值/自由现金流 | 15-20x |
| PEG | P/E / 盈利增速 | <1.5 |

#### 4.2 风险量化扣分机制

| 风险类型 | 风险指标 | 扣分标准 |
|----------|----------|----------|
| 地缘风险 | 中国收入占比>35% | -10分 |
| 客户集中 | Top 3客户>50% | -5分 |
| 出口管制 | 受限产品占比>20% | -15分 |
| 周期风险 | 处于周期高点 | -10分 |
| 技术风险 | 技术代际落后 | -20分 |
| 估值风险 | P/E>历史75分位 | -5分 |

## 三层库存仪表盘

### 库存健康度判断矩阵

| 层级 | 库存主体 | 健康信号 | 警戒信号 |
|------|----------|----------|----------|
| L1 | 设备商（LRCX/AMAT） | 库存↓ + 递延收入↑ | 库存↑ + 收入↓ |
| L2 | 晶圆厂（TSM/Samsung） | 库存正常 + 利用率>80% | 库存积压 + 利用率<70% |
| L3 | Memory（MU/SK） | 价格↑ + 库存↓ | 价格↓ + 库存↑ |

### 库存周期位置判断

```
           库存↑ + 价格↓
              ↓
    [被动补库] ← → [主动去库]
         ↑              ↓
    [主动补库] ← → [被动去库]
              ↑
           库存↓ + 价格↑
```
