## 投资结论

### 评级：强烈推荐（87.9/100）

### 核心投资逻辑

1. **AI驱动的Memory超级周期**
   - HBM需求爆发式增长，LRCX刻蚀设备受益明显
   - Memory CAPEX预计2026年增长45%+

2. **先进制程持续扩张**
   - TSM 3nm/2nm扩产带动Foundry需求
   - GAA架构转换增加刻蚀步骤数量

3. **财务质量优秀**
   - 毛利率50%+，净利率30%
   - OCF/NI > 1.0，盈利质量高
   - 净现金状态，财务稳健

4. **估值合理**
   - PEG < 1.0，相对增速低估
   - 周期上升期估值有支撑

### 主要风险

1. **中国业务风险（首要）**
   - 收入占比42.9%，出口管制持续收紧
   - 若完全丧失中国市场，收入影响约30-40%

2. **周期性风险**
   - 半导体设备为强周期行业
   - 需持续监控下游CAPEX指引变化

3. **地缘政治风险**
   - 台海局势对TSM依赖的影响
   - 全球供应链重构进程

### 建议操作

| 时间维度 | 建议 |
|----------|------|
| 短期（3-6月） | 持有，等待季度业绩确认 |
| 中期（6-12月） | 核心仓位持有，顺周期配置 |
| 长期（1-3年） | 战略性持有，AI半导体核心标的 |

### 关键监控指标

1. **每季必看**：TSM/MU CAPEX指引、LRCX递延收入、中国收入占比
2. **每月关注**：Memory价格走势、设备BB Ratio
3. **事件驱动**：出口管制政策更新、客户法说会指引
