## 五、营运周转因子详解

### 5.1 DSO (Days Sales Outstanding) - 应收账款周转天数

**DSO计算公式**:
```
DSO = (应收账款 / 年收入) × 365
```

**LRCX估算** (基于行业经验):
```
假设: 年收入 $17.5B, 应收账款 $2.8B
DSO = ($2.8B / $17.5B) × 365 ≈ 58天
```

**解读**:
| 指标 | LRCX估算 | 行业标准 | 判定 |
|------|----------|----------|------|
| DSO | ~58天 | 45-75天 | **正常** |

1. **意义**: DSO反映客户付款速度，间接反映客户财务健康度
2. **传导逻辑**:
   - 若下游客户(TSM/MU)现金流紧张 → 付款延迟 → DSO上升
   - DSO突然上升是下游需求疲软的领先指标
3. **监测要点**:
   - 与下游客户的DSO交叉验证
   - 季度环比变化趋势

### 5.2 DIO (Days Inventory Outstanding) - 存货周转天数

**DIO计算公式**:
```
DIO = (存货 / 销售成本) × 365
```

**LRCX估算**:
```
假设: 销售成本 $9.1B (基于52.2%营业成本率), 存货 $4.5B
DIO = ($4.5B / $9.1B) × 365 ≈ 180天
```

**解读**:
| 指标 | LRCX估算 | 行业标准 | 判定 |
|------|----------|----------|------|
| DIO | ~180天 | 150-250天 | **正常** |

1. **设备商DIO普遍较高的原因**:
   - 设备制造周期长(3-6个月)
   - 需要备件库存支持售后服务
   - 高端设备定制化程度高
2. **传导逻辑**:
   - 下游CAPEX扩张 → 设备商订单增加 → 主动备货 → DIO短期上升
   - 下游CAPEX收缩 → 订单取消 → 被动积压 → DIO被动上升 (危险信号)
3. **区分主动vs被动**:
   - 看Backlog(订单积压)是否同步增长
   - LRCX Backlog健康 → DIO上升是主动备货

### 5.3 DPO (Days Payable Outstanding) - 应付账款周转天数

**DPO计算公式**:
```
DPO = (应付账款 / 销售成本) × 365
```

**LRCX估算**:
```
假设: 应付账款 $1.2B
DPO = ($1.2B / $9.1B) × 365 ≈ 48天
```

**解读**:
| 指标 | LRCX估算 | 行业标准 | 判定 |
|------|----------|----------|------|
| DPO | ~48天 | 30-60天 | **正常** |

1. **DPO反映对上游供应商的议价能力**
2. **LRCX上游供应商**: UCTT (超洁净) 占UCTT收入52%
   - LRCX是UCTT最大客户，有较强议价能力
   - 可以延长付款周期，优化自身现金流

### 5.4 CCC (Cash Conversion Cycle) - 现金转换周期

**CCC计算公式**:
```
CCC = DSO + DIO - DPO
    = 58 + 180 - 48
    = 190天
```

**解读**:
| 指标 | LRCX估算 | 行业标准 | 判定 |
|------|----------|----------|------|
| CCC | ~190天 | 150-250天 | **正常** |

1. **CCC含义**: 从支付供应商到收到客户款项的时间差
2. **190天的含义**: LRCX需要约6个月的营运资金来维持运营
3. **优化方向**:
   - 降低DIO (精益制造)
   - 延长DPO (供应商谈判)
   - 缩短DSO (客户信用管理)
