## v20.0核心技能详细说明

### 预测市场相关技能

#### Polymarket Prediction Analyzer Skill
**文件**: `skills/v10.0-engines/polymarket-prediction-analyzer.skill.md`

**主要能力**:
```yaml
概率获取能力:
  - 实时连接Polymarket API获取半导体相关事件概率
  - 支持台海冲突、技术制裁、AI需求等多维度事件
  - 自动筛选高质量事件(流动性>$500K, 参与者>200人)

数据质量评估:
  - 三维质量评分: 流动性+参与者质量+历史准确率
  - 自动过滤低质量事件和操纵风险事件
  - 提供每个概率的置信度评分(A/B/C级)

背离分析算法:
  - 从股价中提取市场隐含概率
  - 计算与预测市场概率的偏差
  - 生成套利信号和预期收益评估
  - 输出具体的交易建议和风险控制

真金白银验证:
  - 基于实际押注资金验证市场情绪
  - 区分机构级大额押注vs散户情绪
  - 提供资金加权的概率修正
```

**输出标准**:
- 每个分析公司必须覆盖≥6个核心预测事件
- 每个概率必须有质量评分和置信区间
- 必须提供概率-价格背离分析和套利建议

#### PPDA Algorithm Engine
**算法文件**: `frameworks/v10.0-prediction-market/algorithms/ppda_algorithm.js`

**分析流程**:
```
1. 市场隐含概率提取:
   - 从当前股价反推市场对各风险的隐含定价
   - 基于历史价格敏感性计算影响系数
   - 考虑估值水平和风险溢价的影响

2. 背离度量化计算:
   背离度 = (市场隐含概率 - Polymarket实际概率) / 实际概率

3. 套利信号生成:
   - 背离度>20%: 强信号
   - 背离度10-20%: 中等信号
   - 背离度<10%: 弱信号

4. 置信度评估:
   - 基于预测市场数据质量
   - 考虑历史背离修复的时间和幅度
   - 结合市场流动性和参与者结构
```
