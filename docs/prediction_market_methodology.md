# 预测市场方法论

> 仅在 Tier 2/3 分析需要预测市场数据时加载。

---

## 核心概念

### PPDA（概率-价格背离分析）

从股价反推市场隐含概率，对比预测市场真实概率，量化背离度。

**计算步骤**:
1. 通过WebSearch获取预测市场实际概率（如Polymarket事件）
2. 从股价/估值推断市场隐含概率
3. 计算背离度 = |隐含概率 - 实际概率| / 实际概率
4. 背离度 > 50% → 潜在套利机会

**重要**: 预测市场概率必须通过WebSearch获取真实数据，不可虚构。无数据时标注"该事件预测市场无覆盖"。

### PMSI（预测市场情绪指数）

基于预测市场构建的量化情绪指标。

```python
PMSI = 宏观经济权重×40% + 行业特定权重×30% + 公司特定权重×20% + 地缘政治权重×10%
```

**构建步骤**:
1. 识别4个维度各2-3个关键预测事件
2. WebSearch获取各事件的预测市场概率
3. 按权重加权计算综合情绪指数（0-100）
4. > 70为乐观 | 30-70为中性 | < 30为悲观

**验证要求**: 与股价走势对照验证合理性

## 数据获取方法

### 预测市场数据源

| 平台 | 覆盖范围 | 获取方式 |
|------|---------|---------|
| Polymarket | 地缘+经济+公司事件 | `WebSearch: site:polymarket.com [关键词]` |
| Kalshi | 宏观+监管+选举 | `WebSearch: site:kalshi.com [关键词]` |
| Manifold | 技术突破+行业趋势 | `WebSearch: site:manifold.markets [关键词]` |

### 数据可信度分级

| 等级 | 来源 | 标注方式 |
|------|------|---------|
| **A级** | 财报+SEC+预测市场（流动性≥$500K） | `[A:源]` |
| **B级** | 第三方数据库+权威预测平台 | `[B:源]` |
| **C级** | 分析师共识/市场预期 | `[C:源]` |
| **D级** | 合理估算 | `[D:估算]` |
| **E级** | 假设/推测 | `[E:假设]` |

### 诚实标注规则

- 有预测市场覆盖 → 标注实际概率+平台+日期
- 无预测市场覆盖 → 标注"无预测市场数据"，使用替代来源
- 估算概率 → 明确标注"分析师估算"，不伪装为"预测市场数据"
