# 行为金融检查清单

> **用途**: Tier 3 Phase 4（对抗审查）强制执行。每份深度报告必须识别并量化认知偏差对估值的影响。

---

## 四大认知偏差检查

### 1. 锚定效应 (Anchoring Bias)

**检测问题**: 当前估值是否被某个"锚点"不合理地牵引？

| 常见锚点 | 检测方法 | 修正方式 |
|---------|---------|---------|
| 历史高点/低点 | 当前价距52周高/低点百分比 | 用DCF内在价值替代价格锚 |
| 分析师共识目标价 | 共识偏离度 vs 历史准确率 | 回测共识3年准确率后加权 |
| 整数关口 ($100, $200) | 价格在整数附近停留时间 | 忽略整数心理，用基本面驱动 |
| IPO价格/成本基础 | 个人持仓成本影响判断 | 用机会成本框架替代沉没成本 |

**输出模板**:
```
锚定效应分析:
- 识别锚点: [具体锚点]
- 牵引方向: [偏高/偏低]
- 估值偏离: [±X%]
- 修正后估值: [$XXX]
```

### 2. 确认偏误 (Confirmation Bias)

**检测问题**: 分析是否只收集了支持既有观点的证据？

**强制反证清单**:
- [ ] 列出≥3条反对投资论点的证据
- [ ] 引用≥1位持相反观点的知名分析师/投资者
- [ ] 量化"如果我错了"的最大损失
- [ ] 检查数据源是否偏向单一视角

**输出模板**:
```
确认偏误审查:
- 主论点: [看多/看空理由]
- 反证1: [具体反对证据] — 来源: [X]
- 反证2: [具体反对证据] — 来源: [X]
- 反证3: [具体反对证据] — 来源: [X]
- 最大损失情景: [价格/百分比]
- 对手方逻辑: [持相反观点者的核心理由]
```

### 3. 可得性偏误 (Availability Bias)

**检测问题**: 近期事件是否被过度外推？

| 偏误场景 | 检测信号 | 修正方式 |
|---------|---------|---------|
| 近期暴涨 → 过度乐观 | RSI>70 + 分析师集体上调 | 回测类似涨幅后12个月表现 |
| 近期暴跌 → 过度悲观 | RSI<30 + 分析师集体下调 | 回测类似跌幅后12个月表现 |
| 新闻轰炸 → 过度反应 | 新闻量>3倍正常水平 | 等待3-5天噪音消退后再判断 |
| 行业热潮 → 忽视风险 | 同行业多只同时创新高 | 检查历史上类似热潮的结局 |

**输出模板**:
```
可得性偏误检查:
- 近期主导叙事: [具体叙事]
- 叙事持续时间: [X天/周]
- 历史基准率: [类似情况发生的历史概率]
- 被忽视的因素: [列出被热叙事掩盖的风险/机会]
```

### 4. 框架效应 (Framing Effect)

**检测问题**: 同一数据是否因表述方式不同导致不同结论？

**双框架测试**:
```
正面框架: "营收增长15%" → 看起来很好
负面框架: "增速从25%降至15%" → 看起来在减速

正面框架: "毛利率62%" → 行业领先
负面框架: "毛利率从68%降至62%" → 持续下滑

→ 两个框架都正确。最终判断应基于:
  1. 绝对水平 vs 行业中位数
  2. 变化趋势 + 未来驱动因素
  3. 与估值隐含增速的匹配度
```

## 投资者情绪综合评分

> 综合四个维度的情绪指标，输出0-10分（0=极度恐惧，10=极度贪婪）

### 评分维度

| 维度 | 权重 | 数据源 | 评分方法 |
|------|------|--------|---------|
| 技术情绪 | 25% | RSI/布林带/成交量 | RSI>70=8-10, RSI<30=0-2, 中间线性插值 |
| 分析师情绪 | 25% | 评级变化+目标价变化 | 上调占比>70%=8-10, <30%=0-2 |
| 机构行为 | 25% | 13F持仓变化+资金流向 | 净增持=7-10, 净减持=0-3 |
| 散户情绪 | 25% | 社交媒体+期权Put/Call | Put/Call<0.7=8-10, >1.3=0-2 |

### 情绪-周期修正

```yaml
P1底部 (情绪0-3): 市场定价过度悲观 → 评分+10~15分修正
P2早期 (情绪3-5): 怀疑中上涨 → 评分+5~10分修正
P3成长 (情绪5-7): 共识形成 → 评分+0~5分修正
P4成熟 (情绪7-8): 乐观过度 → 评分-5~10分修正
P5顶部 (情绪8-10): FOMO驱动 → 评分-15~20分修正
```

### 机构-散户分歧指标

```
分歧度 = |机构情绪 - 散户情绪|

分歧度 > 3: 显著分歧，深入分析原因
  - 机构>散户: 可能是信息优势驱动（偏正面信号）
  - 散户>机构: 可能是FOMO驱动（偏负面信号）

分歧度 < 1: 高度共识，注意反转风险
```

## 执行规则

1. **强制性**: Tier 3 报告的 Phase 4（对抗审查）必须包含完整的四项偏差检查
2. **数据驱动**: 每项偏差分析必须有具体数据支撑，禁止纯定性描述
3. **量化修正**: 必须输出偏差对估值的量化影响（±X%或±$X）
4. **独立性**: 行为金融分析应在估值完成后执行，作为"第二意见"而非修改原始估值
5. **Tier 2 简化版**: Tier 2 报告只需执行确认偏误检查（反证清单），其余三项可选
