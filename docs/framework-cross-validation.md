```

  - 上下游（15%）：CAPEX大砍 → 3分 × 0.15 = 0.45
  基础分 = 6.00 → 60分

Step 3: 背离信号
  - 估值8 > 7 AND 先行指标2 < 3 → 底部背离 +20分

Step 4: 情绪极端
  - 极度悲观 + 低估值 → +10分

最终得分 = 60 + 20 + 10 = 90分 → 强烈买入
```

## 五、框架局限性发现

### ASML的特殊性

ASML在2023年4-10月表现与LRCX/AMAT不同步：
- LRCX/AMAT涨，ASML跌
- 原因：ASML的EUV周期与Memory周期有时间差
- 启示：框架需要考虑公司特定的"周期敏感度"

### 建议的调整

```
公司特定调整因子：

LRCX（Memory敞口60%）：
  Memory周期权重 = 1.2x（放大）

AMAT（Memory敞口35%，代工50%）：
  Memory周期权重 = 0.8x
  代工周期权重 = 1.0x

ASML（EUV垄断，Backlog长）：
  Memory周期权重 = 0.6x
  先进制程周期权重 = 1.2x
  Backlog变化 = 额外指标
```

## 六、验证结论

### 框架有效性

| 结论 | 证据 |
|------|------|
| **v5.0显著优于v4.0** | 准确率从30%提升到93% |
| **跨标的验证通过** | 三家公司都有效 |
| **核心改进有效** | 背离信号在所有公司都成功识别顶底 |

### 仍需警惕

| 风险 | 说明 |
|------|------|
| **过拟合风险** | v5.0是事后设计，参数可能过度优化 |
| **样本有限** | 只测试了1个完整周期 |
| **公司差异** | ASML需要特殊处理 |

### 下一步验证

1. **跨周期验证**：用2018-2022周期测试（不同市场环境）
2. **样本外测试**：用2025-2026新数据持续监控
3. **参数稳定性**：固定参数，不再调整，观察未来表现
