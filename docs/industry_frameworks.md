### PVM 增长质量诊断

```
健康模式:                        不健康模式:
├─ Price-Led: 提价成功，Volume稳定   ├─ Hollow: Price↑ 但 Volume↓ ⚠️
├─ Mix-Led: 高端化成功，毛利↑        ├─ Promo-Dependent: 促销>60%
└─ Volume-Led: 市场扩张，份额↑       └─ Mix-Down: 被迫下沉

关键公式:
  净价 = 标价 - 促销折扣 - 返利 - 优惠券
  促销强度 = (标价 - 净价) / 标价

Kill-Switch:
  □ 空心化增长: Price贡献>0 AND Volume贡献<-5% AND 净价YoY<0
  □ 促销上瘾: 促销销量占比>60% AND 促销强度上升>5pp YoY
```

### 消费品 Kill-Switch 清单

| 类别 | Kill-Switch | 触发条件 | 严重度 |
|------|-------------|---------|--------|
| **增长质量** | 空心化增长 | Price↑ Volume↓ 净价↓ | Hard |
| **增长质量** | 促销上瘾 | 促销销量>60% + 强度↑5pp | Soft |
| **定价权** | 溢价崩塌 | 溢价↓15% + PED↑0.3 | Hard |
| **定价权** | 弹性螺旋 | PED连续3Q上升 + VoC敏感 | Soft |
| **渠道** | 利润侵蚀 | Net Channel Margin↓5pp YoY | Soft |
| **渠道** | 价格崩塌 | 跨渠道价差>20% + 窜货 | Hard |
| **品牌** | 质量危机 | 质量差评>20% 持续4周 | Hard |
| **品牌** | 感官漂移 | 核心感官签名偏离 | Soft |

### 40维度MECE分析体系

#### 银行业15个维度
1. **监管与资本**(5维): 资本充足率/流动性管理/杠杆率/风险暴露/监管评级
2. **资产负债**(4维): 净息差建模/久期管理/存款结构/贷款组合
3. **信贷风险**(3维): PD/LGD/EAD建模/资产质量/集中度风险
4. **运营效率**(2维): 数字化转型/成本收入管理
5. **市场地位**(1维): 竞争优势评估

#### 保险业12个维度
1. **精算估值**(4维): 内含价值/准备金充足性/假设审计/偿付能力
2. **承保业务**(4维): 承保质量/产品组合/渠道管理/客户管理
3. **投资管理**(2维): 资产配置/投资收益
4. **监管合规**(2维): 监管要求/风险管控

#### Fintech业8个维度
1. **商业模式**(3维): 单元经济学/网络效应/产品生态
2. **技术护城河**(2维): 技术优势/风险控制
3. **监管合规**(2维): 牌照价值/数据治理
4. **竞争地位**(1维): 市场地位

#### 横向整合5个维度
1. **ESG影响**: 气候风险量化+社会责任+治理质量
2. **无形资产**: 品牌价值+客户关系+技术资产
3. **AI治理**: 算法伦理+模型风险+监管合规
4. **地缘政治**: 跨境业务+监管差异+政策变化
5. **宏观敏感**: 利率环境+经济周期+政策传导
