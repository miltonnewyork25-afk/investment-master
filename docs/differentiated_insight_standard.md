# 差异化洞察标准 v1.0

> 每个模块必须通过 "So What?" 检验。无差异化洞察的模块压缩为3行摘要。

---

## 设计哲学

**问题**: 模块堆砌公开信息 → 读者看完不知道这意味着什么
**方案**: 每段分析必须回答 "So What?" — 这对投资决策意味着什么？

---

## "So What?" 评分标准 (0-10)

| 分数 | 层级 | 定义 | 执行动作 |
|:---:|------|------|---------|
| **0-3** | 信息复述 | 纯公开数据罗列，无分析 | 压缩为3行摘要 |
| **4-6** | 有分析无独特视角 | 整合了数据但结论与卖方报告一致 | 保留核心段落，删除填充 |
| **7-8** | 独特视角 | 提出市场未广泛讨论的角度 | 完整展开 |
| **9-10** | 揭示未定价信息 | 发现市场可能定价错误的洞察 | 完整展开+高亮为洞察卡 |

### 各层级示例

**0-3分 (信息复述)**:
```
"Costco FY2025营收$254.2B，同比增长5.0%。毛利率12.6%。"
→ So What? 缺失。读者打开Yahoo Finance就能看到。
→ 处理: 压缩为 "FY2025营收$254.2B(+5.0%), 毛利率12.6% [硬数据: 10-K]"
```

**4-6分 (有分析无独特视角)**:
```
"Costco的会员模式创造了稳定的经常性收入，续费率92.9%表明客户粘性极强。"
→ So What? 有，但每个卖方报告都这么写。
→ 处理: 保留核心数据点，删除修饰语
```

**7-8分 (独特视角)**:
```
"会员续费率92.9%掩盖了一个结构性变化: Executive会员占比从2020年40%升至2025年46%，
意味着高价值会员增速2x普通会员。这创造了'隐性提价'——无需调整费率就能提升ARPU。"
→ So What? 明确——市场按会员总数估值，忽略了会员结构升级的增量价值。
→ 处理: 完整展开
```

**9-10分 (揭示未定价信息)**:
```
"Kirkland品牌占销售额28%，但其战略意义被市场低估: 每增加1pp Kirkland占比，
约等于消灭一个中型CPG品牌的Costco渠道收入。供应商的'Costco依赖度'指标
显示前20供应商中14家Costco占其总收入>15%。这意味着COST在不提价的情况下，
通过Kirkland扩张就能隐性压缩COGS 50-100bps/年。"
→ So What? 揭示了一个未被广泛定价的利润增长机制。
→ 处理: 完整展开 + 标记为洞察卡
```

## 模块压缩模板

当模块评分 0-3 时，使用以下模板：

```markdown
### [模块名] (摘要)

**核心数据**: [1-2个关键数字] | **趋势**: [一句话] | **与CQ相关性**: 低
```

当模块评分 4-6 时，保留核心：

```markdown
### [模块名]

[核心段落: 数据+分析+So What, 500-1000字符]

> 注: 本模块与当前核心问题相关性中等，已聚焦于最相关部分。
```

---

## 质量检查集成

Phase 4 对抗审查时，对每个模块执行 "So What?" 抽查：

1. 随机抽取 5 个模块
2. 对每个模块的核心段落问 "So What?"
3. 如果≥2个模块无法回答 → 触发返工
