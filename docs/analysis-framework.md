# 投资大师 - 半导体供应链投资分析框架

**版本**: 1.0.0
**更新日期**: 2026-01-21
**核心等式**: 设备商收入 ≈ 下游晶圆厂CAPEX

---

## 一、框架概述

### 1.1 分析对象定位

```
[设备商/材料商] ─→ [晶圆厂/IDM] ─→ [Fabless设计公司] ─→ 终端需求
     上游              中游              下游
```

### 1.2 核心投资逻辑

设备商的收入本质上是下游晶圆厂CAPEX的函数：
- 终端需求 → 芯片需求 → 产能利用率 → 晶圆厂盈利 → CAPEX决策 → 设备商订单 → 设备商收入

|------|------|----------|
| 估值 | 35% | PE TTM, EV/EBIT, 毛利率, ROIC |
| 证据 | 40% | 供应链边数量、等级、置信度 |
| 动量 | 25% | 收入增长率 YoY |

## 三、估值因子详解

### 3.1 PE TTM (市盈率)

**阈值定义**:
- PE < 15: 低估 → +25分
- PE > 40: 过热 → -20分
- PE < 0: 亏损 → -30分

**解读方法**:
1. 横向对比同行业PE水平
2. 纵向对比历史PE区间
3. 结合增长率计算PEG (PE/Growth)

### 3.2 EV/EBIT (企业价值/息税前利润)

**阈值定义**:
- EV/EBIT < 15: 偏低 → +15分
- EV/EBIT > 35: 高估 → -15分

**优势**: 排除资本结构影响，更能反映经营效率

### 3.3 毛利率 (Gross Margin)

**阈值定义**:
- GM >= 40%: 健康 → +10分
- GM < 25%: 低毛利 → -10分，触发风险标志

**行业参考**:
- 设备商正常区间: 40-65%
- 晶圆厂正常区间: 30-55%
- 材料商正常区间: 35-50%

### 3.4 ROIC (投入资本回报率)

**阈值定义**:
- ROIC >= 15%: 优秀 → +10分
- ROIC < 0: 负回报 → -15分

**计算公式**:
```
ROIC = NOPAT / 投入资本
     = EBIT × (1 - 税率) / (股东权益 + 有息负债 - 现金)
```

## 四、营运周转因子详解

### 4.1 DSO (应收账款周转天数)

**计算公式**:
```
DSO = (应收账款 / 年收入) × 365
```

**解读**:
- 反映客户付款速度
- DSO上升 = 下游现金流紧张的领先指标
- 行业标准: 45-75天

**上下游传导**:
- 下游客户财务恶化 → 付款延迟 → 上游DSO上升

### 4.2 DIO (存货周转天数)

**计算公式**:
```
DIO = (存货 / 销售成本) × 365
```

**解读**:
- 设备商DIO普遍较高(150-250天)，因制造周期长
- 区分主动备货 vs 被动积压:
  - Backlog增长 + DIO上升 = 主动备货(正面)
  - Backlog下降 + DIO上升 = 被动积压(负面)

### 4.3 DPO (应付账款周转天数)

**计算公式**:
```
DPO = (应付账款 / 销售成本) × 365
```

**解读**:
- 反映对上游供应商的议价能力
- DPO越长，现金流越健康
- 行业标准: 30-60天

### 4.4 CCC (现金转换周期)

**计算公式**:
```
CCC = DSO + DIO - DPO
```

**解读**:
- 从支付供应商到收到客户款项的时间差
- 设备商正常区间: 150-250天
- CCC越短，营运资金效率越高

## 五、证据因子详解

### 5.1 证据等级定义

| 等级 | 权重 | 来源 | 示例 |
|------|------|------|------|
| A | 1.0 | SEC官方文件 + 明确数字 | "TSM占收入25%" |
| B | 0.7 | IR材料或无数字的SEC文件 | "TSM是主要客户" |
| C | 0.4 | 其他来源(新闻、研报) | "据悉TSM采购..." |

### 5.2 证据得分计算

```
证据得分 = 30(基准) + Σ(15 × 等级权重 + 置信度 × 10)
上限 = 100分
```

### 5.3 置信度加成

- 包含具体百分比: +0.3
- 包含关键词(largest, significant): +0.1
- 包含具体年份: +0.1

## 六、风险标志系统

### 6.1 触发条件

| 风险类别 | 触发条件 | 严重程度 |
|----------|----------|----------|
| 估值过热 | PE > 40 | 警告 |
| 当前亏损 | PE < 0 | 严重 |
| 低毛利率 | GM < 25% | 警告 |
| 客户集中度高 | 客户数 ≤ 1 | 警告 |
| 证据不足 | 供应链边数 = 0 | 警告 |
| 收入下滑 | YoY增长 < -10% | 严重 |

### 6.2 风险标志影响

- 风险标志不直接扣分
- 在排名相近时作为辅助判断依据
- 多个风险标志叠加需特别关注

## 七、CAPEX传导分析框架

### 7.1 传导链条

```
终端需求(AI/手机/汽车)
    ↓
芯片需求量预测
    ↓
晶圆厂产能利用率
    ↓
晶圆厂盈利能力(毛利率/ROIC)
    ↓
CAPEX决策(指引)
    ↓
设备商订单(Backlog)
    ↓
设备商收入
```

### 7.2 关键验证点

| 验证点 | 数据来源 | 领先性 |
|--------|----------|--------|
| 下游CAPEX指引 | 财报/IR | 领先6-12月 |
| 下游产能利用率 | 财报/行业数据 | 领先3-6月 |
| 设备商Backlog | 财报 | 领先3-6月 |
| 内存价格 | DRAM Exchange | 同步指标 |

### 7.3 上下游财务传导

| 下游指标变化 | 对上游的影响 |
|--------------|--------------|
| 毛利率↑ | 盈利改善→CAPEX有支撑→利好 |
| 库存↑ + 利用率↓ | 供过于求→CAPEX收缩→利空 |
| DSO↑ | 现金流紧张→付款延迟→利空 |
| CAPEX指引↑ | 直接利好设备商订单 |

## 八、单公司分析模板

### 8.1 报告结构

1. **公司定位与商业模式**
   - 业务概述
   - 供应链位置
   - 核心产品/技术

2. **上下游供应链证据**
   - 客户集中度(SEC披露)
   - 供应链边详情
   - CAPEX传导验证

3. **估值因子分析**
   - PE TTM解读
   - EV/EBIT解读
   - 同行对比

4. **质量因子分析**
   - 毛利率解读
   - ROIC解读
   - 趋势分析

5. **营运周转分析**
   - DSO/DIO/DPO计算
   - CCC分析
   - 上下游传导验证

6. **动量因子分析**
   - 收入增长解读
   - 与下游增长交叉验证

7. **风险标志检测**
   - 六项风险扫描
   - 额外风险点

8. **综合评分汇总**
   - 三维得分
   - 加权总分

9. **投资结论**
   - 核心观点
   - 催化剂
   - 风险提示

## 九、公司分类标准

### 9.1 上游设备商

- 特征: 轻资产、高ROIC、强周期
- 代表: ASML, AMAT, LRCX, KLAC, TER
- 关注点: 下游CAPEX指引、Backlog

### 9.2 中游晶圆厂/IDM

- 特征: 重资产、中等ROIC、周期性
- 代表: TSM, INTC, SAMSUNG, GFS, UMC
- 关注点: 产能利用率、CAPEX计划、毛利率

### 9.3 下游Fabless

- 特征: 轻资产、高毛利、设计驱动
- 代表: NVDA, AMD, QCOM, AVGO
- 关注点: 终端需求、库存水位、产品周期

### 9.4 内存厂商

- 特征: 重资产、强周期、价格敏感
- 代表: MU, SKHYNIX, SAMSUNG
- 关注点: 内存价格、库存、HBM需求

```yaml

  valuation: 0.35   # 估值权重
  evidence: 0.40    # 证据权重
```

```yaml
thresholds:
  pe_overvalued: 40
  pe_undervalued: 15
  ev_ebit_overvalued: 35
  gm_healthy: 0.40
  roic_good: 0.15
```
