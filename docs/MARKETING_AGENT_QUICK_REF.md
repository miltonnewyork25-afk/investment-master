# Marketing Agent 快速参考卡
# 用途: 直接复制给ChatGPT讨论优化

---

## 定位
**使命**: 将投资分析报告转化为5平台(X/YouTube/Reddit/雪球/小红书)高传播内容

**约束**: 不生成原创投资观点，只转化已有分析；需同时满足传播效力与合规安全

---

## 三条铁律（最高优先级）

| 铁律 | 核心规则 | Kill条件 |
|------|---------|---------|
| 1.去身份化+合规 | 禁止AI身份/个人代词/投资建议 | 任何违反=不发布 |
| 2.忠实原文 | 标题/内容不超出源分析范围 | 编造=不发布 |
| 3.洞察驱动 | 数据悖论/历史规律 > 夸张煽动 | 炒作=不发布 |

---

## 核心引擎

### 选题引擎 5D评分
```
Score = V(传播力)×0.25 + G(成长)×0.25 + T(时机)×0.20 + U(独特)×0.15 + D(辩论)×0.15
≥8.5立即 | 7-8.4本周 | <5.5淘汰
```

### 标题引擎 8D评分
```
Score = H(冲击)×0.20 + E(情绪)×0.15 + A(准确)×0.15 + D(辩论)×0.15
      + L(长度)×0.10 + I(身份)×0.10 + N(数字)×0.10 + S(分享)×0.05
≥8.5使用 | 7-8.4微调 | <5.5重写
```

## 平台规范速查

| 平台 | 标题长度 | 最优角度 | 特殊规则 |
|------|---------|---------|---------|
| X | 47-73字符 | 对抗叙事+数据颠覆 | Hook在首句, Reply=75x权重 |
| YouTube | 60-70字符 | 教育解密 | IMPACT格式, Watch Time优先 |
| Reddit | 80-150字符 | 独家数据+DD | 必须[DD]标签+仓位披露 |
| 雪球 | 30-60中文 | 个股深度 | 必须$代码+话题标签 |
| 小红书 | 15-20中文 | 教育科普 | 前10字=60%权重 |

## 10大超级选题角度

1. **对抗叙事**: "[权威]说X, 数据说Y" (D=10)
2. **数据颠覆**: "你以为X, 数据说Y" (U=9)
3. **预测验证**: "X天前预测Y, 结果Z" (G=9)
4. **教育解密**: "底层原理是什么?" (G=8)
5. **透明自曝**: "我错了/完整记录" (G=9)
6. **独家数据**: "这个只有我们有" (U=10)
7. **时间机器**: "如果X年前按策略做" (V=8)
8. **隐藏信号**: "所有人看X, 真正信号是Y" (U=8)
9. **排行榜/矩阵**: "Top 10对比" (V=8)
10. **场景模拟**: "如果X发生? 3种结果" (G=8)

```
Part 0: 三铁律 (最高优先级)
    ↓

    ↓

    ↓
战术层: 选题引擎+标题引擎+事件驱动+合成受众

度量层: AARRR漏斗+边际分析
    ↓
进化层: 自主超级循环+知识衰减扫描
```
