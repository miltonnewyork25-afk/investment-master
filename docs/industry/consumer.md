## 双轨路由: 品牌消费品 vs 零售商

消费品研究的核心分歧在于**品牌商**和**零售商**的分析逻辑根本不同。

| 维度 | 品牌消费品 (PG/KO/NKE) | 零售商 (COST/WMT/TGT) |
|------|------------------------|----------------------|
| **利润来源** | 品牌溢价+渠道控制 | 规模效率+周转速度 |
| **护城河核心** | 行为习惯+感官锁定+品牌资产 | 采购规模+成本领先+会员锁定 |
| **增长引擎** | PVM(Price/Volume/Mix) | 同店销售+新店+电商 |
| **估值锚** | 品牌价值+现金流稳定性 | 坪效+会员经济+SOTP |
| **关键风险** | 自有品牌侵蚀+消费降级 | 同店恶化+客流崩塌+份额流失 |
| **行业系数** | ×1.5 | ×1.3 |
| **专用模块** | 行为护城河+PVM+感官营销 | 同店拆解+门店经济+会员经济 |
| **Kill Switch侧重** | 空心化增长+溢价崩塌 | 续费率+同店+毛利压缩 |

**路由规则**: 分析开始前必须确定公司属于哪条轨道。跨轨公司(如COST的Kirkland)需同时覆盖两套逻辑。

## PPDA四维竞争模型

> Price / Product-Quality / Distribution / Advertising 背离分析
> 来源: COST v21 Phase 3 竞争分析, 泛化为消费品通用模型

### 核心理念

行业内每家公司在P/P/D/A四个维度都有"常规做法"。**背离行业常规且能持续的公司，往往拥有不可复制的竞争优势**。

### 背离评分标准 (-3 ~ +3)

| 分数 | 含义 | 示例 |
|------|------|------|
| +3 | 极端正向背离(远超行业) | Costco品质: 低价+品类冠军品质 |
| +2 | 显著正向背离 | PG品牌溢价: 持续高于竞品20%+ |
| +1 | 轻微正向背离 | — |
| 0 | 行业常规 | 大多数公司 |
| -1 | 轻微负向背离 | — |
| -2 | 显著负向背离(落后行业) | 电商渗透率低于行业50% |
| -3 | 极端负向背离(战略性放弃) | Costco分销: 仅921店 vs WMT 4,600+ |

### 行业基准表 (零售子行业)

| 维度 | 超市基准 | 仓储会员基准 | 折扣店基准 |
|------|---------|------------|-----------|
| **Price** | 毛利率28%+ | 毛利率11-14% | 毛利率30%+ |
| **Product** | 30,000+ SKU, 品质中等 | 3,700-5,600 SKU, 品质中-高 | 1,000-2,000 SKU |
| **Distribution** | 高密度门店+全渠道 | 低密度+有限电商 | 极高密度(小店模式) |
| **Advertising** | 收入2-5%投广告 | 低/零广告 | 极低广告 |

### PPDA → 投资含义映射

| 背离模式 | 投资含义 | 可持续性判断 |
|---------|---------|------------|
| 价格极低+品质极高(P+P背离) | 利润来源非商品→需识别真实利润池 | 看制度化程度(创始人遗产 vs 口头承诺) |
| 分销极少+到店消费极高(D背离) | 稀缺性=护城河，但电商趋势是风险 | 看电商补短板速度 |
| 零广告+口碑驱动(A背离) | 营销费节省直接回馈低价→飞轮 | 看零售媒体等新变现是否补位 |
| 四维同时背离 | 系统性竞争壁垒，非单点可模仿 | **极高**，但需监控每个维度的漂移 |

## 会员经济学 (零售轨道专用)

> 提炼自 COST v21: 续费率92.3%, Executive占比47.7%, 会员费贡献51%营业利润

### 核心指标体系

| 指标 | 定义 | COST基准(FY2025) | 预警阈值 |
|------|------|-----------------|---------|
| **续费率** | 到期会员续费比例 | 美加92.3%, 全球89.8% | <90%(美加), <87%(全球) |
| **付费会员数** | 有效付费会员总数 | 81.4M | YoY增速<3% |
| **高级会员占比** | Executive/高级会员比例 | 47.7% | 占比停止提升连续2年 |
| **高级会员销售贡献** | 高级会员贡献的总销售% | 74.2% | <70% |
| **会员费/营业利润** | 会员费占营业利润比例 | ~51% | 理解利润结构的关键 |
| **每会员年消费** | 总销售/付费会员 | ~$3,374 | YoY下降 |
| **涨价弹性** | 涨价后续费率/会员数变化 | 涨价+8.3%后续费率仅降60bps | — |

### 飞轮机制诊断

```
更多会员 → 更大采购量 → 更低成本 → 更低价格
    ↑                                      ↓
    └── 更高续费率 ← 更高满意度 ←──────────┘
                          ↑
                    会员费$XB → 投入低价+员工+自有品牌
```

**飞轮状态判断**:
- **加速**: 续费率稳定/上升 + 会员数增长 + Executive占比提升 → 正常扩张
- **稳态**: 续费率高位横盘 + 会员增长放缓 → 成熟但健康
- **减速**: 续费率下降 + 会员增速<新店增速 → 需要警惕
- **破裂**: 续费率<88% + 会员净减少 → Hard Kill Switch

### 涨价弹性分析框架

每次会员涨价事件需分析:
1. **涨幅**: 绝对金额和百分比
2. **频率**: 距上次涨价间隔(COST=7年)
3. **即时反应**: 涨价后1-2季度续费率变化
4. **滞后反应**: 涨价后4-6季度续费率趋势
5. **净效应**: 涨价增收 vs 潜在流失的NPV对比

## 自有品牌分析 (双轨通用)

> 提炼自 COST v21 Kirkland Signature: $89B收入, 33%渗透率, 30年零供应商流失

### 核心评估维度

| 维度 | 评估内容 | 关键数据 |
|------|---------|---------|
| **渗透率** | 自有品牌销售/总销售 | COST 33%, Sam's ~30%, BJ's ~25% |
| **增长趋势** | 渗透率年度变化 | COST每年+~50bps持续提升 |
| **品质策略** | 品质定位 vs 品类领导者 | Kirkland=品类冠军品质, Member's Mark=价值定位 |
| **毛利贡献** | 自有品牌 vs 品牌商品毛利差 | 通常自有品牌毛利高5-15pp |
| **品类覆盖** | 哪些品类有/无自有品牌 | Kirkland覆盖几乎所有品类 |

### 供应商囚徒困境

自有品牌的战略杀伤力在于对品牌供应商的博弈结构:

```
品牌商选择:
├─ 代工自有品牌 → 短期获得大订单, 长期培养自己的竞争对手
└─ 拒绝代工 → 零售商找其他供应商代工, 品牌商失去货架+大客户

→ 两种选择都对零售商有利 = 囚徒困境
→ COST实证: Kirkland 30年零供应商流失 = 困境已被制度化
```

### 品控风险评估

| 风险 | 描述 | 监控指标 |
|------|------|---------|
| **品类过度扩张** | 进入不擅长的品类导致品质下降 | 新品类退货率, VoC负面提及 |
| **代工商能力不足** | 代工商规模跟不上需求增长 | 供货中断频率, 品质波动 |
| **品牌稀释** | 单一品牌覆盖过多品类 | 品牌信任度调查分数趋势 |

## Kill Switch合集 (消费品+零售)

### 品牌消费品Kill Switch

| 类别 | Kill Switch | 触发条件 | 严重度 |
|------|-------------|---------|--------|
| 增长质量 | 空心化增长 | Price↑ Volume↓ 净价↓ | Hard |
| 增长质量 | 促销上瘾 | 促销销量>60% + 强度↑5pp | Soft |
| 定价权 | 溢价崩塌 | 溢价↓15% + PED↑0.3 | Hard |
| 定价权 | 弹性螺旋 | PED连续3Q上升 + VoC敏感 | Soft |
| 渠道 | 利润侵蚀 | Net Channel Margin↓5pp YoY | Soft |
| 渠道 | 价格崩塌 | 跨渠道价差>20% + 窜货 | Hard |
| 品牌 | 质量危机 | 质量差评>20% 持续4周 | Hard |
| 品牌 | 感官漂移 | 核心感官签名偏离 | Soft |

### 零售商Kill Switch

| 类别 | Kill Switch | 触发条件 | 严重度 |
|------|-------------|---------|--------|
| 同店 | 同店恶化 | 连续3季度同店<0% | Hard |
| 客流 | 客流崩塌 | 客流连续4季度负增长 | Hard |
| 会员 | 会员流失 | 续费率<88% | Hard |
| 会员 | Executive降级 | Executive占比连续4季度下降 | Soft |
| 毛利 | 毛利压缩 | 毛利率YoY<-100bps | Soft |
| 库存 | 库存积压 | 周转天数+20% | Soft |
| 份额 | 份额流失 | 份额连续下滑>1pp/年 | Soft |
| 电商 | 电商落后 | 电商增速<行业50% | Soft |
| 自有品牌 | 自有品牌失败 | 自有品牌份额-5pp | Soft |
| 竞争 | 竞对技术领先 | 关键技术差距>2年(如Scan & Go) | Soft |
| AI | Agentic Commerce | AI购物代理杂货渗透>15%+客流连续4季降 | Soft |

1. **Minimal Task**: 每Agent只做1件事 — PG反例: 1个Agent做"财务+估值+周期"

3. **Trust-but-Verify**: 每Agent完成后XV Agent立即交叉验证 — 不等Phase 4
