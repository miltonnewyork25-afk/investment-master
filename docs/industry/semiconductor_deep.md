## AI双轴评估 L×S 详细维度

### L轴: 技术深度 (AI Technology Depth)
- **L0 概念**: 仅宣布AI战略，无实质产品
- **L1 组件**: 提供AI所需的基础组件(如HBM、先进制程)
- **L2 加速器**: 专用AI加速芯片(训练/推理)
- **L3 平台**: AI硬件+软件生态(CUDA等)
- **L4 收税**: 平台锁定，生态系统收税模式

### S轴: 商业兑现 (AI Revenue Realization)
- **S0**: AI收入<5%
- **S1**: AI收入5-15%
- **S2**: AI收入15-30%
- **S3**: AI收入30-50%，增速>50%
- **S4**: AI收入>50%，成为主引擎
- **S5**: AI收入主导，飞轮运转

### AI溢价矩阵

| L×S | 溢价 | 典型公司 |
|-----|------|---------|
| L≥3 AND S≥3 | 20-50% | NVDA(L4×S5) |
| L2 AND S2-3 | 10-20% | AMD(L2×S2), TSM(L1×S2) |
| L≤1 OR S≤1 | 0% | INTC(L1×S1) |

## 四大分析原则

1. **技术驱动**: 制程代差决定长期竞争力，工艺节点迁移创造价值重分配
2. **周期优先**: P1-P5周期定位 > 短期财务指标，6层雷达信号 > 管理层指引
3. **AI影响**: 区分AI硬件销售 vs AI平台生态，量化结构性 vs 周期性需求
4. **地缘整合**: 出口管制是新常态，供应链韧性 > 成本效率

## PPDA算法 (概率-价格背离分析)

```
1. 市场隐含概率提取: 从股价反推市场对各风险的隐含定价
2. 背离度计算: (市场隐含概率 - Polymarket实际概率) / 实际概率
3. 信号强度: >20%强信号 | 10-20%中等 | <10%弱信号
4. 置信度: 基于预测市场数据质量+历史背离修复时间
```

## PMSI半导体情绪指数

```python
PMSI = (地缘40% + 技术30% + 需求20% + 供应链10%) × 100

地缘模块 = (1-台海冲突概率) × 0.6 + (1-制裁概率) × 0.4
技术模块 = 技术领先概率 × 0.8 + (1-竞争威胁概率) × 0.2
需求模块 = AI需求增长概率 × 0.6 + 数据中心扩张概率 × 0.4
供应链模块 = (1-供应中断概率) × 0.7 + 产能利用率概率 × 0.3
```

## Kill Switch体系

```yaml
Tier1致命级 (立即清仓):
  - 台海冲突概率 > 25%
  - 全面技术禁运概率 > 45%
  - 系统性金融危机 > 30%

Tier2严重级 (减仓50%):
  - AI需求见顶概率 > 35%
  - 竞争威胁升级 > 40%
  - 技术路线颠覆 > 30%

Tier3警告级 (减仓25%):
  - 地缘紧张升级 > 40%
  - 估值过热 > PE 32x持续6月
  - 周期逆转 > 连续2季度下滑
  - PMSI极端值 > 85或<15持续30天
```

## 产业链信号传导

```
DRAM现货价 → Memory厂商利润 → CAPEX扩张 → 设备订单 → 设备商收入
HBM需求 ────────────────────────┘

领先周期:
  DRAM现货 → 设备收入: 12-24个月
  CAPEX公告 → 设备收入: 6-12个月
  设备订单 → 设备收入: 3-6个月
```

### 宏观敏感性

| 因素 | 传导路径 | 敏感度 |
|------|---------|--------|
| 美联储利率 | 利率↑→科技估值↓ | 高 |
| 中美关系 | 出口管制→中国收入↓ | 极高 |
| AI投资热度 | AI↑→HBM↑→设备需求↑ | 正相关 |
| DRAM价格 | 价格↑→CAPEX↑→订单↑ | 正相关 |
