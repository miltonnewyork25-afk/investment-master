## ⚙️ 双引擎 (标准化建模工具)

### 引擎A: MCI垄断指数 (0-100分制)
```yaml
MCI计算公式 = 8维度平均分 × 10

维度1_关键环节份额: [0-10分]
# 在关键工艺/产品环节的市场份额
# 10分: >80%份额 | 5分: 30-80% | 0分: <30%

维度2_替代路径时间: [0-10分]
# 可量产替代方案的时间窗口
# 10分: >5年无替代 | 5分: 2-5年 | 0分: <2年可替代

维度3_客户切换成本: [0-10分]
# 客户更换供应商的成本和风险
# 10分: 极高切换成本 | 5分: 中等成本 | 0分: 低切换成本

维度4_价格议价权力: [0-10分]
# 定价权和价格上涨能力
# 10分: 完全定价权 | 5分: 部分定价权 | 0分: 价格接受者

维度5_供给稀缺程度: [0-10分]
# 产能扩张的难度和周期
# 10分: 3年+扩张周期 | 5分: 1-2年 | 0分: <1年

维度6_生态锁定深度: [0-10分]
# 软件、工艺、平台的绑定程度
# 10分: 全栈锁定 | 5分: 部分锁定 | 0分: 开放标准

维度7_监管敏感程度: [0-10分]
# 出口管制、政策保护的影响
# 10分: 监管垄断保护 | 5分: 政策倾向 | 0分: 无政策影响

维度8_利润抽水能力: [0-10分]
# 长期毛利率和ROIC维持能力
# 10分: 毛利>60%且稳定 | 5分: 40-60% | 0分: <40%
```

### 引擎B: 供给约束模型
```yaml
约束链条分析:
订单(Bookings) → 积压(Backlog) → 产能约束 → 交付(Shipments) → 收入确认

关键指标:
- B2B增长率: [订单年化增长%]
- Backlog覆盖: [积压订单可覆盖月数]
- 产能瓶颈: [关键组件/工艺产能利用率%]
- 交付节拍: [台数/月 或 产能单位/月]
- 确认周期: [从交付到收入确认的滞后期]

约束评分 = (5项指标标准化得分平均) × 10 (0-100分)
```

## 📋 CQ化结构标准 (核心升级)

### CQ标准要求
```yaml
数量要求: 6-8个CQ (不超过10个)
分级要求:
- S级: 1-2个 (估值核心/投资决策关键)
- A级: 2-3个 (重要影响因素)
- B级: 2-3个 (补充完善分析)

质量标准:
✓ 争议性: 市场存在明显分歧
✓ 可回答性: 有客观数据和分析方法
✓ 决策相关性: 直接影响投资决策
✓ 特异性: 针对公司独特商业模式
✓ 层次递进: 从技术→商业→估值的逻辑链
```

### CQ示例模板 (以ASML为例)
```yaml
CQ1 [S级]: EUV垄断可持续性 - 技术护城河何时被突破?
CQ2 [S级]: AI超级周期持续性 - 设备需求见顶信号是什么?
CQ3 [A级]: 地缘政治影响量化 - 中国业务损失vs战略溢价?
CQ4 [A级]: High-NA执行能力 - 下一代技术商业化进展?
CQ5 [A级]: 客户议价权变化 - TSMC/Samsung议价能力提升?
CQ6 [B级]: 估值合理性判断 - 40x PE是否反映增长预期?
CQ7 [B级]: 服务收入增长 - 装机基座如何变现?
CQ8 [B级]: 供应链韧性 - 关键组件断供风险有多大?
```

## ⚡ KS分级SOP (工程化升级)

### A级 - 立即执行触发器
```yaml
定义: 触发后24-48小时内必须执行的信号
响应: 停损/减仓/重新评估投资逻辑
ASML示例:
- 地缘政治升级: 台海冲突概率>25% (Polymarket)
- 技术垄断突破: 中国EUV工作样机验证成功
- 客户集中风险: TSMC订单指引大幅下调>30%
```

### B级 - 警戒监控信号
```yaml
定义: 需要密切关注的早期预警信号
响应: 增加监控频率/准备应对方案
ASML示例:
- AI需求下滑: HPC占TSMC收入比例连续2季下降
- 服务收入恶化: 服务收入占比降至<25%连续2季
- 估值泡沫化: 远期PE>50x持续6个月
```

### C级 - 趋势监控信号
```yaml
定义: 中长期趋势变化的跟踪信号
响应: 定期评估/模型更新
ASML示例:
- 产能利用率: 晶圆厂产能利用率<75%持续3个月
- 竞争加剧: Canon/Nikon在某细分领域份额回升
- 成本上升: 关键组件成本上涨影响毛利率
```

### D级 - 基础监控信号
```yaml
定义: 行业背景和宏观环境变化
响应: 背景信息更新
ASML示例:
- 半导体周期: WFE市场增速变化
- 宏观环境: 全球制造业PMI变化
- 汇率影响: EUR/USD汇率大幅波动
```

### X级 - 黑天鹅信号
```yaml
定义: 极端情景/尾部风险事件
响应: 情景分析/极端压力测试
ASML示例:
- 台海危机: 军事冲突/台海封锁
- 技术颠覆: 全新光刻技术路径出现
- 监管极端化: 全面技术禁运/设备没收
```

### CQ↔KS映射表 (每份报告必备)
```yaml
CQ1(EUV垄断可持续性) ↔ KS-A1(技术突破)/KS-B2(竞争加剧)/KS-X1(技术颠覆)
CQ2(AI周期持续性) ↔ KS-A2(需求下滑)/KS-B1(AI泡沫)/KS-C1(周期拐点)
CQ3(地缘政治影响) ↔ KS-A3(地缘升级)/KS-B3(中国损失)/KS-X2(台海危机)
...以此类推
```
