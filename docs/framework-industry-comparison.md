## 先行指标体系对比

### Tier 1 指标（领先12-24个月）

| 半导体设备 | 集装箱航运 | 逻辑 |
|-----------|-----------|------|
| DRAM现货价 | 新船订单量 | 需求/供给领先信号 |
| Memory CAPEX | 订单簿/船队比 | 未来产能扩张信号 |
| HBM产能公告 | 拆船量 | 供给端变化 |

### Tier 2 指标（领先6-12个月）

| 半导体设备 | 集装箱航运 | 逻辑 |
|-----------|-----------|------|
| 递延收入 | SCFI运价指数 | 订单/价格确认 |
| Memory合约价 | 船队利用率 | 供需平衡状态 |
| 现货-合约价差 | 即期-合约价差 | 供需紧张度 |

**发现**：两个行业的指标体系结构高度相似，都遵循「供给信号→价格信号→确认信号」的层级。

## 估值体系对比

### 半导体设备（以PE为主）

| P/E范围 | 得分 | 说明 |
|---------|------|------|
| < 15 | 9 | 深度低估 |
| 15-20 | 7 | 低估 |
| 20-30 | 5 | 合理 |
| 30-40 | 3 | 高估 |
| > 40 | 2 | 泡沫 |

### 集装箱航运（PE + P/B）

| P/E范围 | 得分 | 说明 |
|---------|------|------|
| < 3 | 9 | 深度低估（周期底部） |
| 3-6 | 7 | 低估 |
| 6-10 | 5 | 合理 |
| 10-18 | 3 | 高估 |
| > 18 | 2 | 泡沫 |

**关键差异**：
- 航运公司盈利波动更大，周期底部常亏损
- 亏损时需用P/B估值（P/B < 0.5 为深度低估）
- 航运估值区间整体下移（历史中位PE ~8x vs 半导体 ~25x）

## 框架适配调整

### 通用部分（无需修改）

1. **周期阶段判断逻辑**
   ```
   衰退期 → 复苏期 → 扩张期 → 过热期
   ```

2. **背离信号检测**
   ```
   底部背离：估值低 + 指标差 → 买入
   顶部背离：估值高 + 指标好 → 卖出
   ```

3. **动态权重分配**
   - 衰退期：估值权重↑
   - 复苏期：先行指标权重↑
   - 过热期：边际变化权重↑

### 需要适配的部分

| 适配项 | 半导体 | 航运 |
|--------|--------|------|
| 核心价格指标 | DRAM价格 | SCFI |
| 估值基准 | PE only | PE + P/B |
| 周期阶段阈值 | YoY -20%/0%/30% | 待回测确定 |
| 特殊风险因子 | 中国出口管制 | 航道安全风险 |

### 为什么航运还不是买入时机？

```
底部背离触发条件：
  ✅ 估值得分 ≥ 7（ZIM PE 2.65x → 得分9）
  ❌ 先行指标得分 ≤ 3（订单簿31.7%极差 → 得分1）
  ❓ 缺失条件：边际变化开始改善

当前状态：
  估值：到位 ✅
  指标：极差 ✅
  边际改善：尚未出现 ❌

结论：
  满足2/3条件，接近底部但未确认
  需要监控SCFI连续3个月环比改善
```

## 监控清单

### 航运买入信号触发条件

1. **SCFI连续3个月环比改善**（边际变化）
2. **订单簿/船队比开始下降**（供给拐点）
3. **拆船量显著增加**（供给收缩）
4. **闲置运力下降至2%以下**（需求改善）

### 时间预期

根据行业分析师预测：
- 供过于求持续至2028-2029
- 盈利底部可能在2028年

**但**：市场会提前反映，股价底部通常早于盈利底部6-12个月。

## 框架验证结论

| 验证项 | 结果 |
|--------|------|
| 核心逻辑通用性 | ✅ 逆周期投资逻辑适用 |
| 指标体系可移植 | ✅ 结构相似，只需替换具体指标 |
| 背离信号检测 | ✅ 适用 |
| 估值体系 | ⚠️ 需要适配（增加P/B） |

**结论**：cycle-investing 框架在航运行业适用，验证了泛化能力。
