# 🔄 反思框架

每次回顾时，使用以下问题引导反思：

### 预测准确性
1. 预测方向是否正确？
2. 预测幅度是否合理？
3. 时间节奏是否准确？

### 框架有效性
1. 哪个指标最有预测力？
2. 哪个指标失效了？
3. 权重分配是否合理？

### 改进方向
1. 需要增加什么指标？
2. 需要删除什么指标？
3. 需要调整什么参数？

### 认知偏差
1. 是否存在过度自信？
2. 是否存在确认偏差？
3. 是否忽略了重要信息？

### 跨行业对比（新增）

# 🔁 迭代优化机制

## 自动迭代触发条件

当满足以下任一条件时，系统将建议启动框架升级：

| 条件 | 描述 | 行动 |
|------|------|------|
| 准确率 < 70% | 样本外准确率低于阈值 | 分析失败案例，调整参数 |
| 连续3次错误 | 同一股票连续3次方向预测错误 | 重新评估该股票特性 |
| 指标失效 | 某关键指标连续失效 | 考虑替换或降权 |
| 新周期阶段 | 市场进入新的周期阶段 | 调整动态权重 |

## 迭代流程

```
1. 收集预测结果
   python3 main.py --predict verify

2. 生成准确率报告
   python3 main.py --predict report

3. 分析错误预测
   python3 main.py --predict errors

4. 识别改进点
   - 哪些指标预测力最强？
   - 哪些规则需要调整？
   - 参数是否需要优化？

5. 更新框架版本
   - 修改 YAML 配置
   - 更新 scorer.py 评分逻辑
   - 记录版本变更原因

6. 重新生成预测
   python3 main.py --predict generate
```

## 跨行业学习

| 学习点 | 半导体 | 航运 | 能源 | 工程机械 |
|--------|--------|------|------|----------|
| 周期长度 | 3-4年 | 3-5年 | 4-6年 | 3-5年 |
| 领先指标 | DRAM价格 | SCFI | WTI油价 | 挖掘机销量 |
| 供给指标 | Memory CAPEX | 订单簿/船队比 | 钻机数 | 设备利用率 |
| 最佳买点 | DRAM触底+估值低 | SCFI触底+估值低 | 油价暴跌+估值低 | 销量暴跌+估值低 |
| 最佳卖点 | DRAM见顶+估值高 | SCFI见顶+订单簿高 | 油价过热+钻机高 | 销量过热+利用率高 |
| 代表标的 | LRCX, AMAT | ZIM, MATX | XOM, SLB | CAT, DE |
