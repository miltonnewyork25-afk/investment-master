# Twitter/X Channel Playbook
# Super Marketing Agent v1.0
# 创建时间: 2026-01-24

## 账号定位
- **领域**: 量化投资 / 周期投资 / 半导体分析
- **人格**: 数据驱动的逆向投资者
- **调性**: 专业 + 透明 + 略带自信（有数据支撑的自信）

## 内容策略

### 日常发布节奏
| 时段 | 类型 | 目的 |
|------|------|------|
| 美东 7:30 AM | Market Open Take (1-2条) | 实时感知 |
| 美东 8:00 AM | Thread (5-15条) | 深度价值 |
| 美东 12:00 PM | Data Viz / Chart (1条) | 视觉吸引 |
| 美东 4:30 PM | Market Close Summary (1-2条) | 日终回顾 |

### 内容类型配比
- 60% 教育/价值内容（框架讲解、方法论、数据分析）
- 20% 信号/实时内容（评分更新、风险预警、市场评论）
- 15% 互动/社区内容（回复、投票、问答）
- 5% 推广/CTA内容（Newsletter推广、工具推广）

### Thread 模板

**结构:**
```
Tweet 1: HOOK (数据冲击/反直觉/好奇缺口)
Tweet 2-3: CONTEXT (背景/问题)
Tweet 4-8: CONTENT (核心内容/数据/分析)
Tweet 9-10: INSIGHT (独特观点/结论)
Tweet 11: PROOF (回测数据/案例)
Tweet 12: CTA (关注/订阅/转发)
```

## 增长策略

### Phase 1: 0→1000 粉丝
- 每日 Thread 发布（不间断）
- 回复大V的投资讨论帖
- 在热门话题下发布相关洞察
- 加入投资推特互助社区
- 引用转发并添加价值评论

### Phase 2: 1000→10000 粉丝
- Twitter Spaces 参与（共同主持）
- 与其他投资博主交叉推广
- 建立固定栏目（如 #WeeklyScore）
- 开始使用 Twitter Blue 长文功能
- 精准使用话题标签

### Phase 3: 10000+ 粉丝
- 开设 Twitter 订阅
- 推出付费深度内容
- 建立私人社区
- KOL 合作推广

## 互动规则
1. 所有回复在2小时内响应
2. 质疑评论 → 用数据回应（不争论）
3. 提问 → 详细解答（展示专业度）
4. 赞美 → 感谢 + 引导订阅
5. 负面 → 承认局限性 + 展示透明度

## 度量追踪
- 每日: 印象数、互动率、粉丝变化
- 每周: Thread 表现排名、最佳发布时间分析
- 每月: 增长趋势、内容类型效果对比
