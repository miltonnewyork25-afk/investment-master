# 超级选题引擎 (Super Topic Selection Engine)
# 版本: v1.0
# 创建: 2026-01-24
# 功能: 从任意信息中识别最具传播效应+最利公司成长的选题
# 覆盖平台: X/Twitter, Reddit, YouTube, 雪球, 小红书

---

## 核心定理

**选题决定一切。** 如果选题弱，后续一切(标题/Hook/内容)都无法挽救。

```
传播效果 = 选题力(50%) × 切入角度(25%) × 执行质量(15%) × 时机(10%)
```

选题不是"写什么"——是"为什么这个话题现在在这个平台能引发传播"。

## Part 1: 选题评分系统 (Topic Scoring System)

### 1.1 五维评分矩阵

对每个候选选题评分(0-10)，计算加权总分：

```
Topic Score = V×0.25 + G×0.25 + T×0.20 + U×0.15 + D×0.15

V = Virality (传播潜力)
G = Growth (公司成长贡献)
T = Timing (时机匹配度)
U = Uniqueness (独特性/不可替代性)
D = Debate (辩论潜力/多态有效性)
```

### 1.2 各维度评分标准

#### V - Virality (传播潜力) 0-10
| 分数 | 标准 |
|------|------|
| 9-10 | 高唤醒情绪(敬畏/愤怒/焦虑) + 社交货币 + 实用价值 |
| 7-8 | 强好奇心缺口 + 反直觉 + 可引用金句 |
| 5-6 | 有价值但缺乏情绪驱动 |
| 3-4 | 信息有用但无传播动力 |
| 1-2 | 纯信息，无分享动机 |

**学术基础（Berger & Milkman, JMR）：**
- 70%病毒内容有强情绪共鸣
- 高唤醒情绪(awe/anger/anxiety) > 低唤醒(sadness)
- 正面内容比负面更易传播(但负面+高唤醒=极强)

#### G - Growth (公司成长贡献) 0-10
| 分数 | 标准 |
|------|------|
| 9-10 | 直接展示核心竞争力 + 建设护城河 + 引导付费转化 |
| 7-8 | 展示方法论 + 积累权威 + 引导关注/订阅 |
| 5-6 | 品牌认知提升但无直接转化 |
| 3-4 | 蹭热点但与产品弱关联 |
| 1-2 | 纯流量无品牌积累 |

#### T - Timing (时机匹配度) 0-10
| 分数 | 标准 |
|------|------|
| 9-10 | 未来48h内有重大相关事件(财报/政策/数据) |
| 7-8 | 本周有相关事件 + 话题热度上升中 |
| 5-6 | 常青话题(任何时候发都有效) |
| 3-4 | 话题已过热度峰值 |
| 1-2 | 完全没有时效性加成 |

#### U - Uniqueness (独特性) 0-10
| 分数 | 标准 |
|------|------|
| 9-10 | 只有我们能产出(独家数据/独家方法论) |
| 7-8 | 独特角度(别人有数据但没这个切入) |
| 5-6 | 好内容但其他人也能写 |
| 3-4 | 常见观点的重述 |
| 1-2 | 任何人都能写的通用内容 |

#### D - Debate (辩论潜力) 0-10
| 分数 | 标准 |
|------|------|
| 9-10 | 观点两极化(支持者狂转+反对者怒怼) = 多态有效 |
| 7-8 | 明确逆向立场(部分人强烈不同意) |
| 5-6 | 有讨论空间但无强对立 |
| 3-4 | 大部分人会同意(无辩论价值) |
| 1-2 | 无争议的事实陈述 |

### 1.3 2阶选美加成 (+0-3 Bonus)

通过2阶思维检验的选题获得加成分：
```
+1: 读者觉得"分享这个显得我懂投资"
+1: 读者觉得"我的朋友也会觉得有用"
+1: 读者觉得"分享这个确认了我的身份标签"(逆向投资者/理性投资者/AI先驱)
```

### 1.4 评分阈值

| 总分 | 决策 |
|------|------|
| ≥8.5 | 立即执行，最高优先级 |
| 7.0-8.4 | 进入内容管线，本周完成 |
| 5.5-6.9 | 储备，等待更好时机 |
| <5.5 | 淘汰，不值得投入 |

## Part 2: 经典选题角度库 (Topic Angle Library)

### 2.1 十大超级选题角度

#### 角度1: 对抗叙事 (Confrontation)
```
公式: "[权威方A]说X, 我们的数据说Y"
偏误触发: 权威对抗 + 好奇心缺口 + 社交货币
V=9 D=10
示例: "25位分析师说买入, AI框架说23.5分卖出"
适用: X(辩论) + Reddit(DD) + 雪球(逆向观点)
```

#### 角度2: 数据颠覆 (Data Disruption)
```
公式: "你以为X, 但数据显示Y"
偏误触发: 确认偏误打破 + 锚定效应 + 权威
V=8 U=9
示例: "你以为NVDA是最好的半导体股? 数据显示LRCX回报更高"
适用: X(thread) + YouTube(视频) + 雪球(深度分析)
```

#### 角度3: 预测+验证 (Prediction + Verification)
```
公式: "我X天前预测了Y, 结果是Z"
偏误触发: 权威建设 + 确认偏误 + 社会认同
G=9 (护城河建设)
示例: "3个月前框架给LRCX 23.5分, 现在跌了30%"
适用: X(验证thread) + 雪球(实盘追踪) + 小红书(结果展示)
```

#### 角度4: 教育解密 (Educational Decode)
```
公式: "X的底层原理是什么? 99%的人不知道"
偏误触发: 知识缺口 + 权威 + 社交货币(学到=可分享)
G=8 (方法论展示)
示例: "周期投资的数学原理: 为什么P/E>50x是危险信号"
适用: X(10-K Diver风格mega-thread) + YouTube(长视频) + 小红书(科普)
```

#### 角度5: 透明自曝 (Radical Transparency)
```
公式: "我错了/我不知道/这是我的完整记录"
偏误触发: 互惠(坦诚→信任) + 一致性(持续透明→品牌)
G=9 (信任护城河) D=7
示例: "框架说买但跌了20%, 这是我的反思和模型更新"
适用: X + 雪球(实盘文化) + YouTube(复盘视频)
```

#### 角度6: 独家数据/工具 (Exclusive Data)
```
公式: "这个数据/工具只有我们有, 免费给你"
偏误触发: 稀缺性 + 互惠 + 实用价值
G=10 U=10
示例: "2004只美股的AI周期评分完整表格, 免费下载"
适用: X(引流) + Reddit(工具帖) + GitHub(开源)
```

#### 角度7: 时间机器 (Time Machine)
```
公式: "如果X年前你按这个策略做, 现在会..."
偏误触发: 损失厌恶(错过的痛) + 锚定(具体数字) + 后见之明
V=8 G=7
示例: "如果2023年你听了框架的话买LRCX, 现在+305%"
适用: X + YouTube(可视化) + 小红书(结果导向)
```

#### 角度8: 隐藏信号 (Hidden Signal)
```
公式: "所有人在看X, 但真正的信号是Y"
偏误触发: 好奇心缺口 + 稀缺(信息不对称) + 权威
V=9 U=8
示例: "所有人在看营收, 但递延收入才是真正领先指标"
适用: X(thread) + 雪球(深度) + Reddit(DD)
```

#### 角度9: 排行榜/矩阵 (Rankings/Matrix)
```
公式: "X个[类别]排名/对比/评分"
偏误触发: 完整性偏好 + 锚定 + 社交货币(可引用)
V=7 G=6 (品牌曝光)
示例: "2026年半导体设备商AI周期评分Top 10"
适用: 全平台(高度可视化+可截图)
```

#### 角度10: 场景模拟 (Scenario Simulation)
```
公式: "如果X发生, 会怎样? 我们模拟了3种结果"
偏误触发: 不确定性降低 + 框架效应 + 实用价值
V=7 G=8
示例: "如果中国限制半导体出口, LRCX三种情景分析"
适用: X(thread) + YouTube(视频) + 雪球(分析帖) + Reddit(DD)
```

### 2.2 选题角度×情绪唤醒矩阵

```
                 正面情绪(Awe/Excitement)     负面情绪(Anger/Anxiety)
高唤醒  ────────────────────────────────────────────────────────
        │ "100%准确率的AI框架"(敬畏)    │ "所有人在走进陷阱"(焦虑)   │
        │ "+305%涨幅验证"(兴奋)         │ "你的持仓可能归零"(恐惧)   │
        │ 角度: 预测验证/数据颠覆        │ 角度: 对抗叙事/隐藏信号     │
        │ V=9, 传播力极强               │ V=10, 传播力最强            │
中唤醒  ────────────────────────────────────────────────────────
        │ "学会周期投资原理"(满足)       │ "多数人不知道的规则"(轻焦虑)│
        │ 角度: 教育解密/排行榜          │ 角度: 透明自曝/场景模拟     │
        │ V=6-7, 稳定传播               │ V=7-8, 较强传播             │
低唤醒  ────────────────────────────────────────────────────────
        │ "温和的市场展望"(平静)         │ "遗憾错过机会"(悲伤)        │
        │ V=3-4, 弱传播                 │ V=2-3, 极弱传播(避免!)      │
        ──────────────────────────────────────────────────────────

结论: 始终选择高唤醒情绪象限的选题角度
最优区: 右上(负面高唤醒) → 最强传播 → 但需平衡品牌形象
次优区: 左上(正面高唤醒) → 强传播 → 品牌正面
避免区: 下方两象限 → 弱传播
```

## Part 4: 选题决策流程 (Topic Decision Protocol)

### 4.1 信息→选题的转化流程

```
┌─────────────────────────────────────────────────┐
│  INPUT: 任意信息源                                │
│  (财报数据/市场事件/框架评分/行业新闻/用户反馈)    │
└──────────────────────┬──────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────┐
│  Step 1: 选题萃取 (Topic Extraction)             │
│  从信息中识别所有可能的选题(MECE穷尽)             │
│  每条信息至少萃取3个不同角度的选题               │
└──────────────────────┬──────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────┐
│  Step 2: 五维评分 (5D Scoring)                   │
│  V(传播) × G(成长) × T(时机) × U(独特) × D(辩论) │
│  + 2阶选美加成(0-3)                             │
└──────────────────────┬──────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────┐
│  Step 3: 角度选择 (Angle Selection)              │
│  从10大角度中选择最匹配的1-2个                    │
│  检查情绪唤醒象限(必须在高唤醒区)                 │
└──────────────────────┬──────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────┐
│  Step 4: 平台适配 (Platform Adaptation)          │
│  同一选题×角度 → 适配到目标平台的规则             │
│  一个选题可同时适配多个平台(飞轮效应)             │
└──────────────────────┬──────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────┐
│  Step 5: 合成受众预审 (Synthetic Audience Test)   │
│  210+ persona模拟 → 选最优变体                   │
│  (仅对≥8.5分的选题执行完整预审)                   │
└──────────────────────┬──────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────┐
│  OUTPUT: 最终选题决策                             │
│  选题 + 角度 + 平台 + Hook + 执行优先级          │
└─────────────────────────────────────────────────┘
```

### 4.2 选题萃取示例

**输入信息**: "LRCX Q4 2025财报: 营收超预期5%, EPS超预期8%, 但guidance低于预期3%"

**萃取选题(MECE):**

| # | 选题 | 角度 | V | G | T | U | D | 加成 | 总分 |
|---|------|------|---|---|---|---|---|------|------|
| 1 | "Beat but Guide Down: 框架的Top Divergence信号验证" | 预测验证 | 9 | 10 | 10 | 9 | 8 | +3 | **9.8** |
| 2 | "为什么Beat不重要: 递延收入才是真信号" | 隐藏信号 | 8 | 8 | 9 | 8 | 7 | +2 | **8.4** |
| 3 | "LRCX 50x P/E的数学不可能: 估值回归计算" | 数据颠覆 | 7 | 7 | 8 | 7 | 8 | +1 | **7.7** |
