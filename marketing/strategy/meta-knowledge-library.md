## 第一层：人性底层原理（Human Nature Primitives）

### 1.1 Kahneman 行为经济学
**核心洞察：人类决策不理性，可以被系统性影响。**

| 原理 | 定义 | 营销应用 |
|------|------|---------|
| **损失厌恶** | 失去的痛苦是获得快乐的2倍 | "错过这个信号等于亏损X%" |
| **锚定效应** | 第一个信息决定后续判断的参照系 | 先展示$50000 Bloomberg，再展示免费框架 |
| **框架效应** | 同一信息不同表述导致不同决策 | "100%准确" vs "从未失败" |
| **禀赋效应** | 拥有的东西被高估 | 免费试用→不想失去 |
| **确认偏误** | 人只接受支持已有信念的信息 | 对逆向投资者说"你的直觉是对的" |
| **可得性启发** | 容易想起的事被认为更常见 | 重复曝光核心信息 |
| **社会认同** | 不确定时模仿多数人行为 | 展示订阅者数量、社区规模 |

### 1.2 Maslow 需求层次
```
自我实现 → "成为更好的投资者"
尊重需求 → "比其他人更聪明的决策"
社交需求 → "加入理性投资者社区"
安全需求 → "保护你的资产不受周期伤害"
生理需求 → "赚钱养家"
```
**应用**：不同阶段用户的主要诉求不同。初期抓安全+尊重，后期抓自我实现。

### 1.3 JTBD（Jobs to Be Done）
**核心洞察：用户不是买产品，是"雇佣"产品完成一个任务。**

投资大师的用户"工作"：
- 功能性工作：在正确时机做出买卖决策
- 情感性工作：在市场恐慌时获得信心
- 社会性工作：在朋友面前展示投资智慧

**推论**：营销不是推销功能，是描述"工作完成后的状态"。

## 第二层：竞争战略原理（Competitive Strategy）

### 2.1 Porter 五力模型
**应用于投资分析产品市场：**

| 力量 | 分析 | 策略 |
|------|------|------|
| 新进入者威胁 | 低（AI框架有技术壁垒+数据积累） | 持续迭代保持领先 |
| 供应商议价力 | 中（依赖FMP API数据） | 多数据源冗余 |
| 买方议价力 | 高（免费替代品多） | 差异化：透明+开源+准确率 |
| 替代品威胁 | 高（券商研报、其他工具） | 蓝海定位：周期投资细分 |
| 行业竞争 | 中高 | 聚焦护城河：方法论+数据 |

### 2.2 护城河理论（Buffett）
**投资大师的护城河分析：**

| 护城河类型 | 现状 | 强化方向 |
|-----------|------|---------|
| 品牌 | 弱（新品牌） | 持续发布+100%准确率品牌资产 |
| 网络效应 | 无 | 建立用户社区→UGC→更多用户 |
| 转换成本 | 低 | 用户定制化配置→数据积累→难迁移 |
| 成本优势 | 有（开源免费） | 保持免费层→付费增值 |
| 数据护城河 | 中（10年回测数据） | 持续积累实时评分历史 |

### 2.3 定位理论（Ries & Trout）
**核心原则**：在用户心智中占据一个独特位置。

**投资大师的定位**：
- 品类："AI周期投资框架"（创建新品类）
- 心智关键词："100%准确率"
- 对标竞品："比Bloomberg更精准，比Seeking Alpha更系统"
- 定位语句：

> **For** 自主投资者 **who** 想在周期底部系统性买入，
> **Investment Master** is an **AI投资框架** that **用100%历史准确率定位周期底部**。
> **Unlike** 主观分析和昂贵的终端，**we** 完全透明、开源、可审计。

### 2.4 蓝海战略（Kim & Mauborgne）
**ERRC 分析（消除-减少-增加-创造）：**

| 行动 | 因素 | 说明 |
|------|------|------|
| **消除** | 主观判断 | 消除分析师意见，只看数据 |
| **消除** | 高成本 | 消除终端费用，免费提供 |
| **减少** | 覆盖广度 | 减少覆盖范围，聚焦周期行业 |
| **减少** | 复杂度 | 减少操作复杂度，0-100分制 |
| **增加** | 透明度 | 增加方法论可审计性 |
| **增加** | 准确性 | 增加回测验证严谨度 |
| **创造** | 供应链图谱 | 创造独家的关系数据 |
| **创造** | AI原生体验 | 创造Claude Skill交互方式 |

## 第三层：增长引擎原理（Growth Engine）

### 3.1 扩散理论（Rogers）
**投资大师的采用曲线策略：**

```
Innovators (2.5%)     → AI工具爱好者、量化交易员
Early Adopters (13.5%) → 自主投资者中的技术派
  --- CHASM ---
Early Majority (34%)  → 普通自主投资者
Late Majority (34%)   → 被市场教育后的跟随者
Laggards (16%)        → 传统投资者
```

**跨越鸿沟策略**：
- 当前阶段：瞄准 Innovators + Early Adopters
- 信息策略：强调"新颖性+技术优势+数据证明"
- 过鸿沟后：转为"社会证明+案例+易用性"
- 触发点：需要达到 10-16% 采用率后可自增长

### 3.2 网络效应与飞轮
**投资大师飞轮设计：**

```
优质内容 → 用户增长 → 社区讨论 → 更多洞察
    ↑                                    ↓
内容改进 ← 用户反馈 ← 社区信任 ← 框架改进
```

**间接网络效应**：
- 更多用户 → 更多讨论 → 更多验证 → 框架更可信 → 更多用户
- 更多数据 → 更好回测 → 更高准确率 → 更多引用 → 更多用户

### 3.3 AARRR 海盗指标
**投资大师的 AARRR 映射：**

| 阶段 | 指标 | 当前重点 |
|------|------|---------|
| **Acquisition** | 渠道触达 → 首次访问 | Twitter Thread → Profile |
| **Activation** | 首次"AHA!" | 看到评分表+理解意义 |
| **Retention** | 持续回访 | 周报订阅+每周打开 |
| **Referral** | 主动推荐 | 转发Thread+邀请好友 |
| **Revenue** | 付费转化 | 付费订阅（未来） |

**优化顺序**：Retention > Activation > Acquisition > Referral > Revenue

### 3.4 边际收益与饱和点
**关键洞察**：每个渠道都有S曲线——初期高回报，中期线性，后期饱和。

**策略**：
- 监测每个渠道的边际CAC
- 当某渠道边际回报下降 → 转投新渠道
- 保持 70/20/10 分配：70%已验证渠道 / 20%实验渠道 / 10%探索渠道

## 第四层：内容与传播原理（Content & Virality）

### 4.1 信息传播的第一性原理
**信息被传播的3个必要条件**：
1. 触发注意力（Hook/Interrupt）
2. 产生情绪共振（Emotion/Relevance）
3. 降低传播摩擦（Ease of Sharing）

### 4.2 社交货币理论（Berger STEPPS + Cialdini）
**六大影响力原则（Cialdini）**：
1. 互惠（给免费内容→用户回报关注）
2. 承诺一致性（一旦订阅→持续阅读）
3. 社会认同（"1000人已订阅"）
4. 权威（"100%准确率"数据支持）
5. 喜好（建立个人品牌→信任迁移）
6. 稀缺（"仅限前100名"/"本周信号"）

### 4.3 叙事经济学（Narrative Economics, Shiller）
**核心洞察**：经济叙事像病毒一样传播，影响经济行为。

**应用**：构建一个强力的投资叙事——"逆周期投资者用AI在底部精准买入"。这个叙事本身就是一个可传播的"模因"（meme）。

## 第五层：品类创新原理（Category Design）

### 5.1 品类之王理论（Play Bigger）
**核心**：不是在现有品类中竞争，而是创造并主导一个新品类。

**投资大师的品类设计**：
- 传统品类："投资分析工具"（红海）
- 新品类："AI周期信号系统"（蓝海）
- 品类名称候选：Cycle Intelligence / Cycle Signal / AI Cycle Framework

### 5.2 品类教育策略
创造新品类需要教育市场：
1. 定义问题（"你不知道自己在周期的哪个位置"）
2. 命名品类（"Cycle Intelligence"）
3. 展示解决方案（"0-100评分系统"）
4. 证明效果（"100%回测准确率"）

## 第六层：系统思维原理（Systems Thinking）

### 6.1 复利效应（Compound Effect）
每个营销动作的效果不是加法，是乘法：
- 内容质量 × 渠道覆盖 × 频率 × 时间 = 复合增长
- 任何一个因子为0，总效果为0

### 6.2 杠杆点识别
系统中改变一个杠杆点可以产生不成比例的效果：
- 投资大师的杠杆点："100%准确率"这个声明
- 原因：它同时触发了好奇、贪婪、怀疑（引发讨论）、权威感

### 6.3 反馈循环
- 正反馈：准确预测→信任增加→用户增长→更多验证→更多信任
- 负反馈：预测失败→信任崩塌→用户流失（风险管理重要）
- 控制策略：始终强调"方向性准确"而非"精确预测"
