**评分标准**:
- A+ (90-100分): 发布级，数据质量优秀
- B (70-89分): 可用级，小幅优化后可发布
- C (50-69分): 需修订，存在较多质量问题
- F (<50分): 不合格，禁止发布

**质量评分公式**:
```
质量评分 = (
    可验证数据占比 × 40% +
    数据来源标注率 × 30% +
    时效性得分 × 15% +
    逻辑完整性 × 10% +
    免责声明 × 5%
)
```

**数据来源分层**:
- Tier 1 (40分权重): SEC文件、财报、官方发布
- Tier 2 (15分权重): Bloomberg、Reuters、知名机构
- Tier 3 (-20分惩罚): 社交媒体、传闻、未证实来源
