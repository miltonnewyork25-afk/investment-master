## 分析深度策略

### 极深度分析 (评分5，16个模块)
**共同特征**: 与CQ1估值争议或多个高权重CQ高度相关

**模块列表**:
- Phase 2: U7财务数据, U8估值数据, U9分析师观点, U10市场分歧
- Phase 3: U12护城河, U14投资命题, U15洞察卡, C6行为护城河, C8定价权
- Phase 4: U19偏差检查, U21看空构建, U24压力测试
- Phase 5: U25评分, U27预测, U28估值修正, U29仓位建议

**执行要求**:
- 字符目标: ≥2,500字符/模块
- 数据标注密度: ≥20个/万字
- So What评分: ≥8分
- 禁止压缩内容

### 深度分析 (评分4，20个模块)
**共同特征**: 与2-3个CQ相关性较高

**执行要求**:
- 字符目标: 1,500-2,000字符/模块
- 数据标注密度: ≥15个/万字
- So What评分: ≥7分
- 可适度压缩非核心段落

### 标准执行 (评分3，4个模块)
**共同特征**: 与CQ相关性中等

**执行要求**:
- 字符目标: 800-1,200字符/模块
- 数据标注密度: ≥12个/万字
- So What评分: ≥5分
- 可压缩至核心段落
