# Fintech信贷周期风暴矩阵 (Credit Cycle Storm Matrix)

> **独创框架**: 类似半导体行业的6层周期雷达，专为Fintech公司设计的5层领先指标系统
> **核心理念**: 信贷周期的转折点不是突然出现的，而是从宏观层逐步传导到微观层，每一层之间有可预测的时滞
> **关联文件**: `sofi_kill_switch_system.md` 第四章

---

## 框架概览

```
                        ┌──────────────────────────┐
                        │   L5: 微观资产质量指标     │  滞后0个月（同步确认）
                        │   NCO率 / 逾期率 / 拨备   │
                        └──────────┬───────────────┘
                                   │ 传导 3-6个月
                        ┌──────────▼───────────────┐
                        │   L4: 借款人行为指标       │  领先3-6个月
                        │   早期逾期 / 还款模式      │
                        └──────────┬───────────────┘
                                   │ 传导 3-6个月
                        ┌──────────▼───────────────┐
                        │   L3: 消费者财务健康指标    │  领先6-12个月
                        │   储蓄率 / 债务收入比      │
                        └──────────┬───────────────┘
                                   │ 传导 3-6个月
                        ┌──────────▼───────────────┐
                        │   L2: 就业与收入指标       │  领先9-15个月
                        │   初领失业金 / 薪资增长    │
                        └──────────┬───────────────┘
                                   │ 传导 3-6个月
                        ┌──────────▼───────────────┐
                        │   L1: 宏观金融条件指标     │  领先12-18个月
                        │   收益率曲线 / 金融条件    │
                        └────────────────────────── ┘
```

---

## L1: 宏观金融条件指标 (领先12-18个月)

> **逻辑**: 货币政策和金融条件的变化是信贷周期最早的驱动力。紧缩的金融条件最终会通过提高借贷成本、减少信贷供给来恶化资产质量。

| 指标 | 领先时间 | 数据来源 | 预警阈值 | 危险阈值 |
|------|---------|---------|---------|---------|
| 美国收益率曲线斜率 (10Y-2Y) | 12-18个月 | FRED (T10Y2Y) | 倒挂>50bps持续3个月 | 倒挂后重新变陡>100bps（衰退确认信号） |
| 芝加哥联储金融条件指数 (NFCI) | 12-15个月 | FRED (NFCI) | NFCI > 0 (条件收紧) | NFCI > 0.5 持续2个月 |
| 联邦基金利率变化方向 | 12-18个月 | FOMC声明/点阵图 | 加息周期末端未降息 | 意外加息或降息预期推迟>6个月 |
| 高收益信用利差 (HY OAS) | 12-15个月 | ICE BofA HY指数 | OAS > 400bps | OAS > 600bps |

**综合判读**: 当L1层>=3个指标同时进入预警区间，启动"信贷寒冬预警"，将所有信贷类KS监控频率提升至月度。

---

## L2: 就业与收入指标 (领先9-15个月)

> **逻辑**: SoFi的核心用户群是高收入、高教育的年轻专业人士。这个群体的就业状况直接决定他们的还款能力。科技和金融行业的裁员潮对SoFi用户的影响远大于整体失业率所显示的。

| 指标 | 领先时间 | 数据来源 | 预警阈值 | 危险阈值 |
|------|---------|---------|---------|---------|
| 初领失业救济金人数 | 9-12个月 | DOL周报 (FRED: ICSA) | 4周均值>250K | 4周均值>300K |
| 科技/金融行业裁员追踪 | 9-12个月 | Layoffs.fyi / Challenger报告 | 月度裁员>30K | 月度裁员>50K连续3个月 |
| 实际薪资增长率 | 12-15个月 | BLS就业报告 | 实际增长<0% | 实际增长<-1%持续2季 |
| JOLTS职位空缺/失业人数比 | 9-12个月 | BLS JOLTS (FRED) | 比值<1.2 | 比值<1.0 |

**综合判读**: 当L2层>=2个指标进入预警 且 L1层已有预警信号 -- 启动"用户群风险评估"，重新审视SoFi借款人信用质量分布。

**SoFi特有洞察**: 传统银行关注总体失业率即可，但SoFi必须额外关注科技行业裁员。其核心用户群（25-40岁、FICO 700+、高收入科技/金融从业者）的就业风险高度集中在科技和金融行业。2022-2023年科技裁员潮期间，SoFi的个人贷款违约率上升速度快于行业整体，验证了这一逻辑。

---

## L3: 消费者财务健康指标 (领先6-12个月)

> **逻辑**: 就业恶化后，消费者会先动用储蓄、增加借贷来维持消费。储蓄缓冲耗尽后，违约才会显现。这个缓冲层的厚度决定了从就业恶化到违约飙升的时滞。

| 指标 | 领先时间 | 数据来源 | 预警阈值 | 危险阈值 |
|------|---------|---------|---------|---------|
| 个人储蓄率 | 6-9个月 | BEA (FRED: PSAVERT) | <4% | <3% 持续3个月 |
| 消费者债务/可支配收入比 | 6-12个月 | 纽约联储消费信贷报告 | 比值>10% | 比值>12%（接近2008年水平） |
| 信用卡利用率 | 6-9个月 | 纽约联储/TransUnion | >30% 整体 | >35% 整体 或 Subprime>50% |
| BNPL逾期率 | 6-9个月 | CFPB BNPL报告/Affirm数据 | 逾期率>5% | 逾期率>8% |

**综合判读**: 当L3层>=2个指标进入预警 -- 这意味着违约潮距离显现还有6-12个月窗口期。此时是最后的主动调仓机会。

**SoFi特有洞察**: SoFi的用户群体储蓄率和债务行为可以通过SoFi Money/Invest的存款/投资数据间接观测。如果SoFi内部看到用户存款余额中位数下降，这是比公开宏观数据更早的微观信号。关注管理层在电话会议中对"用户存款行为变化"的措辞。

---

## L4: 借款人行为指标 (领先3-6个月)

> **逻辑**: 在正式违约之前，借款人会出现可观测的"压力行为" -- 降低自动还款金额、申请展期、只还最低付款额。这些行为指标是最后的预警窗口。

| 指标 | 领先时间 | 数据来源 | 预警阈值 | 危险阈值 |
|------|---------|---------|---------|---------|
| 30天逾期率（早期逾期） | 3-6个月 | 季报/10-Q信贷质量披露 | 30天逾期率上升>50bps QoQ | 30天逾期率上升>100bps QoQ |
| 还款行为模式变化 | 3-6个月 | 管理层电话会议/投资者日 | 管理层首次提及"支付行为变化" | 管理层收紧承保标准 |
| 贷款修改/展期比例 | 3-6个月 | 季报附注 | 修改比例>3% | 修改比例>5% |
| 再融资申请中的高DTI占比 | 3-4个月 | 行业ABS数据/Rating Agency报告 | DTI>40%占比>15% | DTI>40%占比>25% |

**综合判读**: 当L4层任何1个指标进入危险区 -- 触发KS-SOFI-001的L1预警。L4是信号到KS的"最后一英里传导器"。

**SoFi特有洞察**: SoFi的AI承保模型是否能比传统FICO评分更早发现这些行为变化，是其技术护城河的核心验证点。如果管理层在电话会议中表示"我们的模型已提前识别并收紧了高风险群体的授信"，这是正面信号；如果管理层被动表示"我们看到了超预期的逾期上升"，则意味着AI模型失效，这本身是一个技术层面的Kill Switch。

---

## L5: 微观资产质量指标 (同步确认，滞后0个月)

> **逻辑**: 这一层不是领先指标，而是确认指标。当L5指标恶化时，信贷周期的转折已经发生，此时的行动是止损而非预防。

| 指标 | 领先时间 | 数据来源 | 预警阈值 | 危险阈值 |
|------|---------|---------|---------|---------|
| 净冲销率 (NCO) | 同步 | 季报 | >4.0% (个人贷款) | >7.0% (直接对应KS-001的L3) |
| 拨备覆盖率变化 | 同步 | 季报 | 拨备率QoQ上升>30bps | 拨备率QoQ上升>60bps |
| 90天+逾期率 | 同步 | 季报 | >2.0% | >3.5% |
| Gain-on-Sale利润率 | 同步 | 季报 | 利润率QoQ下降>200bps | 利润率转负 |

**综合判读**: L5是"已发生确认层"。当L5>=2个指标进入危险区，KS-001自动升级至L2或L3。

## 风暴矩阵综合运用

```
┌────────────────────────────────────────────────────────────────┐
│                    信贷周期温度计读数                            │
├──────────┬─────────┬─────────┬──────────┬─────────────────────┤
│  L1状态   │  L2状态  │  L3状态  │  L4状态   │  L5状态            │
│  宏观金融  │  就业收入 │  消费健康 │  借款行为  │  资产质量          │
├──────────┼─────────┼─────────┼──────────┼─────────────────────┤
│  🟢/🟡/🔴 │ 🟢/🟡/🔴 │ 🟢/🟡/🔴 │ 🟢/🟡/🔴  │ 🟢/🟡/🔴           │
└──────────┴─────────┴─────────┴──────────┴─────────────────────┘

评估规则:
  全绿   → 正常运行，季度标准监控
  1黄    → 关注，月度监控，不调仓
  2黄    → 预警，提升KS监控频率，准备减仓方案
  3黄    → 高度预警，执行KS-011 L1，停止加仓
  1红    → 确认转折，触发对应KS的L2
  2红    → 危机确认，触发KS-001/KS-011的L2-L3
  3红+   → 全面信贷危机，触发所有信贷类KS的L3
```
