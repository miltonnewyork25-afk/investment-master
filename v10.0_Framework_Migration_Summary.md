## 🚀 使用方式

### 快速启动

在半导体worktree中，用户只需输入：
```
"分析TSM，使用v10.0预测市场框架"
```

框架自动执行：
1. **公司识别** → TSM = 代工厂类型
2. **引擎加载** → 5个v10.0引擎并行启动
3. **概率获取** → Polymarket实时数据
4. **背离分析** → PPDA算法计算
5. **情绪指数** → PMSI半导体指数
6. **报告生成** → 完整v10.0分析

### 高级配置

可自定义：
- 预测市场事件监控列表
- Kill Switch触发阈值
- PMSI指数权重配置
- 报告模板格式
